Main Concept of Switch Case Break Statement :

=> [JS Switch Case Break Statement] is one of Built-in Conditional Function in JS which comparisoing between Several Values or Variables as Cases to Implement certain Codes in Case one of these Contion has Achieved .

=> [JS Switch Case Break Statement] can include so many Cases or Conditions .

=> [JS Switch Case Break Statement] has several progrmaticaly uses and Benefits in all Web Applications

=> [JS Switch Case Break Statement] Can Comparison between Variables, Static Amounts or be as interanl function within other one

Syntax of Switch Case Break Statement

=< Switch Case Break Statement Syntax:

    switch (Expression) {
       case value:
           Codes to be Excecuted;
           break;
       case value:
            to be Excecuted;
            break;
       case value:
            Codes to be Excecuted;
            break;
       default:
            Codes to be Excecuted (in Case non of All Cases is Achieved);
            break;
    }

Properties of [Switch Case Break Statement] :

  1. It can Hold many Cases to be Compared with the main Expression
  2. In each Case there some code(s) will be excecuted in case the Expression is Equal the Case's Value
  3. In each Case there must be an (break;) code to stop exceuting the next Case, and stop at the codes of following case.
  4. At the End of Switch Case there must be [Default] Statement to implement a certain code in case there are non of given Cases is Implemented, to be more logical Code Statement
  5. There is Abilty to merge more than one code with only one Code(s) to be implement

Properties of [JS Switch Case Break Statement] :

=> It is very helpful in Application (in any programming Languages) due to it's features.

=> It Some time can be more Suitable Conditional Statement when dealing with so much probabilies or Conditions .

=> It can hold unlimited Cases & unlimited Codes within each Code, so it is a critical Programing Function in any application, even it was used as main Contaener Function or internal
