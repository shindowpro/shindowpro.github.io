Main Concept of [Javascript Events ] :

=> [Javascript Events] are common in any programing language with some differents, and they are very critical in any programing tasks.

=> [Javascript Events] are to run some code(s) if certain action had been implemeted in web page by the user or even loading webpage.

=> [Javascript Events] are conditional Code(s) of implemented a certain action, and each event has it's specific action.

=> [Javascript Events] understanding will be very useful when dealing with JS Laibraries & J-Querry.

=> [Javascript Events] have some types are Common in use & other are Rarely to use.

[Javascript onload] Event:

=> Onload Event Example:

HTML Codes :
<div> id="DisplayOnload" class="JSonloadExp">
</div>
<button> class="btn" onclick="welcome()"> click to start
</button>

Javascript Codes [Internal]:

   window.onload = function () {

[Javascript onclick] Event:

=> (Reusing Global Variable's Name inside Local Scope as new Local Variable) :

HTML Codes :
<div> id="Displayonclick" class="JSonclickExp">
</div>

</button>
