Main Concept of [Javascript Scope ] :

=> [Javascript Scope] is one of the basic Function in any Programing language, and its about Variables Declaration and Codes Ranges (Ranges).

=> [Javascript Scope] has (2) main Types, [Global] & [Local] , and will be Disclaimed later in this Lesson .

=> [Javascript Scope] has several advanced progrmatical uses and Benefits in all Web Applications

=> [Javascript Scope Part 1] can give great Advantage Specially when used with Functions.

Main Types of [Javascript Scope]

=< [Javascript Scope] has two (2) main Types as Following :

  1. [Global Scope] :

    2. In Global Scope any Variable Declared with it => Can be Invoked or Recalled from any Global Or Local Scope in the same Javascript File

  2. [Local Scope] :

    3. In Local Scope Variable Declared with it => Can be Invoked or Recalled ONLY from the same Local Scope in the same Javascipt File {Local Scopes such as : Functions }

Properties of [Javascript Scope] :

  1. [Global Scope]: any codes or varivbles can be accesseed from any Local Scope in the same JS file. While any Codes , Script or Variables in [Local Scope] can Not be accessed from Global scopes, and Can only accessed from the same [Local scope] only
  2. [Global Scope] :any Variable Defined on it can be Accessed from any [local Scope] in the same JS File , and Access include {Re-Defining, Allocating, Re-Allocating, processing, Invoking or Calling}
  3. Variable's Name Defined in [Global scope] can be used with the Same Name in the [Local Scope] and will be a {Different} Variable totally as a unique Variables in Local Scope only

Usages And Benefits of [Javascript Scope] :

=> It is very helpful in Application (in any programming Languages) due to it's features.

=> Scopes are Critical in any Programing language, and it's Essintial in Rasing Performance of the any Web application in browser by providing the ability to re Use the same Variables in the same JS file by {Re-Defining, Allocating, Re-Allocating, processing, Invoking or Calling, Increament & Decreament }.

=> Scopes including using Funcitions inside other functions as Local Scopes inside each other
