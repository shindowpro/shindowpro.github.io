[Javascript Array Add Items] Concept:

=> [Array Add Items ] : Is one of the most important Function of Array in any programing language which allow to add new or additional element into the array, whether in First position, Last position or in specific postion.

=> [Array Add Items] : There are Different [5] Ways or Methods to Perform or excecute this function in JS in Array, each way or method has specific properties to perform Addition process .

=> [Array Add Items] : Addition in Arrays is a critical function in Programing generally

[Javascript Array Add Items] Importance & Benefits:

=> [Array Add Items] : Is Essential to any programmer or developer to provide more Control of the Array by providing the Ability to Add/Edit/Delete any Array Element of any Array Elements (Regardless of Array Data Type).

=> [Array Add Items] : Is Essential to perform the addition function in array whether it was by interaction with user / browser, or in BackEnd or in programing Appliction .

  5. => Fifth Method (Splice Method):

    6. => [Splice Method] concept:
    This Method is specially desingned fo Js P.language, and it is more advanced due to it's Multiple features it has, Controls it providing in adding/Editing /Substituating/Deleting processes Elements to the Array.

    => [Splice Method] Syntax:
    Due to Many Several Function and Features Splice Function are providing to Array Control, there are different Syntax upon the Function it will be used for.

    =>[Splice Method] General Syntax as following:

    var arrayName = ["Element1", "Element2", "Element3"];
    ArrayName.splice(Index of Element to ADD before {In case of Addition}/ DELELTE start with, Number of Elements to be Deleted {in Case of Deleting Function}, New Items Value(s) to be Added before Determined Element -by Index- );
    console.log(arrayName);
    document.getElementById("HTMLTagName").innerHTML =
    "Array After Adding new Element :" + arrayName ;

      This Sub Function needs to Enter the Value to be added wether it was one Element or more.
