[Javascript onchange] Event:

=> [onchange] Concept: is an event occur when a change is occur on spcific element on page

=> [onchange] Property: It is a common Event which is being used much in Web applications as pragrammers

=> [onchange] Property: is Not Suitable for all elements of HtmL Tags but only following:
