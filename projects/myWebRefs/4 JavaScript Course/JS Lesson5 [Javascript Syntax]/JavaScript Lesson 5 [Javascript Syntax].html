JavaScript [Comments]

Comments in JS Coding Language:

=> [JS] other programing languages has their comments Code, and it is being Written as following:

// Comment Text

=> [JS Comments ] can be written before the Code or After the code, but not iniside the code itself.

Comment Code Examples:

Following codes is about using Comments Js Coding in case Before the JS code:

// This is Comment Before This Code from JS Comment Code ...
document.getElementById("test").innerHTML="Hello Form JS File";

Following codes is about using Comments Js Coding in case Beside (after) the JS code:

document.getElementById("test").innerHTML="Hello Form JS File"; // This is Comment Beside (after) This Code from JS Comment Code ...
JavaScript [Variables]

Main Variables Types of JS Coding Language:

=> [JS] as any other progring Language has Variables types which considered as Data Containers of specific Type of Data.

=> [JS] is a [Sensitive Case] language, and commonly use [Camel Letter Case] in it's coding Syntax.

=> [Cammel Case] which is used by JS Coding Source is recommended to be used when declerating new Variables in JS page source, to be available in smart Editor Manual for Dev user.

=> This Lesson Will consern on the Variables Syntax in [JS] instead of Variables Types itself.

=> [JS] as any other programing language has main Data type of Variables, such as following:

  • String Variable [Text Data]
  • Number Variable [Numeric Data]

JavaScript [Conditional Statements]

=> [JS] as any other programing language has Conditional Statements Codes.

=> This Lesson Will concern on the Conditional Statement JS Syntax Code with JS instead of Conditional Statements concept itself .

=> [JS Conditional Statement] has commonly Types just like other programing languages as Examples (If-else Statement) & (If Statement).

=> [JS Conditional Statement] (If Statement Syntax) as following:

If (Condition/s) {
    [Statement/s]
}

[If Statement] JS Code Examples:

Following codes is about using [if-else Statement] (Positive Condition) with Chrome Console Example 1:

/*global console*/
// Decleration of [Number Vars]..
var
    x = 10,
    y = 10,
    z = 100";

// If Statement Function ..
if (x===y) {
    console.log("Equiled Vars");
} else {
    console.log("UnEquiled Vars");
}

Following codes is about using [if-else Statement] (Negative Condition) with Chrome Console Example 2:

/*global console*/
// Decleration of [Number Vars]..
var
    x = 10,
    y = 20,
    z = 100";

// If Statement Function ...
if (x!==y) {
    console.log("UnEquiled Vars");
} else {
    console.log("Equiled Vars");
}
