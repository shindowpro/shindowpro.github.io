  • UTC Date Extracion Mode :

    • This is The Date & Time to UTC Date of Certian Time-Date variable

=> JavaScript [ Date - Function part 2 ] : In JS programing language there are (9) {Main} functions to (set) Date and time to certian Time-Date Variable as Following :

  1. setDate(): set the Day Index Number of the current Month of Give Date Function (1-31)
  2. setFullYear(): set the Year Number Function (Long Number Form) of Given Date
  3. setHours(): set the Hour Number of given Time Function
  4. setMinutes(): set the Hour Number of given Time Function
  5. setSeconds(): set the Seconds Number of given Time Function
  6. setMilliSeconds(): set the MilliSeconds Number of given Time Function
