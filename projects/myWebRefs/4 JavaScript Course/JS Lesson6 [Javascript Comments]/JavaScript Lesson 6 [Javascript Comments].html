JavaScript [Common Usages & Benefits]

Comments usaualy has many usages and Benefits other than it's main purpose as folowing :

  • Commenting about certain Code (Single Code) or More than one codes (Multiple Lines) to help founder dev or other divs to undestand the main purpose of the following code(s) or the main file objectives or Functions.
  • Providing a legal method to Cancel the desired code(s) working for testing purposes or debugging.
JavaScript [Comments Types]

Comments in JS Language has (2) main Types:

  1. Commenting [Single line Code]

    2. => [JS Single line Code] Can be established as new Comment, or a legal method to Convert a single line existed code to a Single line Comment.

  2. Commenting [Multiple lines Code]

    3. => [JS Multiple line Code] Can be established as new group of Comment lines, or a legal method to Convert group lines of existed codea to a Multiple lines Comment.

Comment Code Examples:

A/ Following codes is an Examples of [ Single Line Comment ] before and beside Main Code:

// This is Comment Before This Code from JS Comment Code ...
document.getElementById("test").innerHTML="Hello Form JS File";
document.getElementById("test").innerHTML="Hello Form JS File"; // This is Comment Beside (after) This Code from JS Comment Code ...
