[Javascript Loop Control Tools ] Concept:

=> [ Loop Control Tools ] : This Lesson Will Present an important, essential and Advanced Loop Control Tools Concept of Any Loop Statements, Which gives Developer more Programing Advantages on Loop Statement controls.

=> [ Loop Control Tools ] : There are (3) main Tools Loop Control, as following :

  • => [ Break ] loop Control Tool : This Tool will Stop Excecuting the whole Loop Statemenst

  • => [ Continue ] loop Control Tool : This Tool will Stop the Current Condition value from Excecuting the current Loop Statement

  • => [ Label ] loop Control Tool : This Tool will create a Programing Name(ID) to the Loop Statemenst, which can be used in other loop Control Tools (Break, Continue)

[ Javascript Loop Control Tools ] Importance & Benefits:

=> [ Javascript Loop Control Tools ] : Is very essential and useful specially in more advanced, Integreated Dynamic Loop Functions in Web Desingning and Applications, by providing the Integration features with other Programing functions such as [Functions, Tools and Elements Creators], which can provide following Programing features:

  • High accuracy in implementing Codes within loop statements.
  • Much more easier in implementing Codes by and inside loop statements.
  • Providing more Featured Controling Tools in implementing Codes in loop statements.

[ Loop - Control ] Mains Syntax & Methods:

=> There will be (3) Following Syntax about (break, continue, label) Loop controling Tools with For loop Statement Type as example:

  1. [Break] Control Loop Tool Syntax

    2. => Stop excecting the Whole Loop Statement in Case of certain Condition example:

    // [Break] Loop Statement Syntax:

       varName;

       for (var = value; condition; FinalExpression) {

         // Continue executing the loop by Internal Condition inside the Loop Statement:
         if (var === Certain value) {
           break;
         }
       }

    =======================================================




  2. [Continue] Control Loop Tool Syntax

    3. => Stop excecting certain value of Loop Statement in Case of certain Condition example:

       varName;

       for (var = value; condition; FinalExpression) {

         // Breaking executing the loop by Internal Condition inside the Loop Statement:
         if (var === Certain value) {
           continue;
         }
       }
    =======================================================
    =======================================================




  3. [Label] Control Loop Tool Syntax

    4. => Setting Labels to (2) Integreated Loop Statements example:

       varName1, varName2;

       // Label of the First Main (Parent) Loop Statement:
       MainLoop:
       for (var1 = value; condition; FinalExpression) {

         // Label of the Second inside (Child) Loop Statement:
         ChildLoop:
         for (var1 = value; condition; FinalExpression) {

           Codes & Scripts ;
         }
       }
