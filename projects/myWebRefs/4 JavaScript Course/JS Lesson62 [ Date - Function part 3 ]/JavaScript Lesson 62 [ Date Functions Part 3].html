JavaScript [ Date - Function part 3 (now-parse-Conv) ]] Concept:

=> JavaScript [ Date - Function part 3 (now-parse-Conv) ] : This Lesson Will Present a Public Concept of Date Specific Function of getting Total (MilliSeconds) of Default-Certain Date-Time from (1970 - Now/Certain date-time) & Date-Time Fomrats convetring Mehods.

=> JavaScript [ Date - Function part 3 (now-parse-Conv) ] : There are (2) Date-Time [Now Functions] which are :

  1. [Date.now();] => Will Get the total Time by Milliseconds from (1970 - now by Default date Value)

    2. This Now Function will Get Total Current Milliseconds from (1970) until Default Date-Time.

  2. [Date.parse("Given Date String Value");]: Will Get the total Time by Milliseconds from (1970 - now by given date Value)

    3. This Now Function will Get Total Current Milliseconds from (1970) until Cutomizerd given date-time.

=> JavaScript [ Date - Function part 3 (now-parse-Conv) ] : There are (3) Date-Time [Foramt Converting ] which are :

  1. [DateVarible.toISOString();] => Will Convert the Default String Format of (given/default) date-time to ISO Format
  2. [DateVarible.toDateString();] => Will Convert the Default String Format of (given/default) date-time to Only Date Format
  3. [DateVarible.toTimeString();] => Will Convert the Default String Format of (given/default) date-time to Only Time Format
