[Javascript Array practise] Mains Methods:

=>[Array practise Functions (Methods & syntaxs] are as following :

  1. => A) [Array Checking] Method :

    2. => It is about cheking the varable's Datatype which is Array or Not

    => [Array Checking] : Is Critical in any programing Process to ensure of the Scripts Correction and avoid the Exestense of any Bugs

    => [Array Checking] Method Syntax as following:

    // Array Checking Function:
    if (Array.isArray(ArrayName)) {
       console.log("Yes the Variable is Array");
    } else {
       console.log("No the Variable is not Array");
    }

  2. => Array (length Set - Get) Method ]

    3. => It is about cheking the varable's Datatype which is Array or Not

    => [Array Checking] : Is Critical in any programing Process to ensure of the Scripts Correction and avoid the Exestense of any Bugs

    // Array Length Get Function:
    console.log(ArrayName.length);
    // Array Length Set Function:
    ArrayName.length = Value;

  3. => Array (Convert to String [toString - toLocaleString]) Methods ]

    4. => It is about Convering the Array Elements to one String (Texutal value)

    => [Array Convert to String Function has (2) Methods] :

    1. [toString] : Which is to Defualt Conversion of the Array Elements one String Value.
    2. [toLocaleString] : Which is to Pc Locale Setting Conversion of the Array Elements one String Value.

    => [Array Checking] Method Syntax as following:

    // Array Convert [.toString] Function:
    console.log(ArrayName.toString());
    // Array Convert [.toLocaleString] Function:
    console.log(ArrayName.toLocaleString());

  4. => Array (Join - Customized Join) Methods

    5. => It is about to merging Between Array's Elements

    => [Array join Function has (2) Methods] :

    1. [join();] : Which is to Defualt Merging with out any special or customized seperator sign.
    2. [join("Special Sign");] : Which is Customized Merging with special or specific seperator sign.

    => [Array Checking] Method Syntax as following:

    // Array Join [.join();] Function:
    console.log(ArrayName.join());
    // Array Customized Join [.join("Special Sign");] Function:
    console.log(ArrayName.join("Special Sign"));

  5. => Array Adding New element(s) TO [The First - The last - Specific location]) Methods

    6. => It is about to Adding new Element to Array .

    => [Array Adding Function has (5) Methods] :

    1. [Adding by (ArrayLength)] : Which is to Add new Element to the Array using the Array Length.
    2. [Adding by (New Elemenet Index)] : Which is to Add new Element to the Array using the Array index of Next Array Element .
    3. [Adding to the First of Array by (unshift) Function] : Which is to Add new Element to first poistion of the Array using the unshift Function .
    4. [Adding to the last of Array by (push) Function ] : Which is to Add new Element to last position of the Array using the (push) Function.
    5. [Adding to specific Location by (splice) Fucntion] : Which is to Add new Element to spcific position of the Array using the Splice Function.

    => [Array Adding] Method Syntax as following:

    // 1) Adding by (ArrayLength) Function:
    ArrayName[ArrayLenght] = new Value;
    // 2) Adding by (New Elemenet Index) Function:
    ArrayName[Next Index] = new Value;
    // 3) Adding the First of Array by (unshift) Function:
    ArrayName.unshift("New Value");
    // 4) Adding the Last of Array by (push) Function:
    ArrayName.push("New Value");
    // 5) Adding to specific Location by (splice) Fucntion:
    ArrayName.splice(Starting After Index, 0, "New Value1", "New Value2", ...);

  6. => Array Deleting element(s) from [The First - The last - Specific location]) Methods

    7. => It is about to Deleting new Element to Array .

    => [Array Deleting Function has (3) Methods] :

    1. [Deleting to the First of Array by (shift) Function] : Which is to Delete new Element to first position of the Array using the shift Function .
    2. [Deleting to the last of Array by (pop) Function ] : Which is to Delete new Element to last position of the Array using the (pop) Function.
    3. [Deleting to specific Location by (splice) Fucntion] : Which is to Delete new Element to spcific position of the Array using the Splice Function.

    => [Array Deleting] Method Syntax as following:

    // 1) Deleting the First of Array by (shift) Function:
    ArrayName.unshift("New Value");
    // 2) Deleting the Last of Array by (pop) Function:
    ArrayName.push("New Value");
    // 3) Deleting specific Location by (splice) Fucntion:
    ArrayName.splice(Starting Deleting by Index, Number of Elements to delete);

  7. => Array (Sorting Array Elements [Alphabatically] & [Reversely]) Methods

    8. => It is about to Sort Array's Element .

    => [Array Sorting Function has (2) Methods] :

    1. [Sorting Array's Elements Alphabaticaly (sort) Function] : Which is to Sort Element values Alphabaticaly by (sort) Function .
    2. [Sorting Array's Elements Reversaly (reverse) Function] : Which is to Sort Array's Elements Reversely (regerdless of the Elemenets Values) by (reverse) Function .

    => [Array Sorting] Methods Syntax as following:

    // 1) Sorting Array's Elements Alphabaticaly (sort) Function :
    ArrayName.sort();
    // 2) Sorting Array's Elements Reversely (reverse) Function:
    ArrayName.reverse();

  8. => Array Array (Slicing Arrays [From Specific Start and Specific End - Form Specific Start Without Specific End]) Methods ] :

    9. => It is about to Cutting or Slicing the Array to several Partial Arrays, starting with sepecific till the Last of Array or specific point.

    => [Array Slicing Function has (2) Methods] :

    1. [Slicing Array [From Specific Start and Specific End] Function] : Which is to Slice Array from Specific Start Index to Specific End Index [slice(#, #)] Function .
    2. [Slicing Array [From Specific Start Without Specific End] Function] : Which is to Slice Array from Specific Start Index to Specific End Index [slice(#)] Function .
    3. [Slicing Array [of each Type] Function by Negative Index] : Which is to Slice Array using the Negative Index of the Array's Element wether by using [slice(#, #)] or [slice(#)] {by the Main Direction or slice - from Start of Array to the End Of Array -} .

    => [Array Slicing] Methods Syntax as following:

    // 1) Slicing Array [From Specific Start and Specific End] Function:
    ArrayName.slice(Start Index, End Index);
    // 2)Slicing Array [From Specific Start Without Specific End] Function:
    ArrayName.slice(Start Index);
    // 3) Slicing Array [of each Type] Function by Negative Index:
    ArrayName.slice(Negative Start Index, Negative End Index);

  9. => Array (Concatenation Arrays [Concatenate (2) Arrays - Concatenate more than (2) Arrays ] ) Methods ] :

    10. => It is about to Merging between Different or Mulitple Arrays to one Big Array, wether there were (2) Arrays or More .

    => [Array Concatenating] Method Syntax as following:

    // 1) Concatenating between (2) Arrays Function:
    ArrayName1.concate(ArrayName2);
    // 2) Concatenating between more (2) Arrays Function:
    ArrayName1.concate(ArrayName2, ArrayName3);

  10. => Array (Searching Arrays [ indexOf() - lastIndexOf() ] ) Methods ] :

    11. => It is about to searching about or Finding the Index of Specific Element in Array using it's Value , wether by Searching from the Start to End or From the End To Start .

    => [Searching Array Funcion] : can be included the option of Starting Searching Point or not (Starting Searching Point is optional) .

    => [Searching Array Funcion] : has (2) Methods :

    1. [indexOf();] : Searching About the Element Index from First of Array to the End of Array
    2. [lastIndexOf();] : Searching About the Element Index from End of Array to the Start of Array

    => [Array Search] Method Syntax as following:

    // A) Searching from the Start to End Function:
    // 1) Without Starting point :
    ArrayName.indexOf("Element Value");
    // 2) With Starting point :
    ArrayName.indexOf("Element Value", Index of Searching Start);
    // ---------------------------------------------------------------- // B) Searching from the End to Start Function:
    // 1) Without Starting point :
    ArrayName.lastIndexOf("Element Value");
    // 2) With Starting point :
    ArrayName.lastIndexOf("Element Value", Index of Searching Start);
