[ Javascript String Methods ] Mains Methods:

  • { StringName.indexOf("Searching Text", Starting Index); } Method :

    • Which Get the Index of matched Index of Inserted Texual Value inside the Function.

    This Method Will Search [From Start to End] for the INDEX or inserted Texual Value .

    This Method is Letter Case Sensitive to the Inserted Searching Value .

    Searching process Will Start from the Inserted Index, but if there was nothing inserted in Index Starting Position, the Default will be {0}.

  • { StringName.lastIndexOf("Searching Text", Starting Index); } Method :

    • Which Get the Index of matched Index of Inserted Texual Value inside the Function.

    This Method Will Search [End to Start] for the INDEX or inserted Texual Value.

    This Method is Letter Case Sensitive to the Inserted Searching Value .

    Searching process Will Start from the Inserted Index, but if there was nothing inserted in Index Starting Position, the Default will be {0}.

  • {StringVariableName.search("Searching Word - Included RE") } Method :

    • Which Get the Index of matched Index of Inserted Texual Value inside the Function.

    This Method Will Search From Start to End for the INDEX or inserted Texual Value .

    This Method is Letter Case Sensitive to the Inserted Searching Value .

    This Method has no Index to Enter, but instead, it provide the ablility to use the Regular Expression Rules {Will Be Taken in Later Lessons}.

    This Method we Shall not use the Double Quoting signs "" IN CASE of using [RE], but in Case of not Using [RE] we can use the Double Quoting Signs .
