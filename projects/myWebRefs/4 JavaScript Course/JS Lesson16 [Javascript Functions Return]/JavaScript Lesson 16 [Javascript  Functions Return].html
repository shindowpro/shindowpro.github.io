  1. [Function Return] Can be Called or Invoked or imported in another Scope of other Function, Variable Value or even a HTML Element.
  2. [Function Return] Multiple Specific Applicaitons basic on Front web by JS.
  3. [Function Return] Critical and Essential for many programing Function Return and Methods.
  4. [Function Return] Can hold several Variables inside (Parameter) [Next Lesson].
  5. [Function Return] It is Critical to Improve Page Loading Speed & Performance.
  6. [Function Return] It can Save time by prevent repeating calling codes which save time and size.
  7. [Function Return] It's Value can be as Text, Logic, Array, number or even a Result of Arithmetic Calculation.
