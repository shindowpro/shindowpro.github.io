JavaScript [ Numbers - Intro]] Concept:

=> JavaScript [ Numbers - Intro] : This Lesson Will Present a general Concept of Numbers in all P.languages and JS Specificlly .

=> JavaScript [ Numbers - Intro] : There are (4) Types of Numbers Formats in JS which are :

  1. [Floating Point Decimal Format] => This is the main Numeral Format that JS Dealing with Numbers, even it was Integer or Decimal.
  2. [Hexa Decimal Format] => This is one of the most common numeric Systems that are widely spread in all P.languages and JS as well, and deal with number by Hexal Base (6) [0x##]
  3. [Octal Decimal Format] => This is one of the most common numeric Systems that are widely spread in all P.languages and JS as well, and deal with number by Octal Base (8) [0##]
  4. [NAN] => This is one of known Script of Undifiened Numbers in JS that means that the Value is not in read as number, and resulting usualy when the resultof calculation is not supported as numeric value
  5. [Infinity Format] => This Numeric Format is resluting when dividing any number > Zero on Zero
  6. [-Infinity Format] => This Numeric Format is resluting when dividing any Negative number > Zero on Zero

=> JavaScript [ Numbers - Intro] : All previous Numeric will result ("Number") Type when we use [typeof ();] Code for Confirmation
