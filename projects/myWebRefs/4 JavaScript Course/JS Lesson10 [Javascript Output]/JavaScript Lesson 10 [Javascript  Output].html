Main Concept of Output :

=> [JS Output] as it in any programing Languages is the Function to display data by code.

=> [JS Output] There are many different Style to display the data in several Framework, using Javascript, using several way or Code for the same purpose (Output)

Output Main Types or Functions (Codes):

=> [JS Output] several Function to do the output function, such as:

  1. [alert() Function]:

    2. => [ alert() ] It is commonlly used in Displaying Messages to the user in sepereted message upon the top of Screen.

    => [ alert() ] can hold different types of messages such as :

    1. [Single Value(s)]:

      2. Such as String values or numbers or any other type of Data.

    2. [Variables Value(s)]:

      3. Variables of any datatype can be displayed on alert() function message .

    3. [Algorithms operations results]:

      4. Alert() can display the result of all types of Math operations, and even of numeric values.

    => [ alert() ] Code can be written Directly in page address bar to Excute a Javascript Code certain code with the certain page, as following :

    javascript:alert("Welcome to Javascript");

    => [ alert() ] Code can also be written Directly in Browser Console FrameWork Page to Excute a Javascript Code certain code with Console Testing FrameWork , as following :

    alert("Welcome to Javascript");
