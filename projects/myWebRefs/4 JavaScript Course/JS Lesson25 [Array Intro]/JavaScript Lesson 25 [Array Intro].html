[Javascript Array] Concept:

=> [Array] : Is a special Variable which is holding and Storing mulutiple Values of unified Data Type

=> [Array] : Is a very important Element in any Programing Language, and has the same Concept in any language with some differences in Syntax Methods

=> [Array] : Is dealing with it's elements by an Index, and JS is a (Zero Based Index Type), Which is meaning it Begain With (0) Number.

=> [Array] : Is has a Count or length for it's elements or Components.

[Javascript Array] Importance & Benefits:

=> [Array] : Is Essential in dealing with big amounts of variables of same Datatype.

=> [Array] : Is providing very useful benefits and features in [Adding, Editing, Deleting] Array Elements by merging it with some other JS Advanced Functions such as (Looping Statements) to make dealing with Arrays More Dynamically

[Javascript Array] Syntax Methods:

  1. => [JS Array Constractor Method] :

      2. => This Method is not Recommeded to use

    • [Constractor Method / Default Decleration]:

      • This Method Array Declaration as following:

      var arrayName = new Array();

      To Add Array's Elements Values (as Defualt Way) as following:

      arrayName[Element's Index Number] = Element's Value;

      To Type All Array's Elements Values (Defualt Way) in console as following:

      console.log(arrayName);

      To Type a specific one Array's Element Value in console as following:

      console.log(arrayName[Element's Index Number]);
    • [Constractor Method / Short Way] Array's Element Inserting :

      • In short Way we write Array Decleration Code in only one Line, but it is NOT useful when dealing with big Amount of Elements, and hard to read the very wide Code

      var arrayName = new Array("Element 1 Value", "Element 2 Value", "Element 3 Value");

      [Constractor Method / Short Way] Practical Example

      // Js Array Constractor Method / Short Way: var friends = new array("Shadi", "Sayed", "Mohammed", "Ahmed");

      // Typing All Array Elements in Console Frame Work:
      console.log(friends);

      // Typing Specific Array Element (by it's Index) in Console Frame Work:
      console.log(friends[0]);  //Typing Array's First Element
      console.log(friends[1]);  //Typing Array's Second Element
      console.log(friends[2]);  //Typing Array's Third Element
      console.log(friends[3]);  //Typing Array's Fourth Element
      console.log(friends[4]);  //Typing Array's Fifth Element
    • [Constractor Method / Long Way] Array's Element Inserting :

      • In Long Way we write Array Decleration Code in multiple lines by writing each Element in a line Accendenly, So it can be long Code Vertically but to read whole Code

      var arrayName = new Array(
          "Element 1 Value",
          "Element 2 Value",
          "Element 3 Value"
      );

      [Constractor Method / Short Way] Practical Example

      // Js Array Constractor Method / Long Way: var friends = new array(
          "Shadi",
          "Sayed",
          "Mohammed",
          "Ahmed"
        );

      // Typing All Array Elements in Console Frame Work:
      console.log(friends);

      // Typing Specific Array Element (by it's Index) in Console Frame Work:
      console.log(friends[0]);   //Typing Array's First Element
      console.log(friends[1]);   //Typing Array's Second Element
      console.log(friends[2]);   //Typing Array's Third Element
      console.log(friends[3]);   //Typing Array's Fourth Element
      console.log(friends[4]);   //Typing Array's Fifth Element
