=> Differences between [ (For) And (While - Do While)] Loop Statement : The Both For and While Loop Statements are similer in Their Main Functions but different in their Syntax's componentes as Follwoing :

  • Initialization Variable is before the While Loop Statement, but it is with in the For Loop Syntax (Except in Short Form of [For] Loop Statement that the Initialization Variable can be before the For Statement syntax too)
  • Final Expression is inside the [While] loop Statement, but it is the same [For] Loop Syntax (Except in Short Form of [For] Loop Statement that the Final Expression can be before the For Statement syntax too) .
  • Condition Statement are with in s the Syntax of both [While] and [For] Loop Statement (Except in Short Form of [For] Loop Statement that the Condition be inside the For Statement with opposite Logic syntax).

=> Differences between [ While - Do While ] Loop Types : The Both For and While Loop Statements are similer in Their Main Functions but different in their Syntax's componentes as Follwoing :

  • [While] : Condition Statement is First then the Scripts , so the Test Condition Function will be Done First before Entering inside the While Loop Statement and Excectuting
  • [Do - While] : Condition Statement is in the Last of Scripts , so the Entering inside the While Loop Statement and Excectuting will be Done First , then the Test Condition Function will be Done at last
