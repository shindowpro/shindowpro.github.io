JavaScript [Variables Syntax & Rules]

[JS Variables] as it in any other programing language has certain rules of code Syntax which is defferent in details but with the same main Concept

=> [JS Variables] Syntax as following :

  • [JS Variables Name] Must start Var Reserved word or the Editor will generate an Error .
    • price = 100;   =>> Wrong Code X
  • [JS Variables Name] Can Not Start with number Ever.
    • var 1price = 100;   =>> Wrong Code X
  • [JS Variables Name] Can Not be as any of [JS] Reserved Word.
    • var var = 100;   =>> Wrong Code X
    var function = 100;   =>> Wrong Code X
    var return = 100;   =>> Wrong Code X
  • [JS Variables] Must have a Space between each of Variable Name and Assigining Mark (=) & space between Assigning Mark (=) and Variable Value.
    • var price=100;   =>> Wrong Code X
  • [JS Variables] Can be started with Small Letter or Capital Letter.
    • var price = 100;   =>> Right Code
    var Price = 100;   =>> Right Code
  • [JS Variables] it recommended to write the variable in [Cammel Case] .
    • var myPrice = 100;   =>> Right Code
    var mySmallDisc = 100;   =>> Right Code
  • [JS Variables] Can Started with ($) Dollar Sign or (_) Underscore Sign.
    • var $myPrice = 100;   =>> Right Code
    var _SmallDisc = 100;   =>> Right Code
  • [JS Variables] Must have a value after Assigning Mark or the Editor Will Generate an error in Work Frame work.
    • var $myPrice;   =>> Wrong Code X
    var _SmallDisc;   =>> Wrong Code X
