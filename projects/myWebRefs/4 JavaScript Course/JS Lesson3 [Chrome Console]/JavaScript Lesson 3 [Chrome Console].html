Dealing with Chrome Console

Google Chrome Console Identification:

=> Google Chrome Console is Built-In Tool inside Chrome Browser, and one of Google Chrome Tabs.

How to Open Chrome Console:

=> [Chrome Console] has two ways to open:

  1. R.Click => Ispect Element => Console
  2. Ctrl + Shift + J

Benefits and Importance of Chrome Console:

=> [Chrome Console] is a very helpful Code Debugger And Analitic Tool For Testing JS, HTLM, CSS Codes before writing them in HTML Inner Page File.

=> [Chrome Console] can deal with any JS Code even doing separated Arthmetic Operations , or changing Elements in the html Page.

=> [Chrome Console] can help to test the existance of and file imported by code.
