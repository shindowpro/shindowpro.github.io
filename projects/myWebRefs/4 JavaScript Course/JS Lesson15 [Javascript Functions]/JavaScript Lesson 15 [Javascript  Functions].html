Main Concept of Functions :

=> [JS Functions] as it in any programing Languages Functions are an Essential component of any programing Langauages as overall.

=> [JS Function] is a Sequence of order(s) and codes which can be called by main Js File or HTML File (Script Tag) directly(by main JS code) or indirectly (using Element Events)

Functions Importance, Advantages and Usages:

=> [JS Functions] can provides several Features and Advantages in programing purposes such as :

  1. Programing Events functions upon certain conditions and Events in HTML file (Tags and elements) to implement certian Codes or Procedures.
  2. Multiple Specific Applicaitons basic on Front web by JS.
  3. Critical and Essential for many programing Functions and Methods.
  4. Can be seperated progarming Container for certain group of Codes and Procedures which can be Called or Invoke inside other HTML Element's events, Functions, or other html Pages.
  5. It is Critical to Improve Page Loading Speed & Performance.
  6. It can Save time by prevent repeating calling codes which save time and size.

  4. [ Functions with Variabels]:

    5. => Functions can Contain and deal with Variable with different types of DataTypes but with in it's scope, but not outside of it's scope.

    => Functions will Not responde to Outsider Variables will, until they be declared before and outside of Function Scope.

    => Responding of Function to each Internal and External Variables depends on the location of Variable Declaration Code, and if it was (Before-After), (Inside-Outside) the Function Scope.

    => Function as Default respond to Internal Vars , but to respond and deal with the External Vars , Vars Declearation Line Must be before the Function and Not After.
