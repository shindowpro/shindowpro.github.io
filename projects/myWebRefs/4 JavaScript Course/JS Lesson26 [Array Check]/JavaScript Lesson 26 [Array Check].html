[Javascript Array Check] Syntax Methods:

  1. => [JS Array First Method (Better Way)] :

    2. => This Method is Recommended to use

    This Method Array Check will be as following:

    if (Array.isArray(arrayName)) {
      console.log("Confirmation Scentense");
    } else {
      console.log("Dis Confirmation Scentense");
    }

    Following Practical Example:

    // Js (2) Variables will be Tested one is ordinary Variable and the other is Array: var friend = "Shadi",
      friends = ["Shadi", "Sayed", "Mohammed", "Ahmed"];

    // Testing First Variable in Console Frame Work by [First Method] :
    if (Array.isArray(friend)) {
      console.log("This is Array");
    } else
      console.log("This is Not Array");
    }

    // Testing Second Variable in Console Frame Work by [First Method] :
    if (Array.isArray(friend)) {
      console.log("This is Array");
    } else {
      console.log("This is Not Array");
    }

  2. => [JS Array Second Method (Other Way)] :

    3. => This Method is less Recommended to use

    This Method Array Check will be as following:

    if (arrayName.Constractor === Array) {
      console.log("Confirmation Scentense");
    } else {
      console.log("Dis Confirmation Scentense");
    }

    Following Practical Example on Second Method:

    // Js (2) Variables will be Tested, one is ordinary Variable and the other is Array: var friend = "Shadi",
      friends = ["Shadi", "Sayed", "Mohammed", "Ahmed"];

    // Testing First Variable in Console Frame Work by [Second Method] :
    if (friend.Constractor === Array) {
      console.log("This is Array");
    } else
      console.log("This is Not Array");
    }

    // Testing Second Variable in Console Frame Work by [First Method] :
    if (friends.Constractor === Array) {
      console.log("This is Array");
    } else {
      console.log("This is Not Array");
    }
