=> JS OOP [Static Properites ]: Inheritance in JS an Class will provide the advanced Control to have a Copy of Specific Propterites and Methods from Parent Class in an new child Class .

JS OOP [Class Inheritance ] Syntax & Methods:

=> JS OOP Defining-Declaring [Class] Conceptual Syntax as following :

// A) JS OOP Defining-Declaring [Parent Class] Conceptual Syntax as following :

class ParentClass {
constructor(Par1, Par2, Par3) {
this.par1 = par1;
this.par2 = par2;
this.par3 = par3;
}

ParentMeth() {
return 'A message from Parent Method';
}

}
//===========================================================

//===========================================================

// B) JS OOP Defining-Declaring [Child Class] Conceptual Syntax as following :

// Main Inheritance Code :
class ChildClass extends ParentClass {
constructor(Par1, Par2, Par3) {
// Inherating Parent class's Parameters from Child Class :
super(Par1, Par2, Par3)
}

// Defining & Overriding the {Parent Method} from (Parent Class) in the (Child Class):
Parentmeth() {
return 'A message from Child Method';
}

}
//===========================================================

// C) Extracting a new one Object from [Child Class] :

// Declaring new Object (obj1) from the [Child Class]:
let obj1 = new ChildClass("par1 value", "par2 value", "par3 value");

//===========================================================

// D) Extracting a new one Object from [Parent Class] :

// Declaring new Object (obj1) from the [Parent Class]:
let obj1 = new ParentClass("par1 value", "par2 value", "par3 value");

//===========================================================

// E) Accessing Inherited (Methods + propeties) from {Parent Class} by [Child Class] {After OverWriting}:
// E1) Accessing inherited {Parent Method} After OverWriting :
console.log(ChildClass.ParentMeth());

// E2) Accessing inherited Parmeters & Variables :
console.log(ChildClass.par1);

console.log(ChildClass.par2);

console.log(ChildClass.par3);

//===========================================================

// F) Accessing Inherited (Methods + propeties) from {Parent Class} by [Extracted Object] {After OverWriting}:
//F1) Accessing inherited {Parent Method} After OverWriting In Child Class :
console.log(obj1.ParentMeth());

// F2) Accessing inherited Parmeters & Variables :
console.log(obj1.par1);

console.log(obj1.par2);

console.log(obj1.par3);

//===========================================================

//===========================================================
