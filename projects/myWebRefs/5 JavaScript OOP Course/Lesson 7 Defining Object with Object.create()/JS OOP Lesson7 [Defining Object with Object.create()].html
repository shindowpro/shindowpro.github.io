JS OOP [Defining Object with Object.create()] Importance & Benefits:

[Defining Object with Object.create() Method] gives us the Following Features :

  • The ability to Create Object with multiple Property for each property as an internal Sub Object, which called [Listed Object].
  • The ability to Create Internal Method(s)/Function(s) inside each Listed Object(s).
  • The abililty to Make Multiple Processes and complicated Calculations on Listed Objects properties Values + Main Properties Values through Internal Objects Methods or Functions OR Main Object Method(s)/ Function(s).
  • The Ability to Call any of Internal Listed Objects Properties Values through the Objects Internal Functions or Methods or Main Object Functions/Method(s) OR other Objects's Methods or Outer Seperated Functions.
  • The Ability to write a short Form of Funtion/Method Defining or Declaration Script, by using [Arrow Function] Method, wether for the Main Object Method OR Internal Listed Object Property(s).
