JS OOP [Defining Object With Object Literals Part 1] Concept:

=> JS OOP [Defining Object With Object Literals Part 1] : This Lesson Will Present a Public Concept of Defining Object by using The most common method [Literaling] + Accessing Object's Properties Values + Object's Methods Returns.

=> JS OOP [Defining Object With Object Literals Part 1] : Any Object has (2) main Componets which are :

  1. Properties :

    2. Which are the Objects Keys and it's Values.


  2. Methods (Functions) :

    3. Which Is/Are the Internal functions that will be Executed with in the Object internal Properties Keys and it's Values / or with in other Outer variables, once it being called, and gives certain {Return} value.

=> JS OOP [Defining Object With Object Literals Part 1] : Object Accessing Properties Values + Methods Return Value can be executed by (2) Methods mainly :

  1. {Dot Notation} :

    2. Calling the Object Property Value / Method Return by using => {Object Name followed by { . } sign and using Object Property / Method Name()}.


  2. {Brackets Notation} :

    3. Calling the Object Property Value / Method Return by using => {Object Name followed by { [] } sign and "Object Property" / Method Name() Inside Square Brackets}.

JS OOP [Defining Object With Object Literals Part 1] Mains Syntax & Methods:

=> JS OOP Defining Object [Literals] Method + Accessing it's Properties Values and Methods/Functions as following :

// A) [ Defining and Declaration of Object by {Literals} Method ] Syntax :

let objName ={
Properties [Keys + Values]
Method(s)() [MethodName + Return]
};
=======================================================

// B) [ Defining of Object and it's Properties (Keys + Values)] Syntax :

let objName ={
Key1Name: "Value",
Key2Name: "Value",
Key3Name: "Value",
};
=======================================================

// C) [ Defining of Object and it's Method(s)] Syntax :

let objName ={

// Defining Internal (Method/Function) inside the Object [by the Object
properites values as Methods variables/Parameters]:

Method1Name: function(){
return'Text: ${objName.Key1Name} ${objName.Key2Name}';
},
};
=======================================================

// D) [ Calling of Object Properites Values + Methods] Syntax :

//1) Calling (Object's Properites) values to Print it in Console:

a) Calling property by [Dot Notation] Method:
console.log(objName.Key1Name);

b) Calling property by [Brackets Notation] Method:
console.log(objName["Key1Name"]);

c) Printing Type of Property Value in console:
console.log(typeof objName.Key1Name]);

=======================================================

//2) Calling (Object's Method) to Execute and Print it's Return in Console:

//a) Calling Object Method/Function by [Dot Notation] Method
& print it's Return value in console:
console.log(objName.MethodName());

//b) Printing Type of Property Value in console:
console.log(typeof objName.MethodName());

=======================================================

// E) [ Final Form of Object and All of it's elements (Properites Values + Methods)] Syntax :

let objName= {

keyName1: "Value",
keyName2: "Value",
keyName3: "Value",
Method1Name: function() {
return'text';
},

Method2Name: function() {
return'text : ' + objName.key1Name;
},

Method3Name: function() {
return'text' + objName["key2Name"] + ;
},

Method4Name: function() {
return'text' + ${objName.key1Name} ${objName["key2Name"]} ;
},
};
