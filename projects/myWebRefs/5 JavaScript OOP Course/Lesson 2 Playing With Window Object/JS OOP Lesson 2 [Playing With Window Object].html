JS OOP [ Playing with window Object] Practical Examples on [window Object] (Using Browser Console):

  1. [ Extracting (Location) Property form [window] by {Notication Method} ] :

    2. 1) Extracting window Location property properites :

    > window.location

    < Will Extract all location Property Properties such as follwoing:


    < host: "elzero.org"


    < hostname: "elzero.org"


    < href: "https://elzero.org"


    < protocol: "https:"


    ========================================================================

    2) [ Extracting (Location) Property form [window] {Brackets Method} ] :

    > window["location"]

    < Will Extract all location Property Properties


    ========================================================================

    3) Getting the Data type of location property & and its inner properties :

    > typeof(window)

    < Will type [Object Data type]



    > typeof(window.location)

    < Will type [Object Data type]


    ========================================================================
  2. 4) [ Extracting (Location internal Properties) Values ] :

    3. Extracting window Location host property :

    > window.location.host

    < Will Extract location host Property value


    > window.location.hostname

    < Will Extract location host Name Property value


    > window.location.protocol

    < Will Extract location protocol Property value


    ========================================================================
    ========================================================================
  3. [ Executing some of Window Object Method ] :

    4. Executing window Reload Method :

    > window.reload()

    < Will Execute the reload Method on the current Web page


    ========================================================================

    > window.replace("https://google.com")

    < Will Execute the Replacement Method on the current Web page Host and Run the new host entered
