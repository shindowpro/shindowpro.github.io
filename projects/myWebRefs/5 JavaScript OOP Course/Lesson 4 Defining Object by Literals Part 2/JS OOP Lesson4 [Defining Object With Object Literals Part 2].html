JS OOP [Defining Object With Object Literals Part 2] Concept:

=> JS OOP [Defining Object With Object Literals (Listed Object + Arrow Function) Part 2] : This Lesson Will Present a advanced Concept of Defining Object by using The most common method [Literaling] + additional Features (Listed Object + Arrow Function).

=> JS OOP [Defining Object With Object Literals Part 2] : Additional Features are :

  1. [Listed Object] Property :

    2. This is an Internal Object property inside the main Object, which contain several properties and even Methods.


    The value of listed object property can be accessed through the Default Method/Ways of the Object Property [Dot Notation, Brackets Notation].


  2. [Arrow Functions] Method :

    3. This is an advanced Shortcut Object's Method/Function Declaration Method.

    This is an advanced Method can be executed to each the main Object Method & the Internal Method(s)/Function(s) of Listed Object Property(s)

JS OOP [Defining Object With Object Literals Part 2] Importance & Benefits:

[Listed Object(s) + Arrow Function Method] gives us the Following Features :

  • The ability to Create Object with multiple Property for each property as an internal Sub Object, which called [Listed Object].
  • The ability to Create Internal Method(s)/Function(s) inside each Listed Object(s).
  • The abililty to Make Multiple Processes and complicated Calculations on Listed Objects properties Values + Main Properties Values through Internal Objects Methods or Functions OR Main Object Method(s)/ Function(s).
  • The Ablility to Call any of Internal Listed Objects Properties Values through the Objects Internal Functions or Methods or Main Object Functions/Method(s) OR other Objects's Methods or Outer Seperated Functions.
  • The Ablility to write a short Form of Funtion/Method Defining or Declaration Script, by using [Arrow Function] Method, wether for the Main Object Method OR Internal Listed Object Property(s).

JS OOP [Defining Object With Object Literals Part 2 (Listed Objects) + (Arrow Funtion)] Mains Syntax & Methods:

=> JS OOP Defining Object [Literals] Method using (Listed Objects) & (Arrow Funtion) + Accessing it's Properties Values and Methods/Functions as following :

// A) [ Defining of {Listed Object} Property inside Main Object] Syntax :

let objName ={
Key1Name: "Value",
Key2Name: "Value",
Key3Names: {

Key3Name2: "Value",
Key3Name3: "Value",
},
};

// B) [ Defining of {Internal Function -Default + Arrow Method-} inside (Listed Object)] Syntax :

let objName ={

Key1Name: "Value",
Key2Name: "Value",
Key3Names: {
Key3Name1: "Value",
Key3Name2: "Value",
Key3Name3: "Value",

//[Internal Function with (Default Method)]:
MethodName1: function () {
return 'Text ${objName.Key3Names.Key3Name1}' ;
},

//[Internal Function with (Arrow Method) (Shortcut)]:
MethodName2: => () 'Text ${objName.Key3Names.Key3Name2}',
},
};

// E) [ Final Form of Object and All of it's elements (Listed Objects + Intreanl Arrow Functions)] Syntax :

let objName ={

Key1Name: "Value",

Key3Names: {

Key3Name2: "Value",
Key3Name3: "Value",

},

//[Internal Function with (Arrow Method) (Shortcut)]:
MethodName2: => () 'Text ${objName.Key3Names.Key3Name2}',
},

//[Main Object Function with Default Method]:

return 'Text ${objName.Key3Names.Key3Name2}' ;
},

//[Main Object Function with (Arrow Method) (Shortcut)]:
