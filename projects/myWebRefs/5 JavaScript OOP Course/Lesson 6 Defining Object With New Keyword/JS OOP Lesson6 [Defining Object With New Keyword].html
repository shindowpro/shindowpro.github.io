  • This Method is Critical in advaned Web Desining and Developing Web applications, which provides the ability to create mulutiple Functions in one object {Object Constructor}
  • This Method is the basic one in more advanced web-Designing and web-Development Technologies such as [DOM], [BOM], [J-Querry]
