JS OOP [Prototype Intro ] Concept:

=> JS OOP [Prototype Intro ]: This lessson will provide the Concept of Protptype which has been Created from the Normal Function or Constructor Function .

=> JS OOP [Prototype Intro ]: Any Prototype will be as an [[Object]] with name of it's Parent Function , which will inherate the properties & Methods Names of its Function as it's Propeties &n Methods 's names, and their Values & fuctionalities as it's values & fuctionalities .

=> JS OOP [Prototype Intro ]: Is a an Object which is being created from any Fucniton wether it was default Function or Constructor Function
and this Prototype will inherate all the Propterites and functions (Methods) thate has been Cereated From.

=> JS OOP [Prototype Intro ]: There are (3) Kinds of Prototype Function (based on it's Construtor Function that Created from ) as following :

  1. [Defualt Function Prototype] =<

    2. This kind of Prototype is Consructed from a Default Function

  2. [Constructor Function Prototype] =<

    3. This kind of Prototype is Consructed from a Constructor Function

  3. [Built in Constructor Function Prototype] =<

    4. This kind of Prototype is Consructed from a Built In Constructor Function

=> JS OOP [Prototype Intro ]: The Extracted Proptotype will inherate All the propteies and Method of the Function that created from (regardless of the type of Consturtor function - with some differences) as the following :

    1. Prototype from A [Constructor Function] :
      • Generated Prototype Object's (name) will be inherated as =< [Object]
      • Generated Prototype Object's (Properies Names) will be inherated as following =<
        • (Constructor) [which is the Prototype Constructor Function ]
        • Prototype's Constructor Function's Internal Properties's Names
        • Prototype's Constructor Function's Internal Methods's Names
      • Generated Prototype Object's (Properies Values) will be inherated as following =<
          • Prototype's Constructor Funciton [which is the the value of Prototype (constructor) Property's Name]
          • Prototype's Constructor Function's Internal Properties's Values
          • Prototype's Constructor Function's Internal Methods's Functions
      • Prototype from A [Built In Constructor Function] :
        • Generated Prototype Object's (name) will be inherated as =< [Data type of the Built In Function (String, Number, Boolean, Array) ]
        • Generated Prototype Object's (Properies Names) will be inherated as folllowing =<
          • (Constructor) [which is the Specified Data Type Built In Constructor Function ]
          • Prototype's Specified Built in Constructor function Data type's Internal Properties's Names
          • Prototype's Specified Built in Constructor function Data type's Internal Methods's Names
        • Generated Prototype Object's (Properies Values) will be inherated as folllowing =<
          • Prototype's Built in Constructor Funciton [which is the Specified Data Type Built In Constructor Main Function]
          • Prototype's Specified Built in Constructor function Data type's Internal Properties's Values
          • Prototype's Specified Built in Constructor function Data type's Internal Methods's Values
          • Prototype from A [Default Function] :
            • Generated Prototype Object's (name) will be inherated as =< [Data type of the Built In Function (String, Number, Boolean, Array) ]
            • Generated Prototype Object's (Properies Names) will be inherated as folllowing =<
              • (Constructor) [which is the Specified Data Type Built In Constructor Function ]
              • Prototype's Specified Built in Constructor function Data type's Internal Properties's Names
              • Prototype's Specified Built in Constructor function Data type's Internal Methods's Names
            • Generated Prototype Object's (Properies Values) will be inherated as folllowing =<
              • Prototype's Built in Constructor Funciton [which is the Specified Data Type Built In Constructor Main Function]
              • Prototype's Specified Built in Constructor function Data type's Internal Properties's Values
              • Prototype's Specified Built in Constructor function Data type's Internal Methods's Values

JS OOP [Prototype Intro ] Importance & Benefits:

[Prototype Intro ] gives us the Following Features :

  • More advanced tools to extract objects from Funcitons Wether it's type was [Constructor Function , Normal Function, Specified Data type Built In Constructor function ].
  • Ability to inherate all certain Function's properites and Methods = < as an [Object] Not as another Function .
