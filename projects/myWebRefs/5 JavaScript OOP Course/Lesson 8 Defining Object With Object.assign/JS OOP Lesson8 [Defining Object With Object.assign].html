JS OOP [Defining Object With Object.assign()] Concept:

=> JS OOP [Defining Object With Object.assign()]: is one of Commonly and usuallly used Method to define the Object in Javascript OOP.

=> JS OOP [Defining Object With Object.assign()] : It is a multiple Object function to create, assign new - Predefined Object(s) properties and methods.

=> JS OOP [Defining Object With Object.assign()] : This Function / Method has a very critical importance Due to following Features :

  1. The ability to Create a new Onfly Object {(Empty) + (Target Object)}, AND the ability to Inherite it's Property(s) and Method(s) from a other Parent Object's, OR even the ability to Assign specific partial Properties Value(s) and Method(s) from one/Multiple Object(s) to this New Object in one Script Method line
  2. The ablility to Assign {All} Properties and Functions from specific Pre-Defined Objects to other Pre-Defined Object in one Script Method line
  3. The ablility to Assign {Specific} Properties and Functions from specific Pre-Defined Objects to other Pre-Defined Object in one Script Method line
  4. The ablility to {Onfly} Assign New Specific Properties and Functions to {Pre-Defined Object / New-onfly Object} (Target Object) in one Script Method line
  5. Ablility to combine between Object Defining Method (onfly empty) & Assigning process in the same line of code

JS OOP [Defining Object With Object.assign()] Importance & Benefits:

[Defining Object With Object.assign() Method] gives us the Following Features :

  • Another unique Method to Define Object, wether an Empty one or with it's properites & Methods-Functions
  • Multiple Functional Objects Tasks in one script code
  • More Complicated and enhanced Object(s) Functions, which is critical in OOP and it's Technologies (DOM, BOM, Bootstrap, Php, ...).

All the previous Featueres of Assinging Mehthod gives great programing ability to design more complicated Scripts and Functional operations

JS OOP [Defining Object With Object.assign()] Syntax & Methods:

=> JS OOP Defining Object by [Object.assign()] Method as following :

// A) [ Defining Object by assing Method ] Syntax :

// (Definig a {New Empty Object} by Const,let,var key words) [Onfly Mode] :

const objName = Object.assign({});
let objName = Object.assign({});
var objName = Object.assign({});
//=================================================================================
//=================================================================================

// B) [ Assinging {All/Specific} of {New onfly Props and Methods/Pre-Defined Object's Properties
and Methods} to {New Empty / Pre-Defined} Object by {Assign Method} ] Syntax :

// B1) [Assinging {New onfly Properties and Methods} to {New Empty Target Object [onfly Object]}]:

const objName = Object.assign({}, {PropName1 : "Value1"}, {PropName2 : "Value2"});
let objName = Object.assign({}, {PropName1 : "Value1"}, {PropName2 : "Value2"});
var objName = Object.assign({}, {PropName1 : "Value1"}, {PropName2 : "Value2"});
//=================================================================================

// B2) [ Assinging {Pre-Defined Object's Properties and Methods} to {New Empty Target Object
[onfly Object]} ] :

const objName = Object.assign( {}, ObjName1.PropName1, ObjName1["PropName2"],
ObjName2.PropName1, ObjName2["PropName2"] );

let objName = Object.assign( {}, ObjName1.PropName1, ObjName1["PropName2"],
ObjName2.PropName1, ObjName2["PropName2"] );

var objName = Object.assign( {}, ObjName1.PropName1, ObjName1["PropName2"],
ObjName2.PropName1, ObjName2["PropName2"] );

//=================================================================================

// B3) [ Assinging {ِAll Pre-Defined Object's Properties and Methods} to {New Empty Target Object
[onfly Object]} ] :

const objName = Object.assign( {}, ObjName2, ObjName2] );
let objName = Object.assign( {}, ObjName2, ObjName2] );
var objName = Object.assign( {}, ObjName2, ObjName2] );
//=================================================================================
//=================================================================================

// B4) [ Assinging {New onfly Properties and Methods} to {Pre-Defined Target Object } ] :

Object.assign(TargetObjName, {PropName1 : "Value1"}, {PropName2 : "Value2"});
//=================================================================================

// B5) [ Assinging {Pre-Defined Object's Properties and Methods} to {Pre-Defined Target Object} ] :

Object.assign( TargetObjName, ObjName1.PropName1, ObjName1["PropName2"],
ObjName1.Method1(), ObjName2.PropName1, ObjName2["PropName2"],
ObjName2.Method2());
//=================================================================================

// B6) [ Assinging {All Pre-Defined Object's Properties and Methods} to {Pre-Defined Object
[Target Object]}] :

Object.assign( TargetObjName, ObjName2, ObjName2] );
//=================================================================================
//=================================================================================

// C) [ Assinging {All/Specific} of {New onfly Props and Methods/Pre-Defined Object's Properties and
Methods} to {New Empty / Pre-Defined} Object by {Assign Method} ] Mult Syntax :

// C1) Multiple Assinging to {New Onfly Target Object} :

const ObjName = Object.assign( {}, Objsrc1, Objsrc2.Prop1, Objsrc3["Prop2"],
{PropName3:"Value3"}, Objsrc4.Method() );

let ObjName = Object.assign( {}, Objsrc1, Objsrc2.Prop1, Objsrc3["Prop2"],
{PropName3:"Value3"}, Objsrc4.Method() );

var ObjName = Object.assign( {}, Objsrc1, Objsrc2.Prop1, Objsrc3["Prop2"],
{PropName3:"Value3"}, Objsrc4.Method() );

//=======================================================================

// C2) Multiple Assinging to {PreDefined Target Object} :

Object.assign( ObjName, Objsrc1, Objsrc2.Prop1, Objsrc3["Prop2"],
{PropName3:"Value3"}, Objsrc4.Method() );
//=======================================================================

//=======================================================================

// D) Accessing to all of the target Object element :
// D1) Accessing All Object's properties and Methods:
console.log(ObjName);
//=======================================================================

// D2) Accessing All Object's properties :
console.log(ObjName.Prop1);
console.log(ObjName.Prop2);
console.log(ObjName.Prop3);
console.log(ObjName.Prop4);
console.log(ObjName.Prop5);
console.log(ObjName.Prop6);
console.log(ObjName["Prop7"]);
//=======================================================================

// D3) Accessing All Object's Methods:
console.log(ObjName.Method1);
