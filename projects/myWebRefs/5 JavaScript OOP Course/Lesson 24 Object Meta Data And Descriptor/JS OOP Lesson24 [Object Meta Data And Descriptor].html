JS OOP [JavaScript Object Meta Data And Descriptor ] Concept:

=> JS OOP [JavaScript Object Meta Data And Descriptor ]: It is one of Powerful points of Object Controling Features.

=> JS OOP [JavaScript Object Meta Data And Descriptor ]: It provides more Accurate controling Features with the Object's Properties and their Attributes.

=> JS OOP [ Object Meta Data (Descriptors) ]: This Definition is about the information of the information of Properties of Object's Properoies , or we call it {Proprerties 's Attributes} [Descriptors].

=> JS OOP [Main object's Meta Data]: The Main Object's Properties's Attributes (Meta Data) [Object's properties Descriptors] as following:

  • {writable} => This Property Attribute Detect if the Propterty's Value is [Editable] (can be Changed), and it has (2) Probabilities:
    • {writable: true} => The Property's Value Can be Edited
    • {writable: false} => The Property's Value Can NOT be Edited
  • {enumerable} => This Property Attribute Detect if the Propterty can respond to {Looping} statements {for each loop}, and it has (2) Probabilities:
    • {true} => The Property's Value Can be printed within Loop statements {specially for-in Loop}
    • {false} => The Property's Value Can NOT be printed with in Loop statements {specially for-in Loop}, without persenting Error
  • {configurable} => This Property Attribute Detect (2) Property's Aspects :
    1. If the configurated Property can be {deleted}
    2. If the configurated Property can be {Re-Configurated} in later Configration
    => And and it has (2) Probabilities:
    • {configruable: true} => The Property {can be Deleted from it's Object} + {can be Reconfigurated with all the property's Meta Data [writable, enumerable, configurable, value]}
    • {configruable: false} => The Property {can NOT be Deleted from it's Object} + {can NOT be Reconfigurated with all the property's Meta Data [writable, enumerable, configurable, value]}
  • {value} => The Value of Object's Property

JS OOP [JavaScript Object Meta Data And Descriptor ] Importance & Benefits:

[JavaScript Object Meta Data And Descriptor ] gives us the Following Features :

  • It will gives more accurate mechanism to Control the Object's Properties, with different controling Aspects {[writable, enumerable, configurable, value]}
  • Accurate Object's Propeties controling provides greater chance for securing Object's Data {due to the advanced programable functions can be implemented by Mete Data}.
  • The Separate Controling of Object's property Meta Data gives great advanced feature for greater accurate Controling for each property .
