JS OOP [Constructor Function Advanced & new keyword] Concept:

=> JS OOP [Object Constructor Function Advanced]: This lessson will provide to developer more enhanced tools and script to control the constructor function [Pattern] and the extracted objects from.

=> JS OOP [Object Constructor Function Advanced new keywords]: These new keywords as following :

  1. {console.log(this)}; = > {AUTO printing the extracted Object once it being created (extracted)} function [Written in constructor Function]
  2. {(2) Mode of Extracting object from Constructor} :
    • {let objName = new ConstrictorName();} = > {Define a new Object as an Extracted Object from the assigned Constructor function}
    • {let objName = ConstrictorName();} = > {Define a new Object as a Global Object with all Properites in the assigned Constructor function}
  3. {(2) Methods to Check of Constructor Name of Certain Object} [each Method can be used as a conditional If , or to Print it directly]
    • {console.log(Extracted ObjectName instanceof Constrauctor Name);} = > {True = > if the mentioned Object is Extracted from the Mentioned Constructor Name, False if Not} .
    • {console.log(Extracted ObjectName.constructor === Constrauctor Name);} = > {True = if the mentioned Object is Extracted from the Mentioned Constructor Name, False = if Not} .
