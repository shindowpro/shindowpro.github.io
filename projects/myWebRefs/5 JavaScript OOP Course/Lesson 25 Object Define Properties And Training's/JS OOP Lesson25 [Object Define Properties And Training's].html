JS OOP [Object Define Properties And Training's ] Syntax & Methods:

=> JS OOP Defining-Declaring [Class] Conceptual Syntax as following :

// A) JS OOP General Methods of {Defining New Object's [single Property]} Syntax as following :

// 1) Original Definging Constructor (Object + Property Method) :

// One line Form:
const myObj = {propName1:value1};

// Multiple lines Form :
const myObj = {
propName1:value1
};

/// ===================================================================================

// 2) Direct Shortcut Definging New Property Method:

myObj.newPropName = value;

/// ===================================================================================

// 3) Full Attributes Object's Property Define Method:

// One Line Form: Object.defineProperty(ObjName, "propName", {writable: true-false, enumerable: true-false, configuable: true-false, value:"value"});

// Multiple Lines Form: Object.defineProperty(ObjName, "propName", {
writable: true-false,
enumerable: true-false,
configuable: true-false,
value:"value"});
/// ===================================================================================

/// ===================================================================================

// B) JS OOP General Methods of {Defining New Object's [Multiple Properties]} Syntax as following :

// 1) Original Definging Constructor (Object + Properties Method) :

// One line Form:
const myObj = {propName1:value1, propName2:value2, propName3:value3};

// Multiple lines Form :
const myObj = {
propName1:value1,
propName2:value2,
propName3:value3,
};

/// ===================================================================================

// 2) Direct Shortcut Definging New Property Method:

myObj.newPropName1 = value1;

myObj.newPropName2 = value2;

myObj.newPropName3 = value3;

/// ===================================================================================

// 3) Full Attributes Object's Property Define Method:

Object.defineProperty(ObjName, "propName", {writable: true-false, enumerable: true-false, configuable: true-false, value:"value"});

// Multiple Lines Form [this Method is a must]:
Object.defineProperties(ObjName, {
newProppName1 {
writable: true-false,
enumerable: true-false,
configuable: true-false,
value:"value",
},
newProppName2 {
writable: true-false,
enumerable: true-false,
configuable: true-false,
value:"value",
},
newProppName3 {
writable: true-false,
enumerable: true-false,
configuable: true-false,
value:"value",
},
});
/// =====================================================================

/// =====================================================================

//C) JSOOP Calling (get) {single Property} Descriptor Method :
Object.getOwnPropertyDescriptor(myObj.propName);
Object.getOwnPropertyDescriptor(myObj[propName]);
/// =====================================================================

/// =====================================================================

//D) Calling (get) All Object's [enumerable:true] Properties's Names (keys) Method :
Object.keys(myObj);

  1. Initiating (Default Object) containing {(3) Default Properties} by Following Methods:
    • Constructual Method => Defining the Object by the Construcutual Mehtod + Defining properties
    • (Shortcut/Direct) Method => Defining an empty Object + Defining properties by Direct Method
    • Full Attibutes Method => Defining an empty Object Mehtod + Defining properties by Adding Group of Properties Method
  2. Adding new property(s) to the Default Object of (defined by first method) by (2) Methods as following:
    • {Adding Single new Property} => Including the following Functions:
      1. Checkinig for The Repeated Names of new inserted property + ability to re-insert new name of the same property in case of repeating name
      2. Assigning all kind of attributes {writable, enumerable , configurable, value} of each new property
      3. The Ability to repeat the single Adding new property by count of user desire
    • {Adding group of new Properties} => Including the following Functions:
      • Checkinig for The Repeated Names of new inserted property + ability to re-insert new name of the same property in case of repeating name
      • Assigning all kind of attributes {writable, enumerable , configurable, value} of each new property
      • The Ability to repeat the single Adding new property by count of user desire
      • (2) method of Adding new properties are available :
        1. Adding new Properties using {Singular Adding Method} looped {by count of desired new properties}
        2. Adding new Properties using {adding group of properties Method}
    • {Displaying of (2) kind of information of all object (including the Original properites + new Added Properties)} as following:
      1. [Displaying Discriptors] => of All object's properties's (Discriptors + Names, values ), with (2) accessing methods to each property :
        1. Accessing By property Name
        2. Accessing By property Value
      2. [Displaying keys] => Of All object's Names + properties {with Attribute: enumerable:true} , with (2) accessing methods to each property :
