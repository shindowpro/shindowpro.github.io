> Main New Syntax & Methods in this lesson are as following :

  1. {Object.defineProperty();} : JS OOP Define new Property and add it in an Object.
  2. {delete ObjName.PropName;} : JS OOP Delete an Property from an Object .
  3. {Configurable: true} : Property {Can} be Deleted Later from an Object .
  4. {Configurable: false} : Property {Can NOT} be Deleted Later from an Object .
  5. {const ObjName = Object.freeze({PropName: "Value"})} : Defining a new Freezed Object {It's Properties Can Not be deleted later}.

JS OOP [Delete Operator + Configurable Flag + Define Property + Freeze Object] Concept:

=> JS OOP [Delete Operator]: Is one of Commonly and usuallly used controling Object's Properties Method to (DELETE) the Property from Object in Javascript OOP.

=> JS OOP [Configurable Flag] : Flag(s) are the Attributes of Property's Properties in Object, and there are many Flags of Object's Property, but in this lesson we will study only [Configurable] Flag.

=> JS OOP [Configurable Flag: true-false] : This Flag or Attribute is about Detecting if the property will be able to be deleted or Not [by Property Delete Funtion Method]

JS OOP [ const ObjName = Object.freeze({PropName: "Value"}); ] : Defining a Freezed Object {It's Properties Can Not be deleted later} even by using Delete Method .

=> JS OOP [Delete Operator + Configurable Flag + Define Property + Freeze Object] : These Methods will allow to developer the following programing features:

  1. { Define Property }: The ability define a new Property(s) with certain Key Name , Value , and Flags/Attributes, AND add it to a certain Object.
  2. { Delete Operator }: The ability to delete a certain property from a certain Object {with true Configurable flag}.
  3. { Configurable Flag: true-false }: The ability to detect the ability of Deletion Attribute/Property of a certain Object's Property.
  4. { Object.freeze({PropName: "Value"}); }: The ability to Define a whole Object with Undeletable Properties (Same like {configurable flag: flase}) .

JS OOP [Delete Operator + Configurable Flag + Define Property + Freeze Object] Importance & Benefits:

[Delete Operator + Configurable Flag + Define Property + Freeze Object Method] gives us the Following Features :

  • Great ability to create unlimitless count of Properties in any Object with Full controling of it's properties or Flags or Attributes
  • Full Control of Object's Property(s) Delelting function, whether of the Whole Object {Freeze Object method}, OR Certain Object's Property {configurable Flag} , with all it's References form Memory and Garbag Collecting ability
  • These Methods will give programer Great ability in dealing with advnaced Techologies such as [DOM, BOM, Bootstrap] .
  • All the previous Featueres of Assinging Mehthod gives great programing ability to design more complicated Scripts and Functional operations

JS OOP [Delete Operator + Configurable Flag + Define Property + Freeze Object] Syntax & Methods:

=> JS OOP Defining Object by [Delete Operator + Configurable Flag + Define Property + Freeze Object] Methods as following :

// A) [Define New Property] Method Syntax :

// A1) Defining an empty Object:
const objName = {};

// A2) Defining and Adding a new Property in an Object + with Ability for Deleting Flag/Attribute (configurable:true)
Object.defineProperty(objName, newPropertyName, {value: #Property Value, configurable:true);

// A3) Defining and Adding a new Property in an Object + without an Ability for Deleting Flag/Attribute (configurable:false)
Object.defineProperty(objName, new Property Name, {value: #Property Value, configurable:false);

// A4) Printing the Whole Object after defining and adding the new Property:
console.log(objName);

// A5) Printing the certain New Object's Property after defining and adding to the Object:
console.log(objName.PropName);

console.log(objName["PropName"]);

//===========================================================
//===========================================================

// B) [Defining a new Freezed Object (Undeletable Object's Properties) ] :
const FobjName = Object.freeze({propertyName: "Value"});

// B1) [Testing the ability for deleting a Property from Freezed Object {will give us [False]} ]:
console.log(delete FobjName.propertyName);

// B2) [Printing the Whole Freezed Object after trying to delete it's Property [Will be Printed Successfully] ]:
console.log(objName);

// B3) [Printing the Property of Freezed Object after trying to delete it [Will be Printed Successfully] ]:
console.log(objName.propertyName);

//===========================================================

// C) [Deleting a Property from a Defined Object] :
delete objName.PropName;

delete objName["PropName"];

// C1) Printing the Whole Object after Deleting a Property from the Object [Will print the Object WITH OUT of the Deleted Property] :
console.log(objName);

// C2) Printing the certain New Object's Property after Deleting a Property from the Object [Will print Undefined because the property has been deleted successfutly]:
console.log(objName.PropName);

console.log(objName["PropName"]);

//=============================================================
