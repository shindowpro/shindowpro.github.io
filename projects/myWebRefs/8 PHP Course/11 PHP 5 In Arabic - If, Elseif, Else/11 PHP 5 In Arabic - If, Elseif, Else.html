PHP [11 PHP5 - If, Elseif, Else] Concept of [Conditional If] in php5:

[Conditional If] is a critical and basic in any P.language, and one of the most important concept & tool that all scripts built on it

[Conditional If] include the following methods :

  1. [if {}] => {if (condition) {code to run if the condition is true}}
    • This method means only (One condition) with out any exceptions
    • If the condition is true => the code will be run
  2. [if {} - else {}] => {if (condition) {code to run if the Main condition is true} || else {Code will be run in the main condition is not true } }
    • This method means (One condition) with exception
    • If the condition is true => the code will be run ,
      but if NOT => the Else Code will be run (as an execption of the main Condition)
  3. [if {} - ifelse {} - else {}] => {if (Condition1) {code to run if the condition1 is true} elseif (Condition2) {code to run if the Conditoin2 is true} || else {Code will be run in the main condition is not true } }
    • This method means (Multiple conditions) with exception
    • If the condition1 is true => the code1 will be run ,
      but if NOT & Conditon2 is true => the code2 will be run ,
      • but if Condition1 & condition2 is false => the Else Code will be run (as an execption of the main Condition)
