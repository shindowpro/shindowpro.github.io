PHP [22 PHP 5 In Arabic - While Loop] main Concept:

For Loop Advanced => To write the For statement more shortly + with all functionlities

For Loop Advanced => Has (3) levels according to numbers of removed Expressions :

  1. [ Removing First Expression ] -> Initial Counter => {for ( ; condition; Increment/Decrement ) {}}
  2. [ Removing Second Expression ] -> Condition => {for ( ; ; Increment/Decrement ) {If condition Statement}}
  3. [ Removing Third Expression ] -> Increment/Decrement => {for ( ; ; ) {If condition Statement + Increment/Decremnent}
