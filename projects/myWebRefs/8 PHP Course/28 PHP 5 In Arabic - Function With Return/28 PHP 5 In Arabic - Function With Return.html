[Function With Return] => Is one of the most important Element in advanced function.

[Function With Return] => It is the recommended to output of the function {better than the echo method} .

[Function With Return] => Return Method is the final procedure inside the function {any further codes after Return will be ignored}

[Function With Return] => Will be used to output the function Result when the function is being called

  • [Function With echo] => Datatype of will display { Function Name & NULL(due to there are no Return Method used in Function) }
