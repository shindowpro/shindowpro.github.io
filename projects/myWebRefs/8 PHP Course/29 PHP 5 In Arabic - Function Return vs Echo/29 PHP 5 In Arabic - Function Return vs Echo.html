PHP [29 PHP 5 In Arabic - Function Return vs Echo] main Concept:

[Function Return vs Echo] => Both of two methods are to output of the function Result.

Defferences betweeen [Function with Return] vs [Function with Echo] =>


  1. {Output result by (Ordinary Calling Function)} ->
    • [Function with Return] : Will display {NULL} (due the funciton has not return syntax)
    • [Function with Echo] : Will dispaly the {function script Result} into the webpage [even without using echo syntax]
  2. {Output result by (Echo Calling Function)} ->
    • [Function with Return] : Will display the {function script Result} into the webpage [even with using return syntax]
    • [Function with Echo] : Will display the {function script Result} into the webpage
  3. {Datatype of Function calling} ->
    • [Function with Return] : Will display the {Function Script Result} && {Datatype of function script result} ]
    • [Function with Echo] : Will display the {Function Script Result} && {NULL} ]
  4. {Processing Function Result} ->
    • [Function with Return] : Providing easy & flexible processing of function result (can be used as raw date || within a variable )
    • [Function with Echo] : Not Providing any processing of function result

Cases recommended to use each case:


  1. [Function with Return] -> Recommended to use when it desired to (process) & (display) data of function to the user in webpage
  2. [Function with Echo] -> Recommended to use when it desired to (only process) the function result (without displaying) it to the user in webpage


PHP [29 PHP 5 In Arabic - Function Return vs Echo] Importance & Benefits:

  • It is very critical in advanced function and any scripts (speciall when dealing the server)
  • It provide the flexiblity to use/process the function result as values or variables inside process (function with return)
  • It provide the flexiblity to use/process & display the function result as values or variables inside process (function with return)
