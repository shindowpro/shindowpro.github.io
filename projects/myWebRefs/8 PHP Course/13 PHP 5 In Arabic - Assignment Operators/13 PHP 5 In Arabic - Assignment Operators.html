PHP [13 PHP 5 - Assignment Operators] main Concept:

They are critical in any scripting calculating in any programing language

They are basic not only in mathematic calculating , but also in any scripting procedures and specially [Logical operations based on (condition If)]

In any type of Arithmetic operator , they can be used as simple value || Variables (not constants) calculation

Main types of Assignment Operators as following :

  1. [Summition - Addition {.=} ] => Adding Textual value [concatenation ]
  2. [Summition - Addition {+=} ] => Adding value calculation
  3. [Substracion {-=} ] => Substracting value calculation
  4. [Multiply {*=} ] => Multiply value calculation
  5. [Division {/=} ] => Dividing value Calculation
