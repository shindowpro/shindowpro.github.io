PHP [14 PHP 5 - Comparison Operators] main Concept:

They are critical in any scripting & function in any programing language

They are basic not only in mathematic calculating , but also in any scripting procedures and specially [Logical operations based on (condition If)]

In any type of Comparison operator , they can be used as simple value || Variables (not constants) calculation

Main types of Comparison Operators as following :

  1. [ Equalued {==} ] => Equalled value operator
  2. [ Typical Equalued {===} ] => Equaled in both value and Datatype
  3. [ Not Equalued {!=} ] => Not Equaled of value only
  4. [ Not Equalued {< >} ] => Not Equaled of value only
  5. [ {<} ] => less than
  6. [ {< = } ] => less than or Equaled
  7. [ {>} ] => greater than
  8. [ {>=} ] => greater than or Equaled
