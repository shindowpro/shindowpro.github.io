[PHP - String Functions - Str_Split || Chunk_Split ] => Concept:

[PHP - String functions [ Str_Split || Chunk_Split ] method syntax & concept as following :

  1. {Str_Split} : $Arr = Str_Split(string, Length); => To split the string varaible into letters(including empty spaces) and piut them into an [array] of elements splitted accroding to certain Length , and syntax includes the following elements :
    1. [String variable] => to be splitted or chunked => {Obligated}
    2. [Length] => Number of splited element in the array => {Optional}
  2. $var = chunk_Split(string, Length, End); => To split the string varaible into chunks(peaces of string) splitted by customized (splitter) and put them in a [new variable] of splitted chunks elements accroding to certain {Length} & {end splitter} , and syntax and include the following elements :
    1. [String variable] => to be splitted or chunked => {Obligated}
    2. [Length] => Number of splited chunk element in the array => {Optional}
    3. [End] => End splitter type [regular expressoin] => {Optional}
