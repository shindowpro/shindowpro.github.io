[PHP Array Methods - Search ] => Array Methods - Search Concept:

{Adding new elements to the array} is One of the most important Controling methods of the Array [Indexed || Associative || Multidimensional] in PHP

Adding new elements methods to the array as following :

  1. {array_push} => To add Element or more to the End of Array's Elements
  2. {array_unshift} => To add Element or more to the Start of Array's Elements

Adding new Elements Methods to the array is based on [Index] , even with associative Array



PHP [37 PHP 5 In Arabic - Array Methods - Add Items] Importance & Benefits:

  • Accurate adding new Elements with no limit, in certain position of the array's elements [first - end]
  • Accurate adding new Elements to any kind of array in php
