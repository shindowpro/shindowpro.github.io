[PHP - String Functions - Explode ] => Concept:

[PHP - String Explode ] method syntax as following :
{$exploadArr = explode("Seperator", String Variable , Limit);

{Explode method} => Include following element:

  1. [Seperator] : The type of (seperator) that the exploding array will split the string after it
  2. [String Variable] : The Text that will be splitted upon given options with in exploding method
  3. [Limit] : This is the count of splited elements in the exploding array, and it's value has (3) options :
    • {Positive} : Which is determine the count of splitted elements in the exploding array
    • {Negative} : Which is determine the count of excepted elements from the last of splited in the exploding array
    • {Zero} : Which is make the exploding array is content of just one element
