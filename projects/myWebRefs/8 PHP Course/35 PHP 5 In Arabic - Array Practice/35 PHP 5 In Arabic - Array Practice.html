One of the main PHP Array Types which remarkable with [Multiple layers/Dimensions of children arrays inside each others] of the main Array

Main Types of Multidimensional Arrays as following :

  1. {Indexed Array inside Indexed Array} :
    => It is a parent Indexed Array with children Indexed Array .
  2. {Associative Array inside Associative Array } :
    => It is a parent Associative Array with children Associative Array .
