[PHP - File System Function - [ 64 PHP 5 In Arabic - File System - File_Exists, Is_Writable , dirname(__File__), file_put_contents ] ] => Concept:

  1. [PHP - File System Function/methods syntax & concept as following :
    • file_exists :
      • Checking if the File is existed Or not
      • Usually used with the if condition
      • Can be used for checking for a file before dealing with it [Including || Editing || Adding contents ]
    • is_writable || is_writeable :
      • Checking if the type of permission File [writable || readonly} ]
      • Usually used before adding new contents to an exited & writable file
    • file_put_contents :
      • To Put/Write inside the file {even the file is existed or not}
      • Usually used with {file_exists} to add new content to an existed file, but {file_put_contents} can add new contents to an unexisted file
      • If the file asked to put new contents is not existed => The fuction will create it first & add new contents inside
    • dirname(__File__) :
      • To get the full path of the current php file
      • Usually used to declare new file variable with full path

  2. There are built in Functions in php pLanguage To get the full path of file in the server [documnent root] || [server root] => will be studied in later lessons
