PHP [15 PHP 5 - Increment _ Decrement Operators] main Concept:

They are critical in any scripting & function in any programing language

They are basic not only in mathematic calculating , but also in any scripting procedures and specially [Increasing & decreasing counter upon certain prcedures or user actions ]

Incrememnt & Decrement are effecting only on Numeric & String Variables

Main types of Comparison Operators as following :

  1. [ Pre Increment {++$var} ] => Increasing then Printing operator
  2. [ Post Increment {$var++} ] => Printing Decreasing operator
  3. [ Pre Decrement {--$var} ] => Decreasing then Printing operator
  4. [ Post Decrement {$var--} ] => Printing then Decreasing operator
