[PHP - Cookies - [ 87 PHP 5 In Arabic - Session - Start||Resume Session] => Concept:

  1. [PHP - Session - Start||Resume Session] => The main [Session] Setting Syntax ] as following :

    Syntax => session_start(advanced option);

    Concept =>
    => This is the main Session Global Variable Definition, to be started in the current page or to be Reusumed other page(s) .
    => Session is one of the obligatory to be used , and not like the [Cookie].
    => Session must be defined before any html output to be working properly, just like [Cookie].
    => Session using is a MUST , specially in the websites pages that include advanced options and features like:

    • Admin-user Login/Registraion
    • Authorization Levels according to the user level
    • User online settings Choices to be saved online by the Server
    / => General Comprison between [Session] & [Cookie] :
    • [Saving Data] Method => [Session]: Saving data as Branched Variables online by Server , [Cookie] : Saving data in the user PC
    • [Server] Method => [Session]: Dealing online with teh user data and setting , [Cookie] : Dealing with any Server (Online - Offline)
    • [Timing out] Method =>
      1. [Session]: Is timing out once the user is out of the browser and be Restarted when he open the webpage / application
      2. [Cookie]: Is timing out once the Delete the Cookie file OR at the end of the time Function

  2. [ Session's Variable(s) ] => :

    Syntax => $_SESSION['Variable Name'] = Value;

    Concept =>
    => They are Declared Globals Variables of the Session, which they can be imported from any other page that using this session to be processed , or used



PHP [87 PHP 5 In Arabic - Session - Start _ Resume Session] Importance & Benefits:

  • [ Session - session_start() } ] => It is the method to be used to set/save user/admin online settings of the web page/application
  • [ Session - session_start() } ] => It is a great tool to control the user/admin Authorizations levels on the web page/application , and providing each level it's features
  • [ $_SESSION['Session variable name'] ] => It is the method to define the user data and save it as variables on the server
