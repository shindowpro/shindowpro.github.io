PHP [16 PHP 5 In Arabic - Logical Operators] main Concept:

Logical operators are used in conditional IF and other functions in any P.language

They are critical in any scripting & function in any programing language

They are basic not only in mathematic calculating , but also in any scripting procedures and specially [testing conditions in server back side ]

Main types of Logcial Operators as following :

  1. [ {and} ] => And logical operator [Not usually recommeded]
  2. [ {&&} ] => And logical operator [Is usually recommeded & and more featured]
  3. [ {or} ] => OR logical operator [Not usually recommeded]
  4. [ {||} ] => OR logical operator [Is usually recommeded & and more featured]
  5. [ {!} ] => NOT logical operator [is the negative operator to for any other condition / Other logical operator ]
  6. [ {Xor} ] => Exclusive OR logical operator [OR operator to accept one or more conditions BUT not all conditions]
