PHP [18 PHP 5 - String Operators] main Concept:

String Concatenation used very commonly in any php script, and had in previous lessons discussed

They are critical in any scripting & function in any programing language

The main String Operator are :

  1. [.] => This Concatenation single operator used to concatenate between any Variables values || String Value || both
  2. [.=] => This Concatenation Overwriting operator used to add new value to a previous value of certain variable
