[PHP Array Intro] main Concept:

Array is a special variable which is holding multiple of variables [Names && Values]

Array can hold different data types of variables

Array is the gate of PHP language to deal with Database (mySQL)

Array is usually being used with Looping , which is one of the most powerful methods in Any pLanguages

Array in PHP has (3) types of Arrays :

  1. [Indexed Array] => Which has index/counter Number to it's elements
  2. [Associative Array] => Which dealing with it's elements by [Names] && [Values]
  3. [Multidimentional Array] => Which is dealing with multiple dimensions of Data & elements

Array next lessons will take each type of Array in single lesson + advanced Example


PHP [31 PHP 5 In Arabic - Array Intro] Importance & Benefits:

  • Array is critical to deal with large numbers of data (variables), which can not be dealt with by any method else
  • Array is the PHP perfect gate/Method to deal with Database (mySQL)
  • Array with Looping provides very powerful tool/method to deal with large data/elements
  • Meriging between Array || Loop || Function => provide very powerful scripting
