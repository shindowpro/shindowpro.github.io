[PHP - File System Function - [ 63 PHP 5 In Arabic - File System - Intro ] ] => Concept:

  1. [PHP - File System Function/methods syntax & concept as following :
    • To have control on the file system on all operating system such as
      • Create File/Directory
      • Copy File/Directory
      • Move File/Directory
      • Paste File/Directory
      • Delete File/Directory
      • Rename File/Directory
      • Checking of existing File/Directory
    • File System functions are very common & provides greate abilities on dealing with files/folders-directories
    • PHP File System functions can provides the following abilities
      • Ability to add contents to an existing File/Directory
      • Ability to Edit contents of an existing File/Directory
      • Ability to Read/Get content of existing File/Directory
      • Ability to Get status/properties of existing File/Directory
      • Ability to check on the existance of File/Directory
      • Ability to change File/Directory Mode-Permision[Readonly, write , execute] || Owner[apatchi , user , wwww]
    • These functions are similer to the one in the command line / linux / mac / Terminal operations-commands
    • Common Examples of php file system :
      1. [chgrp] => Change the group of the file - directory {any file has a group in the OS}
      2. [chmod] => Change the mode/permision of the file - directory {Readonly || Write || Execute}
      3. [chown] => Change the owner of the file - directory {Apatchi , user , www}
      4. [delete] => Delete the file - directory
      5. [copy] => Copy the file - directory
      6. [rename] => Rename the file - directory
      7. [dirname] => Getting the directory name
      8. [disk_free_space] => Getting the Free space of the current partition
      9. [disk_total_space] => Getting the Total space of the current partition
      10. [fopen] => Open a file
      11. [fclose] => Close a file
      12. [file_exists] => Check if the file is existed
      13. [file_get_contents] => Read/get the file contents regardless of it's type [even it was a web site files] in string
        can be filtered with the php functions
      14. [file_put_contents] => Write/set contents/orders/scripts in to a file regardless of it's type [even it was a web site files]
        can be used to sent orders to the php server
      15. [fwrite || fputs] => To add/write contents on a file
      16. [fread] => To get/read contents from a file
      17. [is_dir] => Check the path is a directory or not => usually used before dealing a folder/directory
      18. [is_file] => Check if the current is a file or folder/directory
      19. [is_readable] => Check if the file is readable or not
      20. [mkdir] => Create a new folder/directory
      21. [rmdir] => To Remove the folder/directory [if it was empty]
