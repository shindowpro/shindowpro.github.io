[PHP - String Functions - [ trim() , rtrim(), ltrim() ] ] => Concept:

[PHP - String functions [Trim] method syntax & concept as following :

  1. [trim(String Variable, Customized Charlist)]; => Remove the addtional charachters from the Both sides from String , and syntax includes the following elements :
    1. [String Variable] => The String variable/ Text to process the method in it's content => {Obligated}
    2. [charlist] => The default character & symbols , which will be removed from the string by default => {Optionanl}, and include the following:
      • [/0] => The Null value in the String
      • [/t] => The Horizontal Tab value in the String
      • [/n] => The new line in the String
      • [/r] => The cariage return in the String
      • [/x0B] => The vertical Tab value in the String
    3. [trim($String Variable)] => The String variable/ Text ONLY is to process the default Trim Method [Trim the default Charlist]

  2. [rtrim(String Variable, Customized Charlist)]; => Remove the addtional charachters from the Right side (before) only from String , and syntax includes the following elements :
    1. [String Variable] => The String variable/ Text to process the method in it's content => {Obligated}
    2. [charlist] => The default character & symbols , which will be removed from the Right side (before) the string by default => {Optionanl}, and include the following:
      • [/0] => The Null value in the String
      • [/t] => The Horizontal Tab value in the String
      • [/n] => The new line in the String
      • [/r] => The cariage return in the String
      • [/x0B] => The vertical Tab value in the String
    3. [rtrim($String Variable)] => The String variable/ Text ONLY is to process the default Right Trim Method [Right Trim with the default Charlist]
  3. [ltrim(String Variable, Customized Charlist)]; => Remove the addtional charachters from the Left side (after) only from String , and syntax includes the following elements :
    1. [String Variable] => The String variable/ Text to process the method in it's content => {Obligated}
    2. [charlist] => The default character & symbols , which will be removed from the Left side (after) the string by default => {Optionanl}, and include the following:
      • [/0] => The Null value in the String
      • [/t] => The Horizontal Tab value in the String
      • [/n] => The new line in the String
      • [/r] => The cariage return in the String
      • [/x0B] => The vertical Tab value in the String
    3. [ltrim($String Variable)] => The String variable/ Text ONLY is to process the default Left Trim Method [Left Trim with the default Charlist]
