PHP [10 PHP5 - Constant] Concept of [Constants] && [Magic Constants] in php5:

First Constants Concept:

  1. [Constants] => {Normal constants}
    • It is a fixed data memory, which can Not changed [Re-declared] by develper later once it has declared
    • Constant Can be accessed by any other scope in the php file
  2. [Magic-Constants] => {PHP built-in constants}
    • They are variant fixed functions which has many Programtic features, such as :
      1. {Get current local Host file} => echo __FILE__ ;
      2. {Get current local Host Folder/Direct} => echo __DIR__ ;
      3. {Get current LINE of this script } => echo __LINE__ ;
      4. {Get PHP Integer Max Number } => echo __PHP_INT_MAX__ ;

    3. Second Declaring Constants Methods:

    1. [ define('Constant_Name', 'Constant Value', letter Insensitivity[true || false] )] => {Most Common }
      • {Constant_Name} => It recomended to be upper case [to avoid programaic Problems]
      • {Constant Value} => It can be texual value || logic value (false , true)
      • {Constant Insensitivity[true || false] )} => It a logic value (false , true)
    2. [ const Constant Name = 'Constant Value'; => {Less Common }
      • This uncommon method to declare the constant is not common in usual
      • this method is common likely in {OPP}


PHP [10 PHP5 - Constant] Importance & Benefits:

  • [PHP - Constants] => Provides serveral advantages to the developer to control website server with high security for data
  • [PHP - Constants] => Help to prevent bugs in scripts by saving critical data from manipulating by user or develpers
  • [PHP - Magic-Constants] => Provides very handy feature to the devolpers , by variant built-in functions
