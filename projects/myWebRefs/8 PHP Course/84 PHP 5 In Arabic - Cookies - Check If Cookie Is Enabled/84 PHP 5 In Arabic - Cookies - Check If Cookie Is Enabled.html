      [Cookies] main Concept & Syntax :
      => A) Concept :
    1. Kind of Predefined Variable which is dealing with the Cookies of the website of the user/visitor

      • Website user language
    3. Provides a better solution for storing user Data , better and easier than Database
    4. It has a default location on pc {with SubFolder for each brower}

      8. => B) Syntax :
    8. The [ $_COOKIES('Cookies Name', 'Cookies Value' , time() + #Number , 'Cookies Method' ) ] Global Variable syntax as following:
      1. Cookies Name =>
        It is the name of file of cookies in the path of cookies {which is differing form a browser to anther }
      2. Cookies Value =>
        It is the Value of file of cookies in the path of cookies {which is differing form a browser to anther }
      3. Cookies time =>
        It is the activation duration time of the cookies in the browsered web site
      4. Cookies Method =>
        It is the range of accessing the browsered web site
