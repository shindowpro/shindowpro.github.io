  3. PHP [19 PHP 5 In Arabic - All Operators] practical examples of all Operators:
    1. 1 ) [ Arithmetic operators ] :
      • {+} => Addition operator
      • {-} => Substraction operator
      • {*} => Multiply operator
      • {/} => Divsion operator
      • {%} => Modulus operator
      • {**} => Power operator
    2. 2 ) [ Assignment operators ] :
      • {=} => Assigning value to cerrtain varialbe operator
      • {+=} => Adding numeric value to a variable operator
      • {-} => Substracting numeric value to a variable operator
      • {*} => Multiplying numeric value to a variable operator
      • {/} => Dividing numeric value to a variable operator
      • {%} => Modulus numeric value to a variable operator
      • {**} => Powering numeric value to a variable operator
    3. 3 ) [ Comparison operators ] :
      • {==} => Value Equality operator
      • {===} => Identical Equality operator
      • {> } => Greater than operator
      • {< } => Less than operator
      • {>=} => Greater than / Equal operator
      • {<= } => Less than / Equal operator
      • {!=} => Not Value Equality operator
      • {< >} => Not Equality operator
      • {!==} => Not Identical Equality operator
    4. 4 ) [ Incremnent & Decrement operators ] :
      • {++Var;} => Pre Increment operator
      • {Var++;} => Post Increment operator
      • {--Var; } => Pre Decrement operator
      • {Var--; } => Post Decrement operator
    5. 5 ) [ Logical operators ] :
      • {and} => Adding new Condition to another in Conditional if operator
      • {&&} => Adding new Condition to another in Conditional if operator {more effecicent & common}
      • {or} => Making ability to choose between more than one condition in [Conditional if] operator
      • {||} => Making ability to choose between more than one condition in [Conditional if] operator {more effecicent & common}
      • {xor} => Making ability to choose between more than one condition in [Conditional if] operator {But not all conditions }
      • {!} => Not or negative condition in [Conditional IF]
    6. 6 ) [ Error control operators ] :
      • {@} => To ignore the propable Error could be implemented in the following syntnax operator
      • {or} => To make altrernative action insteasd of the error syntax Default action Message operator
      • {die('')} => To Assign customized Error message instead of the Default message operator
    7. 7 ) [ String operators ] :
      • {.} => To add a single textual to the previous text / variable operator
      • {.=} => To add a commulated textual value to the previous textual variable operator
