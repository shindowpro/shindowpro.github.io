      Predefined Variables - Get - main Concept :
    1. Kind of Built-in Global Variable which is dealing with the Webpage data from form or etc , to be gathered and processed
    2. The [REQUEST] Global Variable syntax as following:
      1. $_REQUEST[$VariableName] =>
        • Concept =>
          • To Collect the page data inorder to be processed [in the same page or be sent to another page according to {Form Action Tag} ]
          • [REQUEST] => Is gathering data form the page and put it in the [Query String] in the address bar of the page
        • Importance and Usages =>
          • Connecting / dealing / Sending / Processing with page data such as [ Login Forms (Inputs fields , Submit buttons) ]
          • Sending Data/Request to the Server & database ]
        • Most Common Methods can be processed by this Variable =>
          1. Getting & Sending login Form to the server , for checking and displaying data/sessions
          2. Sending page cerain data to the daabase /server => for adding processing
        • Most Differeneces between [Get] || [Post] || [Request] =>
          1. Gathering information from the page :
            • [Get] => Displaying the gahtered data in the [query string] at the top of the address bar page
            • [Post] => Do Not displaying the gahtered data in the [query string] at the top of the address bar page
            • [REQUEST] =>
          2. Securing information :
            • [Get] = > Not safe in sending critical/sensitive data [due to displaying the sent information in the {Query String}]
            • [Post] = > It is safe in sending critical/sensitive data [due to Not displaying the sent information in the {Query String}]
            • [REQUEST] = >
          3. Other difference :
