[PHP - File System Function - [ 74 PHP 5 In Arabic - File System - Fopen] => Concept:

  1. [PHP - File System Function/method syntax [Fopen()] as following :
    1. To Handle the file for further Editing (Reading & Writing & Both) , with advanced properties
    2. Include the following :
      • File Full Path Name => The desired (PC : directory/folder) || (Internet: URL)
      • Mode Type => Advanced feature to Mode of Handling File , including the following modes or options:
        1. r : Read only the file , with following Features :
          • File Existance is Required
          • If File Not Existed => Error will be thrown
          • Reading from start of file's contents
        2. r+ : Read & Write the file :, with following Features :
          • File Existance is Required
          • If File Not Existed => Error will be thrown
          • Reading from start of file
          • Writing by Adding to the curent contents
        3. w : Write only the file , with following Features :
          • File Existance is NOT Required
          • Create the file if not Existed
          • Writing & Reading from start of file's contents
          • Writing by Overwriting the curent contents
        4. w+ : Write and read the file , with following Features :
          • File Existance is NOT Required
          • Create the file if not Existed
          • Writing from start of file's contents
          • Writing by Overwriting the curent contents
        5. a : Write only the file , with following Features :
          • File Existance is NOT Required
          • Create the file if not Existed
          • Writing from End of file's Contents
          • Writing by Adding to the current contents
        6. a+ : Write & Read the file , with following Features :
          • File Existance is NOT Required
          • Create the file if not Existed
          • Writing & Reading from End of file's Contents
          • Writing by Adding to the current contents
          • This mode has extra advnaced feature (if seek) And will be taken later
        7. x : Write only the file , with following Features :
          • File Existance is Prohibited
          • The file Must not be Existed
          • Writing from End of file's Contents
          • Writing by Adding to the current contents
        8. x+ : Write and Read only the file , with following Features :
          • File Existance is Prohibited
          • The file Must not be Existed
          • Writing & Reading from End of file's Contents
          • Writing by Adding to the current contents
      • Include_Path => Advanced feature Will be taken later
      • Context => Advanced feature Will be taken later
    3. This method is related to other methods will be taken following lessons [fread() || fwrite() || fclose()]
    4. The End Line Handling is deffer according operating system , as following :
      1. Unix => End file content by {/n}
      2. Windows => End file content by {/r/n}
      3. Mac => End file content by {/R}
