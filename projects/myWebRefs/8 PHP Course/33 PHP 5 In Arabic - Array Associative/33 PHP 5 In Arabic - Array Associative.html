[PHP Associative Array ] => Associative Array Concept:

One of the main PHP Array Types which remarkable with [Keyname + Value] for each element

It differs from the Indexed Array in following :

  1. {Concept} :
    => [Indexed Array] => Elements has only Value.
    => [Associative Array] => Elements has both KeyNames && Values.
  2. {Decleration Method} :
    • {Associative Array} -> only by structural Form :
      $ArrayName = array("keyName1" => "Value1", "keyName2" => "Value2", "keyName3" => "Value3", );
    • {Indexed Array} -> by (2) methods :
      => [Default Structural Form] => $ArrayName = array("keyName1","keyName2", "keyName3" );
      => [Sqaured Brakets Form] => $ArrayName[] = "keyName1", "keyName2" ,"keyName3" ;
  3. {Adding new elements Method} :
  4. {Associative Array} -> can Add new Elements by (2) approaches :
    1. [Solo Approach] : by (1) method:
      $ArrayName["newKeyName"] = "new Value" ;
    2. [Group Approach] : by (1) method:
      [using both (Foreach loop) + (Solo Method)] => {for (counter of new Elements to be added) {Sqaured solo adding}}
  5. {Indexed Array} -> by (2) approaches :
    1. [Solo Approach] : by (2) method:
      => [Sqaured (with default index) method] : $ArrayName[] = "new Value" ;
      => [Sqaured (with determined index) method] : $ArrayName[#num] = "new Value" ;
    2. [Group Approach] : by (2) method:
      => [using both (For loop) + (Solo Method)] => {for (counter of new Elements to be added) {Sqaured solo adding + with default index}}
      => [using both (For loop) + (Solo Method)] => {for (counter of new Elements to be added) {Sqaured solo adding + with determined index}}
