      Predefined Variables - Server - main Concept :
    1. Kind of Built-in Global Variable which has options and authurity to access and deal with the server
    2. The [Server] Global Variable syntax as following:
      1. $_SERVER['Option Type'] =>
        • Concept =>
          • The main Decleration of the Server variable
          • Giving main information about the current server [Host Environment]
        • Importance and Usages =>
          • Critical specially when dealing with Advanced Web Projects and long Scripts with advanced features such as [Host , Users , Header...]
          • Critical specially when dealing with Advanced Web features Technolgies such as [Web Serice] , [Port] , [Socket]
        • Most Common Methods and options =>
          1. $_SERVER['PHP_SELF'] =>
            • Concept => Accessing/Directing to the current Server of the current page -upon Request-
            • Importance and Usages => Critical when dealing with Forms and page data connected to the database
          2. $_SERVER['SERVER_ADDR'] =>
            • Concept => Getting the Current Server Address
            • Importance and Usages => To get the Default server address , and used to get more in-detailed server address, and can be combined with if Conditional Statement
          3. $_SERVER['SERVER_NAME'] =>
            • Concept => Getting the Current Server Name
            • Importance and Usages => Can be used as testing script to get the Server Ip
          4. $_SERVER['QUERY_STRING'] =>
            • Concept => Getting the PHP string of the current Web page (String after ? Sign)
            • Importance and Usages => Usually used to check about certrain details within the page data
          5. $_SERVER['HTTP_REFERER'] =>
            • Concept => Getting the web page of the previous page Visitor come from
            • Importance and Usages => Usually usaed to count the Source traffic of thed page visitors , still common despite of non supporiting of some browsers
          6. $_SERVER['SERVER_PORT'] =>
            • Concept => Getting the Current Server PORT Nu,c545
            • Importance and Usages => Usually used for checking for port number errors such as [blocks] || [Not Valid]
          7. $_SERVER['REQUEST_METHOD'] =>
            • Concept => Getting the used method of the current request to reach the Database
            • Importance and Usages => Getting closer and more details about method used for connecting with database and server certain web page data (such as form)
          8. $_SERVER['DOCUMENT_ROOT'] =>
            • Concept => Getting the current server address untill {htdocs Folder}
            • Importance and Usages => Used comonly in any Server / Host
