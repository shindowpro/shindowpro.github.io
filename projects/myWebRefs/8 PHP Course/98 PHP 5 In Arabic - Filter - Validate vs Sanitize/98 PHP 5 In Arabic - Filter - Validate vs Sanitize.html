    • [Validate] => Only processing the Validation process according to assinged Creteria [Conditional Testing] , without [output]
    • [Sanitize] => Only processing the Sanitizing process according to assinged Creteria [output]

  • Advanced Filter Sanitizing Check method syntax as following :
      Default method {with using the Array}:
    1. filter_var(Input Var Name, $Filter Category & Type, $array(Assigned Filter Advanced Option-s && Filter Option Flag-s) , with following componants :
      • [Input Var Name] => The Defined input value income from the Form, with certain Datatype
      • [Filter Sanitize & Type] => Written Directly (without variables)
      • [Assigned Filter Advanced Option-s] => Could be written as :
        1. Pre-Defiend Array Variable
        2. Array with inner componants of advanced options and Flags {with the Flags in the same array}
      • [Filter Option Flag-s] of Assigned Filter Advanced Option-s => Could be written as :
        1. Pre-Defiend Array Variable
        2. Array with inner componants of advanced options and Flags {with the Advanced Options in the same array}
    2. In The Default method =>
      1. The Pre-Defined Array of the advanced options and Flags, both must be in one same array
      2. The Pre-Defined Array of the advanced options and Flags, could only include the advanced options
      3. The Pre-Defined Array of the advanced options and Flags, could only include the Flags of [Defualt] advanced options
      Direct method {without using the Array (only Direct Option and Flag)} :
    1. filter_var(Input Var Name, $Filter Category & Type, Assigned Filter Advanced Option-s , Filter Option Flag-s )
    2. filter_var(Input Var Name, $Filter Category & Type, $array(Assigned Filter Advanced Option-s && Filter Option Flag-s) , with following componants :
      • [Input Var Name] => The Defined input value income from the Form, with certain Datatype
      • [Filter Sanitize & Type] => Written Directly (without variables)
      • [Assigned Filter Advanced Option-s] => Could be Written as one of the following Methods:
        1. Written Directly {if it is NOT Default} && Could be passed if
        2. Not Written and be passed => If it's value is [Default]
      • [Filter Option Flag-s] of Assigned Filter Advanced Option-s => Written Directly
    3. In The Direct method =>
      1. We write the values Directly , and Default values advanced options could be passed
