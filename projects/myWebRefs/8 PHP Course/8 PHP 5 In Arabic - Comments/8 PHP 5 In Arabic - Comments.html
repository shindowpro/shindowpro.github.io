PHP [8 PHP5 - Comments] main Concept of Comments in general:

  1. [Comments] are a basic concept in all programing language
  2. Comments is a progaramatic tool to commenting a certian line or paragraph
  3. Comments of PHP language will NOT appaer in source page, {but comment of other HTMl or CSS will appear }
  4. Comments types are as following :
    • Single Line Comment => by using {//} || {#} once just before the desired line
    • Multiple Lines Comment => by using {/*} before the desired group lines & and {*/} after the group of lines
  5. Comments styles are as following :
    • Styles of [Single Line Comment] => as following:
      1. // ======= This is a singe line comment style1 [Normal] ======
      2. $var1 ='Txt'; // This is a single line Comment style 2 [Shell]
      3. # ======= This is a singe line commnet style2 [Normal] ======
      4. $var1 ='Txt'; # This is a single line Comment style 2 [Shell]
    • Styles of [Group Lines Comment] => as following:
      1. /*
        $var1 = 'text1';
        $var2 = 'text2';
        $var3 = 'text3';
        */
  6. Comments [Vip Tips] are as following :
    1. Nested group of same or different types of comments are not allowed , due it will cause problems , as following :

      /*
      /*
      These codes will be commented
      /*
      */

    2. It is allowed to write comment code inside an active in [php]

      < h1 >
      < ?php // This is a Comment line inside a php code ?>
      < /h1 >
