[PHP - Cookies - [ 85 PHP 5 In Arabic - Cookies - Set Cookie + Examples] => Concept:

  1. [PHP - set Cookies/methods syntax [setcookie() => The main Cookies Setting Syntax ] as following :
    1. Setting [Cookies] main Concept & Syntax :
      1. => A) Concept :
        • Set [Cookies] => {setcookie()} => Configuration/setting a new cookie file by it's Attributes
        • Set [Cookies] => {setcookie()} => Can set multiple Configuration of cookie file with the same Browser
        • Set [Cookies] => {setcookie()} => Are considered as an Array, and one of types of Super Globals Variables
        • Printing || Reviewing [Cookies] => Once it set, it can previewed by different methods :
          • Code Method =>
            1. Shortcut Method : {print_r($_COOKIES)} => This Method will only view the [Cookie Name] & [Cookie Value]
            2. Indetails Method : Will be studied later
          • Browser Interface Method =>
            1. Brower Settings => Privacy => Cookies Contents => Manage all Cookies
        • Deleting [Cookies] => Brower Settings => Privacy => Cookies Contents => Manage all Cookies
      2. => B) Syntax :
        • setcookie('Cookies Name', 'Cookies Value', time() + #number , 'Cookies (Range /path) Method' , Server domain, secure method , httpOnly ] Global Variable syntax as following:
          1. Cookies Name =>
            It is the name of file of cookies in the path of cookies {which is differing form a browser to anther }
          2. Cookies Value =>
            It is the Value of file of cookies in the path of cookies {which is differing form a browser to anther}
          3. Cookies time =>
            It is the activation duration time of the cookies in the browsered web site [can be set by adding values to current time]
            It is being set by the second (3600 = 1 hour) (86400 = 24 hour / Day)
          4. Cookies (Range /path) Method =>
            It is the range of accessing/applying the browsered web site
            "/" => All webpage
            "/inner page" => Inner webpage
          5. Cookies Server Domain =>
            The server domain or sub domain webpage , which will be applying the cookies File
            "Main webpage domain " => Cookies will be applied on all web page site and all it's inner page
            "Sub webpage domain " => Cookies will be applied on all web page site and all it's inner page [subdomain.maindomain]
            "localhost" => Cookies will be applied on webpage on the localhost
          6. Cookies Secure method =>
            The Securing method of the cookies File will be downloaded and saved on the pc
            "True" : Server will Send cookie file to the pc by using SSL ONLY [https]
            "False" : Server will Send cookie file to the pc by any method [http]
          7. Cookies http olny method =>
            "True" : Using only [http] language to deal with cookie file, and NOT using any other scripting langauges
            "False" : Using any scripting languages to deal with cookie file
        • setcookie() syntax writting Rules :
          1. "" => Must using double quotes NOT Single Quotes , to write the setting cookire file attirbutes
          2. ONLY [Name] || [Value] => Are oblidgated , and other are optional
          3. Any printing [cookie File] code => Will print all defined Cookie Settings, and arrangement is not important
