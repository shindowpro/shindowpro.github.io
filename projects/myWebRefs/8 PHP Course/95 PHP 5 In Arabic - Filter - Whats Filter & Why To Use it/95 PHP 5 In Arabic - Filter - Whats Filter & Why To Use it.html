[Filter - Whats Filter & Why To Use it {Intro} + Examples => Concept:

    Main Filter Concept:
  • One of the most important concepts and tool in the php languge [specially in Web security]
  • It is about Filtering input or output form and to the php script
  • Usually used for External Threatings protection inputs , which can be harmful to the server or Database
  • {Most Common Threatenings Sources } :
    1. Form Inputs
    2. Cookies
    3. Web Services => It is an advnaced [PHP Scripts] web technology
    4. SERVER Globals Variables => Which are affecting your server directly
    5. Database Results => Which are affecting your Database directly
    6. Emails => Which can affect each Database and Server directly
    7. URL => Which can affect each Database and Server directly
  • {Most Common filter_list Elements } :
    1. validate_url => To validate and filter the input as it is an Url link
    2. validate_email => To validate and filter the input as it is an Email
    3. validate_ip => To validate and filter the input as it is your website ip
    4. validate_mac => To validate and filter the input as they are one of Networking Identifications
    5. special_chars => To validate and filter the input as it is Tag scripting codes
  • {Default Browser Protection Policy} => There are Default protection Policy in each brower, as following :
    1. Concept :

      => These policies are only to protect your script from source Code threatinings input that can be read by your backend P.languages such as [PHP , MySQL].


      => These source code {javascript , PHP} threatinings can harm your Server || Database .


      => But unfourtionatily => These policies can NOt protect your from Tag elements codes {such as HTML, CSS} => So you need [PHP Filters] .


    2. Examples :
      1. [X-XSS]
      2. [Content_Security_Policy]
    Main Filter methods / Functions Syntax :
  1. filter_var(variable name , filtering mode) ; =>
    • Concept => as following :
      => To filter a certian variable come from a Form [what ever was it's Datatype] || to prepare it before to send to an external script
      => The filteration process can be determined according to certain creteria(s)
    • Syntax => as following :
      => [Variable name] => [Required] : It is the variable desired to be filtered upon certain Creteria.
      => [Filtering Mode] => [Required] :
      • It is the Filtering Mode or type => to be processed on the determined variable
      • Filtering modes can be as an Example :
        => FILTER_SANITIZE_STRING
        => other modes will be studied in further lessons
  2. filter_list(); =>
    • Concept => as following :
      => To display all elements can be filtered by filter Method
      => It is a built-in Array
    • Syntax => as following :
      => It is usually used with looping statments such as {foreach}
      => It can be used with {print_r} method
