PHP [26 PHP 5 In Arabic - Function Intro] main Concept:

[Function] => Is one of the most critical Concepts in any p.Languages.

[Function] => It is a group of procedures of scripts that can be called to exectuded Multiple times with certain condition .

[Function] => It has main two types :
.

  • Built in Functions => It is the kind of Function that is built in the main Languages syntax, that can be called but not edited
  • User pre-defined Functions {Customized Functions} => It is the kind can be Customized & edited & called

[Function] => It has main 4 Elements :

  1. Function key word: {function}
  2. Function Identifier: {function Name} => which is has the same standards of the varaibles Naming even the [Camel Case]
  3. Function Body: {function script content & procedures } =>
  4. Function Parameters / Arguments => They are the function attached varibles which can beused in function

  • More dynamic in programing and scripts ,by providing the editing ability only one time, and not have to edit all the scripts in content of the function
  • Function is not sensitive to the letter case , so it can be called by different letter cases, But it recommended to use the typical Case letter of the Declerations
  • Function in advanced method will include the {return} which provide the ability to log out certain Customized Return
