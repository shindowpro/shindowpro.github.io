PHP [6 PHP5 - Concatenation] main Concept of Concatenation in general:

  1. Concatenation are a basic concept in all programing language
  2. Concatenation is a merging more than one data in only One [whether it was raw data || Variables ]
  3. Concatenation Method can include follwoing cases :
    • Concatenating One Datatype with the same type, by using raw data or Varialbes
    • Concatenating One Datatype with the other type, by using raw data or Varialbes
    • Concatenating variables Name by using other variables value
    • Concatenating variables of elements values with all it's content with in declaring Variables
