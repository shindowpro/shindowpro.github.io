[PHP Array Methods - Search ] => Array Methods - Search Concept:

One of the most important methods of Array [Indexed || Associative] in PHP

Searching functionality is about one or More of following Tasks :

  1. Getting certain element's Index inside the array
  2. Getting certain element's Value inside the array
  3. Getting certain element's Index && Value inside the array

Searching functionality usgin one of the following element property :

  1. Element Value
  2. Element Index
  3. Element Keyname

[PHP Array Methods - Search ] => Array Methods - Search Methods Sytax && concepts :

  1. in_array(needle[element value], haystack[array Name] , type[true, false]);

    It is content of the following :

    • needle || [element value] => This is what are you looking for
    • haystack || [array Name] => This is what are you looking in
    • needle || [element value] => This is the type of searching by this method , and it is including :
      1. [True] => For the Identical Searching [ applying the datatype searching]
      2. [False] => For the Normal Searching [ not applying the datatype searching]

    Return of this Method is => The Element Index

    Usages :

    • For Checking for the existance of the element in as array
    • For printing the element value || index || both in a certain array



  2. array_search(needle[element value], haystack[array Name], type[true, false]);

    It is content of the following :

    • needle || [element value] => This is what are you looking for
    • haystack || [array Name] => This is what are you looking in
    • needle || [element value] => This is the type of searching by this method , and it is including :
      1. [True] => For the Identical Searching [ applying the datatype searching]
      2. [False] => For the Normal Searching [ not applying the datatype searching]
      7

    Return of this Method is => The Element Index

    Usages :

    • For Checking for the existance of the element in as array
    • For printing the element value || index || both in a certain array



  3. array_key_exists(needle[index||key], haystack[array Name]);

    It is content of the following :

    • needle || [element index || element key] => This is what are you looking FOR
    • haystack || [array Name] => This is what are you looking IN
    • needle || [element value] => This is the type of searching by this method , and it is including :
      1. [True] => For the Identical Searching [ applying the datatype searching]
      2. [False] => For the Normal Searching [ not applying the datatype searching]

    Return of this Method is => The Element Index

    Usages :

    • For Checking for the existance of the element in the array
    • For printing the element value || index || both in a certain array
