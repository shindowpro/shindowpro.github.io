[PHP Array Methods - Sorting ] => Array Associative Methods - Sorting Concept:

{Sorting elements in the array} is One of the most important Controling methods of the Array [Indexed || Associative || Multidimensional] in PHP

{Sorting Methods} is deffer from an array type to another in PHP, In this lesson will be take the Sorting Array Elements in the [Associative Array]

Sorting elements methods to the array as following :

  1. {asort();} Method => [sorting values] :
    1. {asort();} Method:
      • asort(array Name) => The Default sorting Method in the array's Elements's Values {alphabatically}
      • asort(array Name, Sorting Type) => The advanced sorting Method in the array's Elements's Values according to {certain creteria} as following :
        1. asort(array Name, Sorting_REGULAR) => The advanced sorting Method in the array's Elements's Values according to {Alphabatical creteria} Starting from String datatype elements , then other datatypes
        2. asort(array Name, Sorting_STRING) => The advanced sorting Method in the array's Elements's Values according to {Alphabatical creteria} with dealing with all elements as String datatype
    2. {arsort();} Method:
      • arsort(array Name) => The Default Reverserly sorting Method in the array's Elements's Values {alphabatically}
      • arsort(array Name, Sorting Type) => The advanced Reverserly sorting Method in the array's Elements's Values according to {certain creteria} as following :
        1. arsort(array Name, Sorting_REGULAR) => The advanced Reverserly sorting Method in the array's Elements's Values according to {Alphabatical creteria} Starting from String datatype elements , then other datatypes
        2. arsort(array Name, Sorting_STRING) => The advanced Reverserly sorting Method in the array's Elements's Values according to {Alphabatical creteria} with dealing with all elements as String datatype


  2. {ksort();} Method => [sorting keys] :
    1. {ksort();} Method:
      • ksort(array Name) => The Default sorting Method in the array's Elements's Keys {alphabatically}
      • ksort(array Name, Sorting Type) => The advanced sorting Method in the array's Elements's Keys according to {certain creteria} as following :
        1. ksort(array Name, Sorting_REGULAR) => The advanced sorting Method in the array's Elements's Keys according to {Alphabatical creteria} Starting from String datatype elements , then other datatypes
        2. ksort(array Name, Sorting_STRING) => The advanced sorting Method in the array's Elements's Keys according to {Alphabatical creteria} with dealing with all elements as String datatype
    2. {krsort();} Method:
      • krsort(array Name) => The Default Reverserly sorting Method in the array's Elements's Keys {alphabatically}
      • krsort(array Name, Sorting Type) => The advanced Reverserly sorting Method in the array's Elements's Keys according to {certain creteria} as following :
        1. krsort(array Name, Sorting_REGULAR) => The advanced Reverserly sorting Method in the array's Elements's Keys according to {Alphabatical creteria} Starting from String datatype elements , then other datatypes
        2. krsort(array Name, Sorting_STRING) => The advanced Reverserly sorting Method in the array's Elements's Keys according to {Alphabatical creteria} with dealing with all elements as String datatype
