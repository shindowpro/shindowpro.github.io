[PHP Array Methods - Array Fill ] => Indexed Array Methods - Concept:

  1. Array Fill Methods of [Indexed array] as following :
    1. {$newArray = array_fill(index Number , Elements number, single value )} => Solo Filling method

      This Method for filling array with {united value} for each elements && starting with certain index [positive || negative] && certain elements number.

      This Method used for filling the array with one value.

    2. {$newArray = array_fill(index Number , Elements number, array() )} => Multi Filling by array

      This Method filling array with {seperated array elements} for each elements of main array && starting with certain index [positive || negative] && certain elements number.

      This Method used for defining Multidimensional array => by filling the array with another array's elements .

    3. {$newArray = array_fill(index Number , Elements number, array_fill() )} => Multi Filling by Fill inside Fill method

      This Method filling array with {another fill array method (with it's own setting) } for each elements of main array && starting with certain index [positive || negative] && certain elements number.

      This Method used for defining Multidimensional array => by filling the array with another array fill Method.
