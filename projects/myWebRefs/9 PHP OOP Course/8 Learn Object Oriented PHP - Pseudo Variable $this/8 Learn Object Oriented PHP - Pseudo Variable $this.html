[PHP OOP Class Methods ] Concept:

=> PHP OOP [Pseudo Variable $this] :
  • Main Concepts =>

    [$this] : Is one of Pseudo Variable which is (Fake) variable , that can be used Class , to Prefer to the [Object]

    {$this} : Is being used usually in Class Methods , to refers to the Object and then to call onmme of the object properties, that had been defined inside the Class

    {$this} : Is Requiring a defined Property inside the Class to use it in accessing the properites

    {$this} : It is like a Joker Variable to refer to defined Class Object a defined Property inside the Class to use it in accessing the properites

    {$this} : Is commenly used in any P.language supporting the OOP (like JSOOP) and FrameWorks and Libararies (J-Qeury || DOM || BOM)

    {$this} : Is being used as a gate to access to any Object's property

    The Alternative way of using [$this] variable inside the Class Method is => To use function Parameters
