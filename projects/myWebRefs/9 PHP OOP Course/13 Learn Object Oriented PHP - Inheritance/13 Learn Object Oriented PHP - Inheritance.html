[PHP OOP Inheritance] Concept:

=> PHP OOP [Inheritance] :
  • Main Concept =>

    It is a system to copying all Class Properties & Methods to another class

    It is usually used by any OOP System

    This system allowing beside copying => to add addition Properties & Methods in the new Class

    The Inherited Method can not be changed in the new Class

PHP [PHP OOP 13 Learn Object Oriented PHP - Inheritance] Importance & Benefits :

[Inheritance] => Is very common and criticaly to use in saving time in creating new Class with any OOP application and PHP script
