[PHP OOP Class Constants ] Concept:

=> PHP OOP [Class Constants] :
  • Main Concepts =>

    [Constant] : It is a fixed Dimension of value that can NOT be changed any more after Defining

    [Constant] : It is existed in any P.language

    [Constant] : Can be used/called within Class Methods Definging , and also can be used/called inside/outside Class


  • Syntax Recommendation =>

    [Constant] : The main keyword used to define the Constant is => {const}

    [Constant] : To call/use ther Class (Inside) the Class Method, must use keyword => {self} & {::} & {Constant Name}

    [Constant] : To call/use ther Class (Outside) the Class Method, must use the => {Class Name} Or {Object Name} & {::} & {Constant Name}

A) Defining new Class {ClassName} with [default properties values] & [Method using ($this) Pseudo Varable] :
class ClassName{
// Class Properties Definging (Variables + Constants) [with Null Properties Values] :
var propertyName1 = "value" ;
public propertyName2 = 'value' ;
private propertyName3 = 'value';
protected propertyName4 = 'value';
const CONSTANTNAME = 'value';
// Class Method Defining (with Using $this) to access to a certain Property :
// Class Method Defining (using the Class properites variables) :
public function MethodName1() {
echo 'The First Property is ' . $this->propertyName1 ;
echo 'The Second Property is ' . $this->propertyName2 ;
echo 'The Third Property is ' . $this->propertyName3 ;
}

// Class Method Defining (using the Class properties Constant) :
public function MethodName2() {
echo 'The Constant Value is ' . self :: CONSTANTNAME ;
}
}

Defining/Creating new Object {objName} with All Defined Properties & Methods :
$objName= new ClassName();
Calling / Proccessing Object {objName}'s Metods [of properties (variables)] :
$objName-> MethodName1();

Calling / Proccessing Object {objName}'s Metods [of properties (Constant)] :
$objName-> MethodName2();

Calling / (Constant) [using the object Name] :
$objName :: CONSTANTNAME ;

Calling / (Constant) [using the Class Name ] :
echo ClassName :: CONSTANTNAME ;
