[PHP OOP Class Properties] Concept:

=> OOP [Class Properties] :
  • Main Concept =>

    It is a variable defined inside the Class , to be applied with in all Created objects from this Class

    These properties can be with (Null) values or with (default values)

    Any new created Objects will have a copy from this Class

    Any property must has its prior visiblity marker {will be studied next lesson}, which will detect the visiblity Mode property of this property

  • Visibiltly Markers can be one of following type =>
    1. [public] -> Can be seen for any one and any where
    2. [private] -> Can be seen in certain position only
    3. [protected] -> No one can see them
