[ 14 Learn Object Oriented PHP - Inheritance Override Training] Concept:

=> PHP OOP [Inheritance Override] :
  • Main Concept =>

    It is a feature/abilty of Editing or Modifying the Inherited Method(s) Contents , [without Modifying the Method Structure (Parameters)]

    In the Inheritace Overriding => It is able to add new methods or a properties in the new Inherited Class

    There is an ability to add SOLO Property specified for only an Object , by Direct Object Defining

  • New Concepts :
      [Super Class] || [Parent Class] || [Main Class] => It is the Basic Class that other Classed being Inherited from
      [Sub Class] || [Child Class] => It is the Inherited Class from the main/super/parent Class

PHP [PHP OOP 14 Learn Object Oriented PHP - Inheritance Override Training] Importance & Benefits :

[Inheritance Override] => Ablity to Edit/Modify the inherited Method(s) contents, with out the main method Structure (Parameters)

[Inheritance Override] => Ablity to Edit/Modify the inherited Property(s) value, with out the main property Structure (Name)

[Direct Object Property Definition] => Ablity to Solo Adding new specified property for a certain Object
