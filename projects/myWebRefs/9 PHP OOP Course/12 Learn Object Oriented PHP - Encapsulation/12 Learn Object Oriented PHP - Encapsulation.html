[PHP OOP Encapsulation] C1oncept:

=> PHP OOP [Encapsulation] :
  • Main Concept =>

    It is a system to Securing Data by only accessing it by using certain method , and not accepting the direct method

    It is usually used by any OOP System

    It is also used beside the Incripting method {sha1()}

    This system prevent accesing to the Encapsulated peroptery directly , but only by using the specfied Class Method created for this puropose
