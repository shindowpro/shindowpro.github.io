=> OOP [Class Properties Change] :
  • Main Concepst =>

    It is avaiable to Change the properties of the defined Objects after create it from the class

    The Object's properties can be as class default OR be Customized after creating the Object {by using -> object assinging porperty opertaor } [without using the $ sign ]

=> [Class] & [Object] main Syntax as following :

Syntax:

< ?php
Defining new Class {ClassName} [with default properties values] :
class ClassName{
public propertyName1 = "value" ;
public propertyName1 = 'value' ;
private propertyName2 = 'value';
protected propertyName3 = 'value';
}

Defining new Class {ClassName} [with non properties values] :
class ClassName{
public propertyName1;
public propertyName1;
private propertyName2;
protected propertyName3;
}

Defining new Object {objName} with Null parameter value :
$objName= new ClassName();
Changing / Assinging Object {objName} with Customized parameter value :
$objName-> propertyName1 = "value";
$objName-> propertyName2 = "value";
$objName-> propertyName3 = "value";

Printing the Var_dump of new created Object :
echo < pre > ;
echo var_dump($objName);
echo < pre > ;
