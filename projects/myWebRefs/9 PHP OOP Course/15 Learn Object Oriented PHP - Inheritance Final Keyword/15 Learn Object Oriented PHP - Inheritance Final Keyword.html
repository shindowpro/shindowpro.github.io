[ 15 Learn Object Oriented PHP - Inheritance Final Keyword] Concept:

=> PHP OOP [Inheritance - Final] :
  • Main Concept =>

    It is a Feature/Abilty of Protecting/Preventing the Class || Method from being Extended or Inherited or Modified

    [Final Class] =>

    • Prevent the Class from being inherited OR Extending , and then protecting it from being modified
    • If you/other developer tried to Iheritate a Final Class => Will produce an Error

    [Final Method] => Prevent the method from being inherited OR even being re Written, and then protecting it from being modified =>

    • Prevent the Class from being inherited OR Overrdiing OR even Re Written , and then protecting it from being Modified/Overriding
    • If you/other developer tried to Iheritate / Re-wirte a Final Method in other Class => Will produce an Error

PHP [PHP OOP 15 Learn Object Oriented PHP - Inheritance Final Keyword] Importance & Benefits :

[Inheritance - Final] => Ablity to Protecting data in the class , and then prevent Wrong Extending or copying , and Manipulating

[Inheritance - Final] => This ability is provided for each Class and Method
