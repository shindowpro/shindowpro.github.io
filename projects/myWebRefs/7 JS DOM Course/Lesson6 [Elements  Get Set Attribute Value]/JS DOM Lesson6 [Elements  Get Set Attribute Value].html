JS DOM [Elements Get Set Attribute Value] Concept:

=> JS HTML DOM [Elements Get Set Attribute Value]: There are many methods to set-get Element's Attribute's Value.

=> JS HTML DOM [Elements Get Set Attribute Value]: in this lesson we will study only the Direct of set & get element attribute's Value as following:.

  • [Get Function - Direct Method] => {DefinedElement.attributeName;}
    1. => Used to Get certain defined element's Attrbute's value - even if it was empty
    2. => If the Element's Attribute was empty or not even written => it get an empty value
  • [Set Function - Direct Method] => {DefinedElement.attributeName = 'new Value';}
    1. => used to set/replace certain Defined element's Attribute's Value
    2. => If the Element's Attribute was empty or not even written => it will be re-defined
