JS DOM [Element Children ChildNodes] Concept:

=> JS HTML DOM [Element Children ChildNodes]: Every parent Element has Childeren elements can be controled by several Methods.

=> JS HTML DOM [Element Children ChildNodes]: Differeneces between {childeren} & {ChildNodes} as following:

  • {children} => The content only the child Elmemnts inside the parent Element
  • {childNodes} => The content ALL parent elements & any other thing and that contain following :
    1. [child Elements] => which are child tags
    2. [child text] => That include each text & empty spaces
    3. [child empyt Spaces] => That will count as text also
    4. [comments] => which is the HTML Comments

=> JS HTML DOM [Element Children ChildNodes]: Js dealing with the parent Elements s and its childeren as an Object, and each child element will be as one of it;s property

=> also each child element will be dealt as object, and each property is one of it's property

=> JS HTML DOM [Element Children ChildNodes]: Syntax as following:.

  • [{elementName.children;}] =>
    1. => This Method to get all elements child as an object
  • {elementName.children.length;}
    1. This Mehtod to get all child Elmemnts (elments only) count of ceratain parent Elemnet
  • {elementName.childElementCount;}
    1. This Method also to get all child Elmemnts (elments only) count of ceratain parent Elemnet as it is (elementName.children.length)
  • {elementName.children[x];}
    1. This Method to get certain child element of parent elemnt by it's index
  • {elementName.children[x].textContent;}
    1. This Method to get certain child element's (textContent) property
  • {elementName.children[x].innerHTML;}
    1. This Method to get certain child element's (innerHTML) property
  • {elementName.children[x].className;}
    1. This Method to get certain child element's (className) property
  • {elementName.children[x].id;}
    1. This Method to get certain child element's (id) property
  • {elementName.childNodes;}
    1. This Method to get parent Element's all Nodes with types [elements + text + empty spaces + comments] as an Object
  • {elementName.childNodes.length;}
    1. This Method to get parent Element's all Nodes with types's count number
