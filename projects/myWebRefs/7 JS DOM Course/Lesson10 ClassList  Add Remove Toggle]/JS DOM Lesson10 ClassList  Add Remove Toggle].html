JS DOM [Elements ClassList Add Remove Toggle] {part2} Concept:

=> JS HTML DOM [Elements ClassList Add Remove Toggle]: There are are ability to has several Class Names for the same Element, and [classList] is an a tool to control the Class Names of this Element.

=> JS HTML DOM [Elements ClassList Add Remove Toggle]: Syntax {part2} as following:.

  • [{elementName.ClassList.add('className');}] =>
    1. => This Method add a new class into the Element's ClassList {with remaining on the previous ClassNames}
    2. => This Method will Not Add the Class Name if It already existed in Element's Class List
  • [elementName.ClassList.remove('ClassName');] =>
    1. This method will Remove the Class Name from the Class List {if it Existed in class List}
  • [elementName.ClassList.toggle('ClassName');] =>
    1. This method will Add-remove the Class Name from the Class List [add=> if it not existed] [remove if it existed]

=> JS HTML DOM [Elements ClassList Add Remove Toggle] {part2}: These Methods can be done previous Methods, but it make it easer.
