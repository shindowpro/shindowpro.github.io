JS OOP [Find Elements By Object] Concept:

=> JS HTML DOM [Find elements by Object]: It refers to accessing elements in web page as an object(s) as following:

  • {main Parent Object} => which is the main object's name, which has an index(due to there are Many elements in page)
  • {Child} => Which is each the a delt as a property inside the main object {parent element}, it be accessd by it's pragraming name {attribute : name}
  • {Other child's properties } => Which are the property's values {child attributes}.

=> JS HTML DOM [Find elements by Object]: Is one of accessing Methods to elements in Web page , with differences in syntax

[ JS HTML DOM Find Element by Object ] Syntax & Methods:

=> JS HTML DOM [finding & Accessing Elements by Object Methnod] Conceptual Syntax as following :

// A) JS HTML DOM Defining variable with certian Element of web page by [DOM] Syntax as following :

var ElementVar = document.getElementById('ElementID');
//===========================================================

// B) JS HTML DOM [changing Defined element Value by {Documnet title}] Syntax as following :

ElementVar.innerHTML = document.title;
//===========================================================

// C) JS HTML DOM [changing Defined element Value by {Documnet images}] Syntax as following :

// 1) Adding images {Count} in Whole page :
ElementVar.innerHTML = document.images.length;
//===========================================================

// 2) Adding [certain image]'s {source} :
ElementVar.innerHTML = document.images[x].src;
//===========================================================

// 3) Adding [certain image]'s {width} :
ElementVar.innerHTML = document.images[x].width;
//===========================================================

// 4) Adding [certain image]'s {height} :
ElementVar.innerHTML = document.images[x].height;
//===========================================================
//===========================================================

// D) JS HTML DOM [changing Defined element Value by {Document certian form's child element's Attribute}] Syntax as following :

// 1) Adding an input's{value} of Certain Form in web Page :
ElementVar.innerHTML = document.forms[index].input'Name.value;
//===========================================================

// 2) Adding an input's{type} of Certain Form in web Page :
ElementVar.innerHTML = document.forms[index].input'Name.type;
//===========================================================
