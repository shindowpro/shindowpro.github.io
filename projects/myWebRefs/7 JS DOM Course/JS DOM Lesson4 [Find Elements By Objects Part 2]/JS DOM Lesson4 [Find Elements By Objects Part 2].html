JS DOM [Find Elements By Object part2] Concept:

=> JS HTML DOM [Find Elements By Object part2]: They are another group of Methods to Accessing/Editing elements in web page {as an object(s)} as following:

  • {document.body;} => which is the an Object name of {body} parent Element , and it contains all other child elements in web page , and include following Mehtods :
    1. document.body.innerHTML; => This is the Mehtod of body's interanl HTML Coding Text
    2. document.body.innerText; => This is the Mehtod of body's interanl Text String only {text Node} of Elements inside the Body Parent Element, and it can be accept all JS string Methods such as :
  • {document.body.innerText.indexOf('searching text');} [any element.innerText can accept JS String methods]
  • document.body.anchor; => This Method has been deprecated & it is to find all defined links <a> in page with {name="validName"}
  • document.body.links; => this Method to find all hyper reference links <a> in page with {href="#validlink"}

[ JS HTML DOM Find Element by Object part2 ] Syntax & Methods:

=> JS HTML DOM [finding & Accessing Elements by Object Method] Conceptual Syntax as following :

// A) JS HTML DOM Defining variable with certian Element of web page by [DOM] Syntax as following :

var ElementVar = document.getElementById('ElementID');

// B) JS HTML DOM [setting Defined element Value by {Document body element HTML Text Coding }] Syntax as following :

ElementVar.innerText = document.body.innerHTML;

ElementVar.innerText = document.body.innerText;

if (document.body.innerText.indexOf('Searching Text') > -1) {

console.log('your Text is Not Existed!');
}
