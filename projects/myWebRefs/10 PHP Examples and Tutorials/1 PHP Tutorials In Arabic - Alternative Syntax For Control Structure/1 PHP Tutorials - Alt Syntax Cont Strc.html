=> PHP Tutorials [Conceptual Examples] :
  • Alternative Controling Structure Mechanism => can be applied as following :
    > Removing the Curluy Brackets [ {} ] and replace it by
    1. [ : ] => SemiColoumn => Instead of the Starting Curly Bracketes [ { ]
    2. [ end... ] => End + FunctionName => Instead of the Ending Curly Bracketes [ } ]
    > Alternative Controling Structure Applying Methods and Functions => can be applied on the following [Controling] Functions and Methods :
    1. [ if(){} ] => If Condition && Ifelse && else
    2. [ for(){} ] => End + FunctionName => Instead of the Ending Curly Bracketes [ } ]
    3. [ foreach(){} ] => End + FunctionName => Instead of the Ending Curly Bracketes [ } ]
    4. [ while(){} ] => End + FunctionName => Instead of the Ending Curly Bracketes [ } ]
    5. [ switch(){} ] => End + FunctionName => Instead of the Ending Curly Bracketes [ } ]
