=> PHP Tutorials [Parse Url And Get Components] Concept :
  • PHP Tutorials [Parse Url And Get Components] =>

    To control the url's components, by using certain functions, and certian procedures, as following :

    • Convert the url string to the parsed url video components, upon the assigned Flag by => [parse_url(Url String Variable, Url Flag );] => and Url Components[Flags] can be one of the following:
      1. [PHP_URL_QUERY] =
        • This Flag will get the Url Query
        • Any => &argument=value
        • which is any value after {&} sign and {=}
      2. [PHP_URL_PASS]
      3. [PHP_URL_PATH]
      4. [PHP_URL_USER]
      5. [PHP_URL_PORT]
      6. [PHP_URL_HOST]
      7. [PHP_URL_SCHEME]
      8. [PHP_URL_FRAGMENT]
    • This method is applicable with all kinds of Video URL [youtube , facebook, Daily Mototion]
    • Converting the parsed variable to an array [Key Array with url component name and it's value] , as following [as examples]
      • ['v'] => url Video ID => This Component can be used in i-frame coding to get Dynamic Generated url embed code
      • ['list'] => url Video list Name
      • ['start_radio'] => url Video list Name

  • PHP Tutorials [Embeded] concept =>

    A method to merge a certain video of another website with the current page => using [I-frame] HTML Tag

    This Method can be used in any video whether it was a single or in a playlist, and in any website

    Syntax of the Embed Video :
    [ https://www.domain.com/embed/VideoId ]

    The Video id used to embed it inside a I-frame html Tag => to merge a video from another website to the current webpage
