PHP [9 PHP Examples In Arabic - Use Nested If Condition] Concept:

=> PHP Tutorials {Use Nested If Condition} Concept :

  1. Condtional If is one of the most critical function in any language
  3. Condtional If => Can be one of the following types :
    • [One condition If Statement]: [ if(){} ]
      1. It is the condition if statement with only one conditon , with no other conditons {elseif}
      2. Usually used to test one condition with out any other probabilities
    • [Multiple conditions If Statement]: [ if(){} elseif(){} elseif(){} ]
      1. It is the condition if statement with myltiple conditons , with other conditons {elseif}
      2. Usually used to test Multiple conditions with other probabilities in the same statement
    • [Multiple conditions + Negative Condition If Statement]: [ if(){} elseif(){} elseif(){} else{} ]
      1. It is the condition if statement with myltiple conditons , with other conditons {elseif} , with the negative condition
      2. Usually used to test Multiple conditions with other probabilities in the same statement , with the negative probability
    • [One condition + negative If Statement]: [ if(){} else{} ]
      1. It is the condition if statement with only one conditon & and probability for the negative condition , with no other conditons {elseif}
      2. Usually used to test one condition with out any other probabilities
  4. Condition If statment Writing Methods :
    1. Ordinary method [using the Curely brackets] => Long form
    2. Ordinary method [using the Curely brackets] => Short form
    3. Alternative method [using the (:) & (endif;) brackets]
  5. Recommondation to use each type of if Statement :
    1. Checking for certain value and taking certain procedures
    2. Filtering some values and taking certain procedures
