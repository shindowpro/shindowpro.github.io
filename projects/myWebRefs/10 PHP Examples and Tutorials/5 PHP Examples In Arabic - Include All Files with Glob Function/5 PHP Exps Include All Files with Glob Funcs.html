> New Methods in this lesson are as following :

  1. glob('path/single File name.extension') => To import one Single file upon it's path
  2. glob('path/*.extension') => To import one Type of Files upon it's path
  3. glob('path/*.{extension1,extension2, extension3}' , GLOB_BRACE) => To import Multiple Type of Files upon it's path
  4. foreach (glob('path/*.extension')) => To loop inside a certain path to reach and deal with one type of Files inside
  5. foreach (glob('path/*.{extension1,extension2, extension3}' , GLOB_BRACE)) => To import Multiple Type of Files upon it's path
