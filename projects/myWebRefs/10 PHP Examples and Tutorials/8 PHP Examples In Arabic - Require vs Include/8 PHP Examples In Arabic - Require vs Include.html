PHP [8 PHP Examples In Arabic - Require vs Include] Concept:

=> PHP Tutorials {Require vs Include && Require_once vs Include_once} Concept :

  1. Requiring & Including Files is very important to save time of developer by directly importing files and get access to all it's defined variables
  3. [Requiring] & [Including] => The same identical Functionality by including , Main differences between [Require] & [include] :
    • Requiring & including unexisted file :
      1. [Require] => will Through an {Error-Fatal Error} if the file is NOT existed in the defined given path
      2. [Include] => will Through an {Warning} if the file is NOT existed in the defined given path
    • Loading the page after :
      1. [Require] => will {Halt} the page , and will NOT continue the Page {if the file is not existed} => And produce an Error : (if there are no escaping Error written before the include)
      2. [Include] => will {Continue} the page {if the file is not existed} => And produce an Warning: (if there are no escaping Error written before the include)
  4. Recommondation to use each type of include and require && include_once and require_once :
    1. [Include] & [Include_once] :

      Used when the desired file to include in the page -> Is not critical to properly operate the page such as (simple message|| Artical || ..)

    2. [Require] & [Require_once] :

      Used when the desired file to requried in the page -> Is a critical to properly operate the page such as (Laibrary|| Format Scripting || Functionality ..)

  5. It is Recommneded to use the Scaping Error Before Including || Requiring Files to avoid displayiong Error or Warnings to the user
  6. It is Recommneded to use the [once] instead of the regular include or require => in case of the inlcuding || requiring the file just for one time
  7. In case of using the both method of one family [include && include_once] || [require && require_once] => to take in consideration the position of each method, as following :
    1. Usage of the [main] method at first before the [once] will prevent the process the second of [once] method
    2. Usage of the [once] method at first before the [main] will allow the process of the second of [main] method
  8. Including and Requiring for format files [as text Contents] will only add it as syntax codes only [not as scripting], but when using the file name in an embeded code {link} => this will add it's Effects as Code Script
  9. To prevent the probable Error or Warning => We can use one of the following Metjhods :
    • Canceling the Error in syntax by using the Scaping Error Method
    • Controling the Error in following Syntax by using the Conditional If [in case of the file is not existed]
