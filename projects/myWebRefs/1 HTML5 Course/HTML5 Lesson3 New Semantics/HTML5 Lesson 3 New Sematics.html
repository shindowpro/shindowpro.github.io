HTML5 [ New Semantics ] Main Concept

> There is major classification to Tag's Name by its meaning.[Semantic] and [Non Semantic]

> [Semantic] Tags: It's name indicates the tag function.

> [Non Semantic] Tags: It's name Does NOT indicate the tag function.

> HTML5 has Different new semantic tags [Semantics].

New HTML5 Semantic:[ Header ]

New Semantic:[ Header ] Concept:

Instead of using Div tag and give it class with "Header", in HTML5 there is a [Header] semantic tag already.

New Semantic:[ Header ] Syntax:

HTML Coding Page Editor:
< Header >   </ Header >
