  1. HTML5 Tag [SVG - Rect , Circle] Concept :

    2. [Rect] <= Refers to Rectangle Shape sas a Element inside the SVG which can be written by Code of HTML5

    [Circle] <= Refers to Circle Shape as a Element inside the SVG which can be written by Code of HTML5

    [SVG] <= can be created in (Stand alone Sepreated File) with SVG extension (file.svg) and be re-Called from other Web page Document or in the same HTML page in other location

    [SVG] <= All SVG Elements Codes must be self-Closed Tag due to XML Syntax , Except the SVG Tag itself which is being delt as HTML Tag

  2. HTML5 Tag [SVG - Rect, Circle ] Syntaxes :

    3. [SVG] main Atttributes as following :

    • width , height
    • xmlns > = which is the name space of the XML Page properites ("http//:www.w3.org/2000/svg")
    • There are so many Attributes of SVG Tag, and will be studed in later lessons

    [SVG] Main elements as following :

    1. < rect /> : Which is the Rectangle shape as an internal Element inside the SVG Tag

      2. The Rectangle SVG Tag Element has the Following Attributes :

      1. [x]= "" ,[y] = "" Attributes <= Which are the Coordinaition of Rectangle Start point position inside the SVG
      2. [width]= "" ,[height] = "" Attributes <= Which are the Dimensions of Rectangle inside the SVG, and it could be as a dare Numeric Value / Percentage of the SVG Tag in the Webpage
      3. [[rx]= "" ,[ry] = "" ]Attributes <= Which are the Curvy Border property of Rectangle Start inside the SVG
      4. [fill =""] Attributes <= Which are the Filling Style (Method & Color) property of Rectangle Start inside the SVG
    2. < circle /> : Which is the Circle shape as an internal Element inside the SVG Tag

      3. The Circle SVG Tag Element has the Following Attributes :

      1. [cx]= "" ,[cy] = "" Attributes <= Which are the Coordinaition of Center Start point of Circle position inside the SVG
      2. [r]= "" <= Is the main diameter of the Circle inside the SVG
      3. [fill =""] Attributes <= Which are the Filling Style (Method & Color) property of Circle Start inside the SVG
