Form Input New field [ < input type = "color" > ]

Form Input New field [Color] Concept:

=> [Color Field] is one of the new fields in HTML5 and Not supported in each Internet Explorer and Safari untill 2014.

=> [Color Field] has (2) main Attributes :

  • Name => which is the programable name for the input element of the form
  • Value => which is the Defualt Color been existed in the Color tool on page one page is loaded, And it must be written with long Hexa Color Code (6 Digits)

=> [Color Field] is mainly made to send a Color value with in the form to the server by user, and it usually followed by [Submit] button.

Main & Features of Form Input [Color Field] Usages:

=> It can be be programatically used in Events to as an actions or reactions to change the color of page or element/s using Javascript codes.

Example:

Selecting specific color by user from the Color tool input field

Select your favourite Color:

<section>
   <h3> Select your favourite Color: </h3>
   <form>
     <input   type = "color"  name = "theme"  value = "#FF0000" >
     <input   type = "sumbit"  name = "submit-color"  value = "send" >
   </form>
</section>
Form Input New field [ < input type = "range" > ]

Form Input New field [Range] Concept:

=> [Range Field] is one of the new fields in HTML5 and that allowing input a specific Value from Value Range Bar.

=> [Range Field] has (5) main Attributes :

  • Name => which is the programable name for the input element of the form
  • Value => which is the Defualt Numeric Value Auto selected Range tool on page once the page is loaded.
  • Min => which is the Minimum accepted value of the range tool bar, that the tool will not accept any less numeric value.
  • Max => which is the Maximum accepted value of the range tool bar, that the tool will not accept any greater numeric value.
  • Step => which is specify the amount of step when changing the value of Range tool.

=> [Range Field] is mainly made to send a Numeric value with in the form to the server by user, and it usually followed by [Submit] button.

Main & Features of Form Input [Range Field] Usages:

=> It can be be programatically used in Events to as an actions or reactions to change the Value of certain input or element/s of page using Javascript codes.

Example:

Selecting specific Value from by user from the Range tool Bar input field

Select your a number from range [0-10] (step = 1):

<section>
   <h3> Select your a number from range [0-10]: </h3>
   <form>
     <input   type = "range"  name = "Choose-Value"   min = "0"   max ="10"               step = "1"  value = "1" >
     <input   type = "sumbit"  name = "submit-Value"  value = "send" >
   </form>
</section>

Form Input New field [Number] Concept:

=> [Number Field] is one of the new fields in HTML5 which allowing to input a numeric value from up/down selector tool .

=> [Number Field] has (5) main Attributes :

  • Name => which is the programable name for the input element of the form
  • Value => which is the Defualt Numeric Value Auto selected of Number Selector tool on page once the page is loaded.
  • Min => which is the Minimum accepted value of the Number Selector tool, that the tool will not accept any less numeric value.
  • Max => which is the Maximum accepted value of the Number Selector tool, that the tool will not accept any greater numeric value.
  • Step => which is specify the amount of step when changing the value of Number Selector tool.

=> [Number Field] is mainly made to send a Color value with in the form to the server by user, and it usually followed by [Submit] button.

Main & Features of Form Input [Number Field] Usages:

=> It can be be programatically used in Events to as an actions or reactions to change the color of page or element/s using Javascript codes.

=> It mainly an east way to input a numeric value and send it with in form to server.

Example:

Selecting specific color by user from the Color tool input field

<section>
