  1. HTML5 Tag [SVG -Ellipse Line] Concept:

    1. [SVG- Ellipse] <= It is a Shape of Semi Circle that have 2 Dimensions horizontal & Vertical, and has the following Attributes:

      • [cx , cy] => These are the Ellipse Shape Coordinations, (cx) Horz & (cy)Vert, and they have Numeric values
      • [rx , ry] => These are the Ellipse Shape Dimensions, (rx) Horz & (ry) Vert , and they have Numeric values
      • [fill = ""] => This is the Fill Style Format of the Ellipse Shape, and it has a color value
      • [stroke = ""] => This is the Fill Style Format of the Ellipse Shape, and it has a color value
      • [stroke-width = ""] => These are width of the Ellipse borders
      • [opacity = ""] => This is the Degree of Appearance of the Ellipse Shape

    2. [SVG- line] <= It is a Straight line , and has the following Attributes:

      • [x1 , y1] => These are the Start Point Line Shape Coordinations, (x1) Horz & (y1)Vert, and they have Numeric values
      • [x2 , y2] => These are the End Point Line Shape Coordinations, (x2) Horz & (y2) Vert, and they have Numeric values
      • [stroke = ""] => This is the Stroke Style Format (Line color) of the Line Shape, and it the only Format that the line has to be ( Shown inside the SVG) ,and it has a color value
      • [stroke-width = ""] => These are width of the line, and it has a numeric Value
      • [opacity = ""] => This is the Degree of Appearance of the Ellipse Shape
