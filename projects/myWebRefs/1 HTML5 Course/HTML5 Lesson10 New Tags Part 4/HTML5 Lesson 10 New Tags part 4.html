Details Tag [ < details > < /details > ]

[Details Tag] Concept:

=> [Details Tag] commonly wide used in sites as a show more details button (Spoilers) in Articles and paragraphes.

=> [Details Tag] function usually was formated and coded by Javascritp and J-Querry, but this advanced tag is already coded!

=> [Details Tag] has 2 common Attributes:

  1. [< Summary > < /Summary >] => to specify certain title instead of details (defaulted).
  2. open => to make it open stauts when loaded (Expanded contents).

Main & Common Usages:

=> It is a usful tool in Article's content capsulating or compressing function.

=> Commonly used in Showing/Hiding Certain content in page from being visible or Non Vislible upon user.

Mark Tag [ < mark > < /mark > ]

Main Concept:

=> This is the Text highlighting Tag property, and it can be further formated by CSS.

Main & Common Usages:

=> [Mark] Tag can be used to highlight the important Contents or Words or even Paragraph to give them special Attraction with different Colors, backgrounds, effects.

Example / Highlighting vip Words:

This is main Article Details and this is an IMPORTANT WORD

Example Script:

< p > This is main Article Details and this is   < mark > IMPORTANT WORD       </ mark > </ p >

Wbr Tag [ < wbr > < /wbr > ]

Main Concept:

=> [Wbr] Tag is an advanced Word Breaking tag.

=> [Wbr] Tag is similer to [word-break] in CSS3, but [wbr] is much more easy and can also be formated by CSS.

Main & Common Usages:

=> [Wbr] Tag can be used to cut or split words, Sentences, Links,Entries upon certain rules in [CIO] Techniecs and [Word Press].

Example / Spliting Long Link:

https://www.testinglonglink.com
/products.php?id=20
&cat=100
&test=mainsection

Example Script:

< article >

https://www.testinglonglink.com

< wbr > /products.php?id=20 < /wbr >

< wbr > &cat=100 < /wbr >

< wbr > &test=mainsection < /wbr >

< /article >
