Time Tag [ < time > < /time > ]

[Time Tag] Concept:

=> [Time Tag] is to display formated and supported Time, Time & Date in Web Page .

=> [Time Tag] can be written as simple text, but [Time Tag] allowing many progrming features and supports.

Main & Features of [Time] Tag Usages:

=> Allowing Browsers to Time Zones Converting by user location.

=> It is a base in programable features and events in JavaScripts, J-Querry.

=> It is a base in Searching Engines advanced or Smart Reselts.

=> [Time Tag] can be used as full Time and Date format (year-month-dayTHour:Min:Sec) or partially (year-month), (Hour:Min).

Example:

This Article about using Time Tag in Full (datetime) Format on June, 6th, 2020 10:30:30

<article>
   <p> This Article about using Time Tag in Full (datetime) Format on
     <time datetime="2020-6-7T10:30:30"> June, 6th, 2020 10:30:30</time>    </p>
</article>
Dialog Tag [ < dialog > < /dialog > ]

[Dialog Tag] Concept:

=> [Dialog Tag] has an internal attribute (open) to control its standby Status.

=> [Dialog Tag] is usaully used as pop up message and can be programed by JS or J-Querry .

=> [Dialog Tag] it is block tag by defualt, and it's position property can be set by Css .

Main & Features of [Dialog] Tag Usages:

=> [Dialog Tag] can be used as hidden message , or container to be shown when clicking on certain button (programed by JS or J-Querry) which is simpler alternative for Popups.

Example:

Social Websites
  • Google
  • Youtube
  • Facebook
<dialog class="Note" open>
  Click on Links bellow to hypernate to other sites...
  <details>
    <summary> Social Websites </summary>
    <ul>
      <li> <a href="https://google.com" >Google</a></li>
      <li> <a href="https://youtube.com" >Youtube</a></li>
      <li> <a href="https://facebook.com" >Facebook</a></li>
    </ul>
  </details>
</dialog>
Bdi Tag [ < bdi > < /bdi > ]

[Bdi Tag] Concept:

=> [Bdi Tag] is an advanced easy text direction Converter [RtL <=> LtR] and vise versa.

=> [Bdi Tag] is an Inline Tag and its a better alternative for the Spin Tag [with its text direction changing & Display changing to Block in Css].

Main & Features of [Dialog] Tag Usages:

=> [Bdi Tag] can be used as the best solution and fastest method for local text directional changing for specific text.

=> [Bdi Tag] NOT supported by [Internet Explorer , Opera , Safari] browsers until 2014.

Example:

Top Members:

  • Shadi : 80 points
  •   شادي : 100 points
  •    سيد   : 150 points
<section>
    <h3> Top Members: </h3>
    <ul>
        <li> <bdi> Shadi </bdi>: 80 points </li>
        <li> <bdi> شادي </bdi>: 100 points </li>
        <li> <bdi> سيد </bdi>: 150 points </li>
    </ul>
</section>
