Output Tag [ < output > < /output > ]

[Output Tag] Concept:

=> [Output Tag] is a container to hold the result of a arithmetic calculation with in a Form.

=> [Output Tag] usually dealing with Input Tags's values as thier arithmetic calculation input, to hold the result with in it.

=> [Output Tag] absolutely being get their arithmetic operation results via Javascript codes.

Main & Features of [Output] Tag Usages:

=> It is critical to display the out put result of Arthmetic operation with in form.

=> It is very handy by using Javascript codes to manage it.

=> It is also can be formated by CSS codes.

Example:

Calculating User's Age by Days and Months put the Result in output field

Enter your Age by years to get by days & months:

   
<aside>
  <section>
     <h3> Enter your Age by years to get by days & monthss: </h3>
     <form   oninput = "resultD.value = parseInt(age.value) * parseInt(days.value) + ' Days'; resultM.value = parseInt(age.value) * parseInt(months.value) + ' Months'">
         <input    type = "text"     name = "age">
         <input    type = "hidden"   name = "days" value = "365" >
         <input    type = "hidden"   name = "months" value = "12" >
         <output   name = "resultD" >
         <output   name = "resultM" >
     </form>
  </section>
</aside>
