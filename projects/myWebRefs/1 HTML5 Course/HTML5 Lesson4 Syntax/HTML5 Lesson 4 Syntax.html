HTML5 [ New Syntax ] Main Concept

> [Syntax] in any programing language means the Language Coding Structure.

> HTML5 has new several Tricks in its [syntax] which is helping developers in coding and make it much easeir.

New HTML5 Syntax:[ Brackets ]

New Syntax:[ Brackets ] Concept:

HTML5 provides the ability to write codes by Single Codes Brackets , Double codes Brackets, or even WITH OUT Code Brackets totally ! .

Note: If you are going to use more than one Class Value to the same Tag, It's better to use Codes Brackets here .

New Syntax:[ Brackets ] Syntax: Example:

HTML Coding Page Editor:
=>     < Img   class= "Image1   Image2   Image3"   src= "image.jpg"   alt='Image'   title= My Image />
New HTML5 Syntax:[ BreakingSpace ]

New Syntax:[ Breaking Space ] Concept:

HTML5 provides the ability to Seperate between Syntax Components by Spaces to be more human readable and easy to modify.

New Syntax:[ BreakingSpace ] Syntax: Example:

HTML Coding Page Editor:
=>     < Img   class   =   "Image1   Image2   Image3"     src   =   "image.jpg"     alt   =   'Image'    title   =   My Image  />
New HTML5 Syntax:[ Special Javascript Classes ]

New Syntax:[ Special Javascript Classes ] Concept:

HTML5 provides the ability to write Javascript special classes with in HTML5 Tag Codes as an element Extra attributes.

New Syntax:[ Special Javascript Classes ] Example:

HTML Coding Page Editor:
< Img   class= "Image1   Image2   Image3"   src= "image.jpg"   alt='Image'   title= My Image   data-wow-duration= "1 S"   data-wow-delay= "2 S"   />

New Syntax:[ Breaking Line ] Concept:

HTML5 provides the ability to seperate HTML5 Tag Code between it's attributes to multiple Lines to be more human readable and easier to modify.
