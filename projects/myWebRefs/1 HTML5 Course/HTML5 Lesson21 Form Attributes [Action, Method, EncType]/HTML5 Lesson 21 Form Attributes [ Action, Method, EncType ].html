Form Attribute [Action] Concept:

=> [Action Attribute] is one of the old Form Attributes in HTML4 and HTML5.

=> [Action Attribute] is to specify the location or file in server where the Form inputs fields data will go to, and once it has been sent to the server the Back End langauage will start process it and then send it to the data base by database language like Mysql.

=> [Action Attribute] if it was empty or unspecified => the action of the form will be the current page by default.

Input override Attribute [FormAction = ""] Concept:

=> [FormAction Attribute] is one of the new Form Input override Attributes in HTML5.

=> [FormAction Attribute] is to specify another location or file in server to specific Input (type="submit" or type="image" only) where the Form inputs fields data will go to.

=> [FormAction Attribute] is Valid only for Form Input with Type (Submit) or (Image) only, and can NOT work with Input with type (Button).

Main & Features of Input Override Attribute [FormAction] Usages:

=> It mainly designed to specify a special Action value (location File) to another Submit Input in the form, to allow sending the same the whole form inputs value to more than one Actions.

=> This over Attribute can only work with each (Submit & Image) Input Types, And also can work with input type (button) but with previous programing by javascript.

Form New Attribute [EncType] Concept:

=> [EncType Attribute] is one of the new Form Attribute in HTML5.

=> [EncType Attribute] is property meant by Encoding form's data which is being sent to the server (Action), and this Encoding has (3) Types:

  1. Default Type [EncType = "application/x-www-form-urlencoded"]:

    2. This is the default Value of EncType Form Attribute

    This Type is Encoding the whole Form Data by Hexa Coding for Special Characters & + for Spaces by a Encoding System Called [ASCIIHEX] Value.

  2. Without Endcoding Type [EncType = "multipart/formdata"]

    3. This Type is not Encoding any part of Form data

    This Type is usually used in case of uploading Files to the servers as it is {input type = "file"}

  3. Text-plain

    4. This Type is Encoding the spaces only.

    This Type is usually used in [Debugging processes].

Main & Features of Form [EncType] Attribute Usages:

=> It mainly designed to secure the form data sent to the server & selecting the suitable method to it.

=> It can be programatically used in Events to as an actions or reactions to make change by page or any of its element/s using Javascript codes in Front End & PHP Codes in Back End.

=> It can be applied as a Form's Attribute for the whole Form's Input Feilds, OR for special Submit Input Field.
