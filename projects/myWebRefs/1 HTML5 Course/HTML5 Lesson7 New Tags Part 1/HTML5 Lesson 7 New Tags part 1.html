[HTML5 New Tags] Concept

> In this is part one of HTML5 Tags we will display the main HTML5 Page structure by the HTML5 New Tags.

> The New HTML5 Tags in this lesson as following :

  1. < header >     < /header >
  2. < nav >         < /nav >
  3. < section >   < /section >
  4. < article >     < /article >
  5. < footer >     < /footer >
Header Tag [ < header > < /header > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as self Block-Displayed can be a better Alternative of Div Element with Class of heading.

Main & Common Usages:

Header Tag can be used in Main Page Title or in subtitle as well.

Header Tag can be displayed as Heading Sector that contain Smaller Headings < h > and other main Paragraph < p >

Example Script:

< header >

< h1 > This is Head Line   < /h1 >

< p > Created By   < span > Shindow < /span >   < /p >

< /header >

Navigation Tag [ < Nav > < /Nav > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as self Block-Displayed can be a better Alternative of Div Element with Class of internal or external links related to       page.

Main & Common Usages:

Nav Tag can be used as [Internal Index Sector] which contain Internal links directed to Page sector to navigate through.

Nav Tag can be used as [External Index Sector] which contain External links directed to other webPages related to the page to hypernate to.

Nav Tag is rarely to be use in any other function except [Internal Index Sector] & [External Index Sector].

Example Script:

< nav >

< ul >

< li > Home         < /li >

< li > About         < /li >

< li > Services     < /li >

< li > Contact Us < /li >

< /ul >

< /nav >

Section Tag [ < Section > < /Section > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as self Block-Displayed can be a better Alternative of Div Element with Class of Section or Part of page.

Example Script:

< section >

< header >

This is Section Title

< /header >

< p > Created By   < span > Shindow < /span >   < /p >

< /section >

Article Tag [ < article > < /article > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as self Block-Displayed can be a better Alternative of Div Element with Class of Article Sections in page.

Article Tag can contains Smaller article tags such as: < p > , < quote > , < q > which have a Paragraphical Function.

Example Script:

< article >

< header >

This is Article Title

< /header >

< p > This is Article contents ...   < /p >

< /article >

Footer Tag [ < footer > < /footer > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as self Block-Displayed can be a better Alternative of Div Element with Class of Footer Section in page.

Footer Tag can contains Smaller article tags such as: < p > , < span > , < a > which have a Paragraphical Function and Links related to page.

Example Script:

< footer >

This is Page Footer contents...

< p > copyright 2020 © all rights reserved by Shindow ™   < /p >

< /footer >
