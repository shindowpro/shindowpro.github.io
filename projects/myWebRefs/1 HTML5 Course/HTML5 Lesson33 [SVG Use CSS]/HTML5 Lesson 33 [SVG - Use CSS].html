  1. HTML5 Tag [SVG -use CSS Codes] Concept :

    2. [SVG- Use CSS codes] <= you can use CSS codes to formating SVG Elements instead of Element Attributes

    [SVG- Use CSS codes] <= There are Basic Attributes of SVG Elements that must be written in HTML page, as following :

    1. [rect]
      1. x="#Numric Value" , y="#Numric Value" => Which are the Starting point of Rectangle Element inside the SVG
      2. width="#Numric Value" , height="#Numric Value" => Which are the main Dimensions of the Rectangle
      3. rx="#Numric Value" , ry="#Numric Value" => Which are the degree of Curvy of the Rectangle Borders
    2. [circle]
      1. cx="#Numric Value" , cy="#Numric Value" => Which are the Starting center point of Circle Element inside the SVG
      2. r ="#Numric Value" => Which are the main Circle Diameter

    [SVG- Use CSS codes] <= You Can use CSS Codes with all its methods as following :

    1. External CSS file by class or name or Element name itself
    2. Internal CSS (style="CSS Codes") Attribtue with in Element Attributes

    [SVG- Elements Addtional Attributes] <= There are more SVG Element Attriubtes as follwoing :

    • [opacity = "#Numeric Value"] => Its is the degree of Contract of elements on the SVG {This Attirbute is existed in CSS properties of Element Name}
    • [fill = "#Color"] => Its is the fill Style Filling (inside Filling Color) of elements on the SVG {This Attirbute is existed in CSS properties of Element Name}
    • [stroke = "#Color"] => Its is the Stroke Style Filling (Border Color) of elements on the SVG {This Attirbute is existed in CSS properties of Element Name}
    • [stroke-width = "#Numeric Value"] => Its is the Stroke Style Filling (Border Color) of elements on the SVG
