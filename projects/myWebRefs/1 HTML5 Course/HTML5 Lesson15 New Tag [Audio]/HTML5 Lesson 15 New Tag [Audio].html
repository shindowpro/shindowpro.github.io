Audio Tag [ < audio > < /audio > ]

[Audio Tag] Concept:

=> [Audio Tag] is tag specified to deal with audio files on web Pages with advanced controls and tools in comparison with Html4 which werer dealing with audio files using flash with difficult controls and hard formating mechanism.

=> [Audio Tag] has several properties inside:

  1. [Src = ""]: Here where we write the Audio File Path, and this property can be written in the main < Audio > Tag or it's partical tag < Source >
  2. [Controls]: Which is allowing tools bar to the Audio file coded.
  3. [Autoplay]: It provides the (Auto play) feature to the allocated audio file once the website is loaded.
  4. [Loop]: It provides the Looping feature to the allocated audio file once it finished Automaticlay.
  5. [Mute]: It provides the ability to mute the audio file inspite of being played, and this can be a programable feature in Javascipt.
  6. [Preload = ""]: It provides the ability to control the download method of the audio file, and it offer (3) options :
    1. [Preload = "Auto"] : It is meaning to auto load the allocated audio file once the website is loaded.
    2. [Preload = "metadata"] : It is meaning to load the information of the allocated audio file once the website is loaded.
    3. [Preload = "none"] : It is meaning Not to load the allocated audio file while website is loaded unless the user click on play button form the control audio file.

=> [Source Tag] (< source >) is the only Partial Tag of [Audio Tag], and it has 2 properties:

  1. [src=""]: Which can be written as mentioned in Audio Tag it self also, but it is better to write it in Source Tag , to have a chance to add many format type of audio file to gurantee that the browser will support one of file formats.
  2. [type = ""]: It specify the [ MIME ] type of the allocated Audio file, and here some of mime type to common audio file extensions:
    1. (mp3) extension => (audio/mpeg)
    2. (wav) extension => (audio/wav)
    3. (ogg) extension => (audio/ogg)
  3. [Media]: It is one of Unsupported property of [Source Tag] until (2014) to any Browser.

Main & Features of [Audio] Tag Usages:

=> It has solved many problems of older way in HTML4 with in controling audio file, by the [Audio] tag & it's attributes and properties

=> It is very handy by using Javascript codes to manage it.

=> It is also can be formated by CSS codes.
