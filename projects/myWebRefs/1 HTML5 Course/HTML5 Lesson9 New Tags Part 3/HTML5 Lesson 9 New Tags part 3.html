  1. < progress >     < /progress >

Progress Tag [ < progress > < /progress > ]

[Progress Tag] Concept:

=> [Progress Tag] commonly wide used in sites as an Indicator of several varity process and operation.

=> [Progress Tag] has 2 common Attributes:

  1. Value => showing the current value of progress
  2. Max => showing the maximum value of progress bar

Main & Common Usages:

=> It is a usful tool in showing currently static or dynamic progress as form of Bar of any process.

=> Commonly used in showing progress bar of Upload / Download status .

Example Script:

< section >

          Loading Progress:

< progress value= "7" max= "10" > < /progress >

< /section >

Meter Tag [ < meter > < /meter > ]

Main Concept:

=> This is an advanced guage Tool with more complex Attributes and properties to provide further control to simple / integrated Process or status progress.

=> [Meter Tag] has 6 common Attributes :

  1. Value => showing the current value of progress
  2. Min => showing the minmum value of meter bar (default:"0").
  3. Max => showing the maximum value of meter bar (default:"1").
  4. Low => showing the minimum Accepted value to the meter bar .
  5. High => showing the maximum Accepted value to the meter bar .
  6. Optimum => showing the Optimum value the meter bar.

=> [Meter] Bar has 4 Color Cases:

  1. [Green] : if (Value = Optimum) or (Value < High & > Low) or (Value = High)
    •   [Value   =   Optimum]
    •   [Value   <   High   &   >   Low]
    •   [Value   =   High]
  2. [Yellow] : if (Value < Low) or (Value > High)
    •   [Value   <   Low]
    •   [Value   >   High]
  3. [Red] : if (Value < Optimum & Value < low) or (Value > Optimum & Value > Max)
    •   [Value < Low < High < Optimum]
    •   [Value > High > Low > Optimum]
  4. [White (Empty)] : if (Value = 0) or (Value < Min)
    •   [Value = 0]
    •   [Value < Min]

Main & Common Usages:

=> [Meter] Tag can be used as an advanced meter bar for Website Server Performance.

=> [Meter] Tag can be used as an advanced meter bar for Application CPU Usage Performance.

=> [Meter] Tag can be used as an advanced meter bar for Total Distribution of Common usage of Website Categories.

=> [Meter] Tag can be used as an advanced meter bar for Total Distribution of Overall Statics of a company Products/Services Sells .

Example Script:

< aside >

< section   class= "Meters" >

< h3 > Performance < /h3 >

< div >

CPU:  

< meter

min = "0"

max = "1"

value = ".6"

low = ".4"

high = ".8"

optimum = ".7" > < /meter >

< /div >

< /section >

< /aside >
