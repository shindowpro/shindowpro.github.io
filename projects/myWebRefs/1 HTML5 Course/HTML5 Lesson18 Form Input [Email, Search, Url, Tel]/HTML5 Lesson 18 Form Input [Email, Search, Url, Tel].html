Form Input New field [ < input type = "email" > ]

Form Input New field [Email] Concept:

=> [Email Field] is one of the new fields in HTML5 and Not supported in by Safari browser until 2014.

=> [Email Field] in unsupported browsers need to programming it with Javascript codes in Front End, and PHP or similer in Back End.

=> [Email Field] has (1) main Attributes :

  • Name => which is the programable name for the input element within the form

=> [Email Field] is mainly made to input an Email address with in the form to the server by user, and it usually followed by [Submit] button.

=> [Email Field] is pre-programmed to accept only Email Address with in the Email structures Rules.

Main & Features of Form Input [Email Field] Usages:

=> It mainly designed to input Validated Email Address with out perprogrammin efforts.

=> It can be programatically used in Events to as an actions or reactions to change the color of page or element/s using Javascript codes.

Example:

Entering user's Email Address in an Email input field type.

Enter your Email Address:

<section>
   <h3> Enter your Email Address: </h3>
   <form>
     <input   type = "email"  name = "email"  >
     <input   type = "sumbit"  name = "submit-color"  value = "send" >
   </form>
</section>
Form Input New field [ < input type = "search" > ]

Form Input New field [Search] Concept:

=> [Search Field] is one of the new fields in HTML5 and Not supported in by each Safari, Internet Explorer, Firefox browsers until 2014.

=> [Search Field] in unsupported browsers need to programming it with Javascript codes in Front End, and PHP or similer in Back End.

=> [Search Field] has (1) main Attributes :

  • Name => which is the programable name for the input element within the form

=> [Search Field] is mainly made to Textual, Numeric, Symbols or any other value to search about, also it has and auto delete button with it to auto delete the inserted input.

Main & Features of Form Input [Range Field] Usages:

=> It can be be programatically used in Events to as an actions or reactions to change the Value of certain input or element/s of page using Javascript codes.

=> It provide an easy searching tool with in the website about any kind of value.

Example:

Searching about certain value

Enter your value to search about it in search field:

<section>
   <h3> Enter your value to search about it in search field: </h3>
   <form>
     <input   type = "search"  name = "search" >
     <input   type = "sumbit"  name = "submit-Value"  value = "send" >
   </form>

Form Input New field [ < input type = "url" > ]

Form Input New field [Url] Concept:

=> [Url Field] is one of the new fields in HTML5 and Not supported by Safari browser until 2014.

=> [Url Field] in unsupported browsers need to programming it with Javascript codes in Front End, and PHP or similer in Back End.

=> [Url Field] has (1) main Attributes :

Form Input New field [ < input type = "tel" > ]

Form Input New field [Tel] Concept:

=> [Tel Field] is one of the new fields in HTML5 which allowing to input a numeric Telephone value only.

=> [Tel Field] is not supported in any browser except Safari Browser.

=> [Tel Field] has (1) main Attributes :

  • Name => which is the programable name for the input element of the form

=> [Tel Field] is rarely to be used in browsers due to few borwers supporting it.

Main & Features of Form Input [Tel Field] Usages:

=> It can be be programatically used in Events to as an actions or reactions to change the color of page or element/s using Javascript codes.

=> It mainly an east way to input a numeric Telephone value only and send it with in form to server.

Example:

Inserting a phone number in tel input field

Enter your Telephone number:
