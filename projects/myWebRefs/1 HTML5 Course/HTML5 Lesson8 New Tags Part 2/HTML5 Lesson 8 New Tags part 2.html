[HTML5 New Tags] Concept

> In this is part one of HTML5 Tags we will display the main HTML5 Page structure by the HTML5 New Tags .

> The New HTML5 Tags in this lesson as following :

  1. < main >     < /main >
  2. < aside >         < /aside >
Main Tag [ < main > < /main > ]

Main Concept:

[Main] New tag Must contains a unique information or items.

[Main] Tag must be as a Partrent tag to contain another types of tags, but Can Not be contained by any other kind of tags.

[Main] Tag Can Not be Repeated, and can only written once.

[Main] can be written in any location in the page but with previos Conditions.

Main & Common Usages:

Show the main purpose of site or its company's Product or Service.

Welcome message for the Site visitor.

Showing the Firm's Logo , Moto or Slogan.

The most important New or Show or Programe [Hot Item].

Example Script:

< main >

< p > Welcome to   < span > Html5 Course < /span >   < span > Lesson 8 < /span > < /p >

< /main >

Aside Tag [ < aside > < /aside > ]

Main Concept:

This is one of HTML5 new Semantic Tag that could be used as Private Sider Bar which can be as special Widget for important or handy items.

Main & Common Usages:

[Aside] Tag can be used as Special Seperated sector holding handy stuff with same category.

Each [Aside] Tag can hold one same Categorized items or more than one Classification.

[Aside] Tag can be formated by CSS upon the total size of all page to be allocated in certain percentage of page within [Width] Property.

[Aside] Tag better to be formated by CSS as floated Section to be free to display seperated from the main page.

Example Script:

< aside >

< section   class= "links" >

< h3 > Categories < /h3 >

< ul >

< li > Home         < /li >

< li > About         < /li >

< li > Services     < /li >

< li > Contact Us < /li >

< /ul >

< /section >

< /aside >
