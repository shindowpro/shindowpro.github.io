HTML5 New Form Elements [Datalist] Concept

> The New HTML5 Form Elements in this lesson as following :

  1. < input   list = "List name"   >
  2. < datalist   id = "list name" >     < /datalist >
  3. < option   value="" >  
Datalist Tag [ < datalist > < /datalist > ]

[Datalist Tag] Concept:

=> [Datalist Tag] is to display a list of data below with in upper input list type.

=> [Datalist Tag] must be identified by id by value of it's upper input list name in the form.

=> [Datalist Tag] allow to activate (Auto Suggest) feature used in Ajax with simple way.

=> [Datalist Tag] includes data with in [option tag] one for each value below datalist tag.

=> [Datalist Tag] usually be followed by submit button to record the choosed value from Datalist.

=> [Datalist Tag] is like the database for the [Input list], so the [Datalist]'s id must be with the same value of [Input list]. Also the [Option] is the data of this database (datalist).

Main & Features of [Datalist] Tag Usages:

=> Provides an advanced way to display and control data with in a list more proffeciently.

=> Allowing [Auto Suggest] Feature with choosing from available data below.

=> The [Auto Suggest] of Datalist more easy and simple from Ajax.
