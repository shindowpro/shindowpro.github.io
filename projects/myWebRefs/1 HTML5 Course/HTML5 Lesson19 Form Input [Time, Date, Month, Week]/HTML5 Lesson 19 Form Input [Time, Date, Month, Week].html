Form Input New field [ < input type = "time" > ]

Form Input New field [Time] Concept:

=> [Time Field] is one of the new fields in HTML5 and Not supported in by both Internet Explorer & FireFox browser until 2014.

=> [Time Field] in unsupported browsers need to programming it with Javascript codes in Front End, and PHP or similer in Back End.

=> [Time Field] has (1) main Attributes :

  • Name => which is the programable name for the input element within the form

=> [Time Field] is mainly made to input a Time details (Hours + Mins + Am/Pm) with in the Form to the server by user, and it usually followed by [Submit] button.

Main & Features of Form Input [Time Field] Usages:

=> It mainly designed to input Validated Time Format with out pre-programing efforts.

=> It can be programatically used in Events to as an actions or reactions to make change by page or any of its element/s using Javascript codes in front end.
