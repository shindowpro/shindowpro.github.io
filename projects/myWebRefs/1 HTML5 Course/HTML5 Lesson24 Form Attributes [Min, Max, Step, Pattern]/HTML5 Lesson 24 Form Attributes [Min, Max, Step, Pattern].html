Form => Input Attribute [ < input   min=""   > ]

Input Attribute [Min] Concept:

=> Input [Min Attribute] is one of the old HTML5 Form's Numeric Input Attributes.

=> Input [Min Attribute] is to minimum value limit of numeric value inserted into the numeric input Feild .

=> This Attribute is working only with numeric Input Feild such as :

  • [Number] Input Feild
  • [Range] Input Feild
  • [Date] Input Feild
  • [DateTime] Input Feild
  • [Time] Input Feild

=> Input [Min Attribute] can be allocated for more than one Input Tag outside the form with the page.

Main & Features of Input Attribute [Min Attribute] Usages:

=> It mainly designed to set a minimum Value allowed of inserted numeric by user.

=> This Attribute can be controled programticaly using Javascript

Form => Input Attribute [ < input   pattern=""   > ]

Input Attribute [Pattern] Concept:

=> Input [Pattern Attribute] is one of the old HTML5 Form's Input Attributes.

=> Input [Pattern Attribute] is to set creteria(s) of specific Textual Input Feild using [Regular Expression] Rules.

=> Input [Pattern Attribute] applicable to any kind of input feild (Except Submit & Button).

=> Input [Pattern Attribute] can be allocated for more than one Textual Input Tag outside the form with the page.

=> Input [Pattern Attribute] can NOT work with any type of Numeric Input Feilds.

=> Input [Pattern Attribute] can be works only with Textual Input Fields Types such as:

  • [Text] input feild
  • [Search] input feild
  • [Tel] input feild
  • [Url] input feild
  • [Email] input feild

=> Input [Pattern Attribute] must has the {Required} Attribute to guarantee that the customized Pattern will be Obligatory implemented within the Form.

Main & Features of Input Attribute [Pattern Attribute] Usages:

=> It mainly designed to set specific Input Feild(s) Customized Form Obligatory and preveting the form to Submit other Input feild(s) with out it / them.
