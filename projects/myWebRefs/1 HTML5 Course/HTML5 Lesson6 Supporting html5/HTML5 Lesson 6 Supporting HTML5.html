HTML5 [ Supporting HTML5 ] Main Concept

> The old browsers versions not supporting the new version of HTML5, so we need to add some identificating code to be supported by.

> The process of this identification can be done by importing javascript script file in < script > Tag, and this file can be downloaded from internet [html5shiv.min.js]

> This file [html5shiv.min.js] enables use of HTML5 Sectioning elements in legacy Internet Explorer and provide basic HTML5 Styling for following borwsers:

  1. Internet Explorer 6 / 9
  2. Safari 4
  3. FireFox 4
  4. I-phone 3
