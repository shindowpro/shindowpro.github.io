Figure Tag [ < figure > < /figure > ]

[Figure Tag] Concept:

=> [Figure Tag] commonly wide used as an explination container, which can hold different types of contents (Code/s, Image/s, Diagram/s, Flow Chart/s, Statistic Chart/s, Schema/s).

=> [Figure Tag] has an internal Tag [Figcaption] which is the heading or the legend of this Figure.

=> [Figcaption Tag] can be upper or lower the Figure.

Main & Common Usages:

=> It can be an additional Tag, Not an Structural Tag.

=> Commonly used in Scientific Articles, Products Title display as an addition illustrative element to the main Subject article.
