7 Learn JS BOM - Window Methods - scrollBy, scrollTo Concept:

=> [ window Methods ] :

  1. Main concept of {window Methods} =>
    1. {scrollBy()} Method =>

      To Set a certain amount of pixels and position [x & y] to move the current window towoard

      This method does Not depends on current window to be process

      This method keep moving the Element each time the fuhnction is being processed , until reaching the Edge of the Curren Page

      Using the egative value with the heigth and width => will give a result only witjh the following Conditions:

      • The current page position is not at the Edge of the Page
      • The current page Width and Heigth has a free to move and navigate through

    2. {scrollTo()} Method =>

      To Set a certain amount of pixels and position [x & y] to move the current window towoard

      This method depends on current window to be process -> and will be implemented only the element is NOT in the assigned location

      This method move the Element to the assigned location coordination , and stop working if the elmement has reached it's destination

      Using the Negative value with the heigth and width => will give a result only witjh the following Conditions:

      • The current page position is not at the Edge of the Page
      • The current page Width and Heigth has a free to move and navigate through
