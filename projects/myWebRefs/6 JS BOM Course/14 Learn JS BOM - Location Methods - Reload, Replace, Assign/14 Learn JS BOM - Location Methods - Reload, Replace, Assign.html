14 Learn JS BOM - Location Methods - Reload, Replace, Assign Main Topics :

> Main Subjects in this lesson are as following :

  1. location.assign(); => To Assign new value of location's Href of the current Webpage [with keeping the History]
  2. location.replace(); => To Assign new value of location's Href of the current Webpage [with Replacing the History]
  3. location.reload(); => To Reload the Current Webpage [from Browser Cache (false: default Value) || from Server Cache (true: Strong Reload)]

14 Learn JS BOM - Location Methods - Reload, Replace, Assign Concept:

=> [ Location - Reload || Location - Assign() || location - Replace() ] :

  1. Main concept of {location.reload()} || {location.assign()} || {location.replace()} =>
    1. {location.reload()} Method =>

      To Relaod the Current webpage by using Browser Cache OR the Server Cache

      Reload => has (2 Cases) , as following :

      • reload(false) || reload() => The Normal Reloading [Using the browser Cache to reload the web page]
      • reload(true) => The Strong/Full Reloading [Using the Server Cache to reload the web page] => {More safer for Accurate & Most Recent Server Data}

    2. {location.assign()} Method =>

      To Assign new value of the Location's Href of Current webpage

      This Function will keep the Current Webpage in the History before Changing it

    3. {location.replace()} Method =>

      To Assign new value of the Location's Href of Current webpage

      This Function will Replace the Current Webpage with the new Webpage in the History


14 Learn JS BOM - Location Methods - Reload, Replace, Assign importance & Benefits :

14 Learn JS BOM - Location Methods - Reload, Replace, Assign gives us the Following Features :

  • {location.reload} => A JS useful Advanced Tool to Control the method of Reloading mode [from Browser Cache || from Server Cache]
  • {location.assign} & {location.replace} => A JS useful Advanced Tool to Control change the current locatoin Href value, [with keeping Current Webpage in History || Replace it with the new one]
