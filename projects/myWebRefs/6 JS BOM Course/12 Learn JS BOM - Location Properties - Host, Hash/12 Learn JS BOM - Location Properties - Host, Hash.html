12 Learn JS BOM - Location Properties - Host, Hash Main Topics :

> Main Subjects in this lesson are as following :

  1. window.location.host; => To Get the Host Name of the page/window
  2. window.location.host = ''; => To Set/Reset/Generate the Host Name of the page/window
  3. window.location.hash; => To Get the Hash part of Href (if it exists) of the page/window
  4. window.location.hash = ''; => To Set/Reset/Generate the Hash part of Href of the page/window

12 Learn JS BOM - Location Properties - Host, Hash Concept:

=> [ window - Location - Host || window - Location - Hash ] :

  1. Main concept of {window.location.host} || {window.location.hash} =>
    1. {window.location.host} property =>

      One of the Basic properties of the Location Object

      One of JS Tool to control the Set/get the Value of the Host Name of the current WebPage

    2. {window.location.hash} property =>

      One of the Basic properties of the Location Object

      One of JS Tool to control the Set/get the Value of the Hash Extension Practical Href of the current WebPage

      Note => When setting a new valu of the Hash by the JS prompt => JS will add a # sign by default


12 Learn JS BOM - Location Properties - Host, Hash importance & Benefits :

12 Learn JS BOM - Location Properties - Host, Hash gives us the Following Features :

  • {location.host} => A JS useful Advanced Tool to Control [Set/Reset/Generate] the Host Name of Webpage as A Value of a Property , and can be used in Customized Redirecions application
  • {location.hash} => A JS useful Advanced Tool to Control [Set/Reset/Generate] the Hash Href Extension of Webpage as A Value of a Property , and can be used in Customized programming proceses application

12 Learn JS BOM - Location Properties - Host, Hash Syntax & Methods:

[A]// {JSBOM file} :

[1] {BOM} ( {window.location.host} ) Examples:
a) Getting the value of [location.host] of the current page:

console.log(window.location.host);
console.log(location.host);
alert(window.location.host);
alert(location.host);
//===========================================================

b) Setting the value of [location.host] of the current page:

window.location.host = 'www.HostName.com';
location.host = 'www.HostName.com';
//===========================================================

console.log(window.location.hash);
console.log(location.hash);
alert(window.location.hash);
alert(location.hash);
//===========================================================

b) Setting the value of [location.hash] of the current page:

window.location.hash = "#ElementID";
location.hash = "#ElementID";
