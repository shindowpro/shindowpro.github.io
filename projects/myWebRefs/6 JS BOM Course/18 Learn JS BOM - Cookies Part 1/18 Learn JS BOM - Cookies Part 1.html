=> [ document - cookie] :

  1. One of the Document object element , which designed for store of save the user settings in the webpage in this website
  2. Cookie saving time of user , and auto reset the user's settings each time the user visti the webpage/ webdsite
  3. Cookie can be set and contorlerd by several p.langage [JS . PHP , Java]
  4. Cookies depeneds on Data mainly to detect the duration of the cookie to be available
  5. Javascript can dealing with the Cooikie by the Followinhg methods :

    2. Editing or Modifying an exited file of the cookie or property
    3. Deleting or Remove an exited file of the cookie or property
