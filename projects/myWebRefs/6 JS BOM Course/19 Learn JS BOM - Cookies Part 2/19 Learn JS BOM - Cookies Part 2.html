19 Learn JS BOM - document - Cookie Part2 , Concept:

=> [ document - cookie - Methods] :

  1. Cookie Object Functionality Can include the followikng:
  2. Cookies main Methods as following :
    • Get/Display the Cookie value => JS + Console :

      a) Getting Cookie by the Console :

      1. Get all cookie properties :

        document.cookie;

      b) Getting Cookie by the JS :

      1. Get all cookie properties :

        document.cookie.all();

      2. Get certain cookie property :

        document.cookie('key1');

      3. Get certain cookie with Exipry Date property :

        document.cookie('key1', {Exipry});

      4. Get X cookie properties :

        document.cookie('key1' , 'key2');

        document.cookie(['key1' , 'key2']);

      5. Get certain cookie with Exipry Date properties :

        document.cookie(['key1', 'key2'], {Exipry});


    • Set/Add the Cookie value => JS + Console

      Setting new Cookie by the Console :

      1. Set new one cookie with value only :

        cookie.set('key1','value1');

    • Setting new Cookie by the JS:

      1. Set new X cookie with prop name and prop value :

        cookie.set( {'key1' : 'value1', 'key2':'value2'} );

      2. Set new X cookie with prop name and prop value :

        cookie.set( {'key1' : 'value1', 'key2':'value2'} , {expiry:7} );


    • Modify the Cookie value => JS + Console

      Modifying new Cookie by the Console :

      1. Edit one cookie property with one specification :

        document.cookie = 'propName=new value';

      Modifying new Cookie by the JS :

      1. Set new X cookie with prop name and prop value :

        cookie.set( {'key1' : 'new value1', 'key2':'new value2'} );

      2. Set new X cookie with prop name and prop value :

        cookie.set( {'key1' : 'new value1', 'key2':'new value2'} , {expiry:7} );


    • Remove/Delete the Cookie value => JS + Console

      Removing Cookie by the Console :

      1. Remove cookie properties specifications :

        document.cookie = 'propName=value; Exipry='String Date value; path=/path';

      Removing Cookie by the JS :

      1. Remove one cookie property with all specifications

        cookie.remove('key1');

      2. Remove X cookie properties with all specifications

        cookie.remove(['key1', 'key2']);

        cookie.remove('key1', 'key2');
