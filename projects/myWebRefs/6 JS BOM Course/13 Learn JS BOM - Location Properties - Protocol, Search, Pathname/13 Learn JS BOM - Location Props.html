> Main Subjects in this lesson are as following :

  1. location.protocol; => To Get/Set the Potocol Type the current Webpage
  2. location.pathname; => To Set/Get the website Domain name [website name after the host name and before the Search] page/window
  3. location.search; => To Get the Query String used for search with the of the page/window

13 Learn JS BOM - Location Properties - Protocol, Search, Pathname Concept:

=> [ Location - protocol || Location - search || location - pathname ] :

  1. Main concept of {location.protocol} || {location.search} || {location.pathname} =>
    1. {location.protocol} property =>

      One of the Location Object

      One of the href content, and it is the resposible for transfering the current WebPage file to the browser

      protocol => has several Types , as following :

      • http / https = > Hyper Text Transfer Protocol / Secured
      • File = > Protocol for transfering the webpage file to the current Browser
      • Mailto = > Protocol reponsilble for transfering the mail to the mail Application
      • FTP = > File Transfer protocol

    2. {location.search} property =>

      One of the Basic properties of the Location Object

      One of the href content and it is The [Querry String] of the Searching functionality by php to the Database

      Note => Will appear after the php, and starting with ? sign

      Note => Will appear only in case of using the Admin panel for searching for items in the Database

    3. {location.pathname} property =>

      One of the Basic properties of the Location Object

      One of the href content , and it is The domain name of the current website

      It is comming after Protocol part and host name part , and before the search part


13 Learn JS BOM - Location Properties - Protocol, Search, Pathname importance & Benefits :

13 Learn JS BOM - Location Properties - Protocol, Search, Pathname gives us the Following Features :

  • {location.protocol} => A JS useful Advanced Tool to Control [Set/Get] the Protocol of Webpage as A Value of a Property, and can be used for changing the current webpage protocol type
  • {location.search} => A JS useful Advanced Tool to Control [Set/Get] the cerrent Querry String of searching item in the current webpage admin searching panel, and can be used in advaned application
  • {location.pathname} => A JS useful Advanced Tool to Control [Set/Get] the damain of Webpage as A Value of a Property, and can be used JS Application

13 Learn JS BOM - Location Properties - Protocol, Search, Pathname Syntax & Methods:

=> {JS BOM - Window Location Property - Protocol, Search, Pathname => by following Conceptual Syntax as following :

[A]// {JSBOM file} :

[1] {BOM} ( {location.Protocol, Search, Pathname} ) Examples:
a) Getting the type of [location.protocol] of the current page:

console.log(location.protocol);
alert(location.protocol);
//===========================================================

b) Setting the value of [location.protocol] of the current page:

location.protocol = 'https';
location.protocol = 'http';
location.protocol = 'ftp';
location.protocol = 'mailto';
location.protocol = 'file';
//===========================================================

[2] {BOM} ( {location.search} ) Examples:
a) Getting the Querry String value of [location.search] of the current page:

console.log(location.search);
alert(location.search);
//===========================================================

b) Setting the Querry String value of [location.search] of the current page:

location.search = "?Edit&itemid=23";
//===========================================================

[3] {BOM} ( {location.pathname} ) Examples:
a) Getting the pathname value of [location.search] of the current page:

console.log(location.pathname);
alert(location.pathname);
//===========================================================

b) Setting the pathname value of [location.pathname] of the current page:

location.pathname = "google.com";
//===========================================================

//===========================================================
