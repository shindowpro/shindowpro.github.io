  1. Main concept of {window Methods} =>
    1. {setInterval()} Method =>

      To Set a Timer [by Milliseconds] to start Certain Code or function {with Repeatation}

      This function depend on Obligatories elements and other Optional elements:

      • Obligatories :
        1. [Function Name] => Which will be implemented Every The Timer set in the setInterval Method
        2. [Timer] => The Seconds by (Milliseconds) Which the Function will be processed each time the time is set
      • Optionals :
        1. [Parameters] => The main Parameters of the main setInterval

      This method is just simmiler to the setInterval() Method , with major Dieffernce which is {Repeatation}

      This method will be repeated unless we stop it by using { clearInterval() }

    2. {clearInterval()} Method =>

      To Stop processing and Repeating the Declared setInterval Function

[1] {BOM} (setInterval()) Examples:
a) Declaring a setInterval() function + [with inner Function decleration] :

setInterval(function(){
'use strict' ;
console.log('Write This Every # Seconds');
}, #Milliseconds ) ;
//===========================================================

b) Declaring a setInterval() function + [with Calling previous Declared inner Function ] :

setInterval(functionName, #Milliseconds ) ;
//===========================================================

c) Defining a variable of Declaring a setInterval() function + [with Calling previous Declared inner Function ] :

var myTime = setInterval(functionName, #Milliseconds ) ;
//===========================================================

[2] {BOM} (ClearInterval()) Examples:
a) Declaring a ClearInterval() Method [to stop Processing the Previous setInterval()]:

ClearInterval(myTime) ;
//===========================================================

b) Creaing a Button Stop Processing the Previous setInterval() [by onclick event]:

// Defining the Button Variable :
var btn = document.getElementByID('btn');

// Defining onclick event => to ClearInterval() :
btn.onclick = function {

};
