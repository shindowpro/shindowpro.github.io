11 Learn JS BOM - Window Location Properties - Href Concept:

=> [ window - Location - Href ] :

  1. Main concept of {window.location.href } =>
    1. Location {href} property =>

      One of the Basic properties of the Location Object

      One of JS Tool to control the Setting/geting the Value of the URL fo the current WebPage

      The Value of the Href can be set by different Types of Value :

      • Absolute Full URL => 'http://www.domain.com'
      • WebPage Name with Extension with Page Web File Path => 'fileName.extension'
      • Hash Internal link in the Same Page => '#idofElement'
      • Different Protocol Type => such as
        1. File Protocol => 'file:///FilePath.FileName.Extension'
        2. Mail Protocol => 'mailto:email@host.com'
        3. FTP Protocol => 'ftp://'
