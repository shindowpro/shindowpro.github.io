9 Learn JS BOM - Window Properties - innerHeight, innerWidth, outerHeight, outerWidth Concept:

=> [ window - Properties ] :

  1. Main concept of {window Properties} =>
    1. {innerWidth} property =>

      Consider the Net Width space of the current view of page [Not including space of the (Side Scroll bar) ]

    2. {innerHeight} property =>

      Consider the Net Height space of the current view of page [Not including space of the (Menu bar) ]

    3. {outerWidth} property =>

      Consider all the gross Width space of the current view of page [ Including space of the (Side Scroll bar) ]

    4. {outerHeight} property =>

      Consider all the gross Height space of the current view of page [ Including space of the (Menu bar) ]
