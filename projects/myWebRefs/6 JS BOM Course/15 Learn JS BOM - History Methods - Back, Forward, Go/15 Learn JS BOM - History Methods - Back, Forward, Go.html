15 Learn JS BOM - History Methods - Back, Forward, Go Main Topics :

> Main Subjects in this lesson are as following :

  1. history.back(); => To Go Back one step in the History list items
  2. history.forward(); => To Go Next one step in the History list items
  3. history.go(index number); => To Go to certain number [forward or previous] of the link in the Histroy
  4. history.length; => To get the number of links in the History list

15 Learn JS BOM - History Methods - Back() || Forward() || Go() , Concept:

=> [ history - Reload || history - Back() || Forward() || Go() ] :

  1. Main concept of {history.back()} || {history.forward()} || {history.go()} =>
    1. {history.back()} Method =>

      To go back one step in the history list

    2. {history.forward()} Method =>

      To go Next one step in the history list

    3. {history.go(+Number / -Number)} Method =>

      To go certain linkn in the history list by using index of the item in the history list

      [+#]: To go next link in the Histroy list by steps of number

      [-#]: To go back link in the Histroy list by steps of number
