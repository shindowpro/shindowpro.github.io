JS HTML BOM [Intro & What is BOM] Concept:

=> [ Intro & What is BOM ] :

  1. Main concept of {BOM} =>

    Refers to => {Browser Object Model}

    {BOM} => A collection of Objects than Related and controling the Browser

    It is related to controlling to the Browser Features and properties

    It is NOT related to Document functions and properties

  2. Examples of Functions of {BOM} =>

    Changing the URL of the Broweser

    Background Requets

    Window Open || Window Close || Window move

    Changing the information of the Status Bar

    Some Functionalities and Methods as :

    1. alert() || cofirm() || prompt()

  3. Examples of main Objects in {BOM} :
    1. {window} => Main Default object in any browser
    2. {screen} => To control the browser screen
    3. {navigation} => To control the browser Navigation
    4. {location} => To control the browser Location
    5. {history} => To control the browser History
    6. {timing} => To control the browser Timing
    7. {cookies} => To control the browser Cookies
  4. Extras Information about the {BOM} :
    • Not have it's own Standards , but each broweser Add it's own BOM Standards, but there are common standards and objects such as the main BOM Objects upper mentioned
    • All JS Objects are Included by the {BOM : window}
