> Main Subjects in this lesson are as following :

  1. Window Methods - stop() => To Stop loading the page or element
  2. Window Methods - close(); => To close an opend window by the JS
  3. Window Methods - Focus(); => To Get focus on a certain page , instead of the current page

8 Learn JS BOM - Window Methods - Stop, Close, Focus Concept:

=> [ window Methods ] :

  1. Main concept of {window Methods} =>
    1. {stop()} Method =>

      To stop loading a certain element or page

    2. {close()} Method =>

      To close an opend page by the JS [Not to Close a window opend by the user]

    3. {focus()} Method =>

      To get the focus on an opend page [by using a varialbe defined by this window]

8 Learn JS BOM - Window Methods - Stop, Close, Focus Syntax & Methods:

=> {JS BOM - Window Metods: {stop()} || {close()} || {focus()} by following Conceptual Syntax as following :

[A]// {JSBOM file} :

[1] {BOM} ( {stop()} ) Examples:
a) Stoping loading an Element on the current page :

// Defining the Button Variable [To precess the stop() Method] :
var btn = document.getElementByID('btn');

// Defining onclick event => to stop() :
btn.onclick = function {
'use strict' ;
window.stop() ;
};

//===========================================================

b) Closing an opend window [Opend by Js]:

// Defining the Button Variable [To precess the close() Method] :
var btn = document.getElementByID('btn');

// Defining onclick event => to close() :
btn.onclick = function {
'use strict' ;
window.close() ;
};

//===========================================================

c) Focusing on an opend window:

// Defining the Button Variable [To precess the focus() Method] :
var btn = document.getElementByID('btn');

// Defining onclick event => to focus() :
btn.onclick = function {
'use strict' ;
window.focus() ;
};

//===========================================================

//===========================================================
