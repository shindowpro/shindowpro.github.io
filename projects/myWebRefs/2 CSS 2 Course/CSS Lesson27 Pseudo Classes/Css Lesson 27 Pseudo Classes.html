[Pseudo Classes] Main Concept

These Special Reserved Classes in CSS help to allocate specific Elements and assigning it with special Properties.

[Pseudo Classes] ara very helpful to programer and developers in coding with (CSS) and Assigning specific porperties to specific element/multiple elements in web-page .

Type of [Pseudo Class] : [ First-Letter ]

Pseudo Class: First-Letter Concept

The First-Letter Pseudo Class means to Determine the first Letter of detected element to allocate it with special properties by Css.

The First-Letter Pseudo Class can be initiated to muliple elements by mixind this First-Line Type with other types of Pseudo Classes & Selectors.

Pseudo Class: First-Letter Examples

(1) Specifing first letter of a simple Paragraph with special Properties:

Write:     =>     P: First-Letter {Css Properties}

This is Noraml Paragraph


(2) Specifing first letter of multiple Divs with special Properties::

Write:     =>     div: First-Letter {Css Properties}
This is Noraml Div 1
This is Noraml Div 2
This is Noraml Div 3
