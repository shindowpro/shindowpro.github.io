CSS Course Lesson 3 [CSS Styling Modes]

This lesson about Main Styling modes in CSS language

Introduction to CSS   Styling   Modes

There are 3 main CSS Styling modes:

  1. External Style Mode
  2. Internal Style Mode
  3. Inline Style Mode
(1)External Style Mode

This Mode is about use an external CSS file and import it in Current Web-Page.

To implement this importing, we type file path & Kind of relation with page in < link/ > Tag like following:

<link   rel= "stylesheet"   href= "Filename.css"   />

You can also import more than 1 CSS file in the same page design as following:

<link   rel= "stylesheet"   href= "style1.css"   />
<link   rel= "stylesheet"   href= "style2.css"   />

CSS File is created by the same Editor but with different extension (.css)

CSS File containing the Style declerations of tags properties and its vlaues. And its being written directly with out [ <style> </style> ] tag, as following:

Selector{ Property Name: Value of Property }

For Example:

div{ text-align: center;   padding: 10px;   margin: 0px 0 }
p{ text-align: left;   color: blue;   font-weight: bold }

If there are more than one external CSS files im the same page design, the priority in intercepted styling tags for the Latest CSS File

(2)Internal Style Mode

This Mode is about type Styling codes directly within <style> </style> in Current Web-Page.

The priority in implementation is for [Internal Style Mode] over [External Style Mode]

<style>
Selector{ Property: value }
</style>

For Example:

<style>
p{ text-align: left;   color: blue;   font-weight: bold }
div{ text-align: center;   padding: 10px;   margin: 0px 0 }
</style>
(3)Inline Style Mode

This Mode is about type Styling codes as one of Tag/Element(Selector) Property in HTML Current Web-Page Designing.

The priority in implementation is for [Inline Style Mode ] over [Internal Style Mode]

<SelectorName(Element) Style= "Property: value" > The Element Contents </SelectorName(Element)>

For Example:

<p Style= "font-weight: bold" > This is Test Paragraph. </p>
Style Mode General Implementation Priority

The Priority of Style Mode Implementation in CSS even it was by [Class] or [Id] tool as following:

  1. (1) Inline Style Mode
  2. (2) Internal Style Mode
  3. (3) External Style Mode
