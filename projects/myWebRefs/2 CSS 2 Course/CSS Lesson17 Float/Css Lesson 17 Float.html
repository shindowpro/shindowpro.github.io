First/ Float Property:

(A)Float: None

(1)None float value (Default Float) Concept:

The [None] default value of Float property is to cancel Floating of the element.

(B)Float: left

(1)left Float Concept:

The element with left float will be floating towards left of page.

(2)left Float elements Examples:

Graphic Softwares:

Adobe Photoshop
Illustrator
Corel Draw
(C)Float: Right

(1)Right Float Concept:

The element with right float will be floating towards right of page.

(2)Right Float elements Examples:

Graphic Softwares:

Adobe Photoshop
Illustrator
Corel Draw

Second/ Clear Property:

Clear property

(1)Clear Property Concept:

[Clear] property is improtant to deactivate [Float] property to the previous or following element/s
therefore maintain the stability and acceptance of the page's elements format.
[Clear] property need an separated element to be formated with it to be activated properly.
[Clear] property has [3] values (Left, Right, Both).
