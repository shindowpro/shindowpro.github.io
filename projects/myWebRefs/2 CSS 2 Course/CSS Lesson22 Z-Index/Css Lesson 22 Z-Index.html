[Z-Index] Main Concept

This property is an Numeric Index to arrange elements display priority in Screen Web Page.

The value of this property can (Auto), or (#) numeric Value.

The element with lower numbers will be lower in Display priority.

Z-index: Auto

[Z-index: Auto] (Concept):

By this [Auto] value of z-index property the elements will arranged by thier coded priority in Editing programe

The elements will be arranged (from older to most recent)

Z-index: Auto (Example):

Div 1
Div 2
Div 3

Z-index: Numeric Value

[Z-index: # Numeric Value] (Concept):

By this [# Numeric] value of z-index property the elements will arranged by it index

The elements will be arranged (from smaller to larger)

[Z-index: # Numeric Value] (Example):

Div 1 This div [Z-Index:2] will be up on div 3 & and under div 2
Div 2 This div [Z-Index:3] will be in top of all divs
Div 3 This div [Z-Index:1] will be in bottom of all divs
