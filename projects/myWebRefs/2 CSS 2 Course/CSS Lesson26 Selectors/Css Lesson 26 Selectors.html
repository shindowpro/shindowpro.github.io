[Selectors] Main Concept

These Symbols is meant by calling specific elements in page with certain conditions.

[Selectors] ara very helpful to programer and developers in coding with (CSS) and Assigning specific porperties to specific element in web-page .

Type of [Selector] : [ (*) (All) ]

Selector: (*) (All) Concept

The Astric Selector means to select all elements in the following range.

The Astric Selector can be as public selector of all webpage or all elements in specific range.

Selector: (*) (All) Examples

(1) Selecting all elements in the whole webpage:

Just writing:     =>     * {Css Properties}

(2) Selecting all elements with in specific Parent element:

Writing:     =>     Element Name(Parent) * {Css Properties}

Type of [Selector] : [ (>) (Direct Child) ]

Selector: (>) (Direct Child) Concept

The Greater than Selector means to select only the direct Child/s element/s of certain type with in a specific parent element.

The Greater than Selector will NOT select a Grand Child element (Indirect Next Child element).

The Greater than Selector will NOT run select the parents of determined element. (the Parent of the parent Element)

Selector: (>) (Direct Child) Example

Just writing:     =>     Parent name > Child name {Css Properties}

Selecting all Direct Childs Paragraphes elements of the Parent and assign unified Properties :

Just writing:     =>     Parent name > p {Css Properties}

Type of [Selector] : [ (+) (First Next Sibling) ]

Selector: (+) (First Direct Next Sibling) Concept

The Plus Selector means to select the first next element with the same level (with same parent-Sibling-) ONLY.

The Plus Selector will select next sibling type writen even if it was Not the next line, but must be with same parent.

The Plus Selector will NOT select further more the next sibling of determined element.

Selector: (+) (First Next Sibling) Example

Just writing:     =>     Sinbling name + Sibling name {Css Properties}

Selecting the next sibling Paragraph element to div element and assign specific Properties :

Just writing:     =>     Div + p {Css Properties}

The Hymin Selector will ONLY select next following siblings of detected type writen , and NOT the previous slibling of detected type.
