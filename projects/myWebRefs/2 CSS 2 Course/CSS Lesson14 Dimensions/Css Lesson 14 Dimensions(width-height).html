(A)[Width] Dimension Property

(1) Width Default Value:

The defualt value of [Width] dimension of element is [auto].
And this is mean that the element width will be as its parent container element width.

(2) [Min-width] property:

The defualt value of [Min-Width] property is [auto].
Default value of min-height means that the element minimum width will automaitcally stretch to fit its container width.
A specific value of [min-width] means that element will start its widht with this value, even if it empty or its containings were less than min-height.

(3) Width [Max-width] property:

The defualt value of [Max-Width] property is [auto].
Default value of max-width means that the element maximum width will be as much as its parent container element width value.
A specific value of [Max-width] means that the element maximum width will not exceed this value even if its containings is longer than its width.
(And here we can use [over-flow] property to control item's containings over flow -hidden/scroll- to maintain the acceptable design)

(4) [Percentage Value] of Width property:

The defualt value of width dimension of element is [100%].
And this is mean that the element width will be as ALL its parent container element width.
Usage this type of value requires a pre determined value of element's parent width value to be calculated upon it.

(B)[Height] Dimension Property

(1) Height Default Value:

The defualt value of [Height] dimension of element is [auto].
And this is mean that the element height will automatically fit its containing vertically.

(2) [Min-height] property:

The defualt value of [Min-height] property is [auto].
Default value of min-height means that the element height will be nothing if its containings was empty, and will keep stretching until reaching its max height.
A specific value of [Min-height] means that the element start height will initiate with this value of height even if the item was empty or its containing is less than its min-height value.

(3) [Max-height] property:

The defualt value of [Max-Height] property is [auto].
Default value of max-height means that the element height will keep stretching until reach this value of [max-height].
A specific value of [Max-height] means that the element maximum height will not exceed this value even if its containings is longer than its height.
(And here we can use [over-flow] property to control item's containings over flow -hidden/scroll- to maintain the acceptable design)

(4) [Percentage Value] of Height property:

The defualt value of width dimension of element is [100%].
And this is mean that the element height will be as ALL its parent container element height value.
Usage this type of value requires a pre determined value of element's parent height value to calculate its percentage upon parent height value.
