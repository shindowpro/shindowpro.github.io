Lesson Index

Position Property is a very important one.
It is provides [4] positioning modes for the element
And allows to full control of any element'position in web-page.

  1. Position: Static
  2. Position: Fixed
  3. Position: Relative
  4. Position: Absolute

(1)Static position (Default position) Concept:

This default position for the element to be holded in its position and not to be attached with Scrolling.

(C)Position: Relative

(1)Relative Position Concept:

In this position element will be relative to its container Position.
Also the element margins are reserved to the element even after changing its position and can't be replaced by any other surrounding element

(D)Position: Absolute

(1)Absolute Position Concept:

In this position element will be in absolute Position mode but in parent container only with Relative Position.
And if there are not any of it container with relative position ... the element will be in absolute position mode to the body it self.
