[Content] Main Concept

This property is about to add extra additions before/after any element/selector in Web-Page.

The value of this property can be added [Before], or [After] any Element by CSS 2 codes.

In CSS 2 course we will only use string value [text] Content, but in further coureses we can use Content to Add many differents elements & Values.

[Content: Before]

[Content: Before] (Concept):

By this [Before] value of Content property the elements will be previous by written Text Value in CSS Editor Application.

  Div 1
Div 2
Div 3

Content: After

[Content: After] (Concept):

By this [# Numeric] value of z-index property the elements will arranged by it index

The elements will be arranged (from smaller to larger)
