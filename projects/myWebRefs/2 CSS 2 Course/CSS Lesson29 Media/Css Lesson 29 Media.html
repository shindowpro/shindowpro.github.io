[ Media ] Main Concept

The Media Tag is a an special attribute for <Link> Tag that is provide several Methods to display the page in certian multimedia devices [ TV, Print, ... ].

[Media] type can be totally specified for 1 CSS file (as over all) and applied for 1 web-page or more than 1 web-page (multiple Pages) at the same time by Css     File multiple allocating [HTML Method].

[Media] type can be partially specified for certain webpage elements and specifying other type of Media for other elements, in CSS page [CSS Method].

Media Control 1 [ HTML Method ]

Media Control [ Method 1 (In HTML Page) ] Concept:

Main Concept of Media Control by HTML method is specifing the Media type for the whole HTML WebPage with in the html pagecode itself.

This can be done through the <Link> Tag in [Media] Attribute.

Media Control [ Method 1 (In HTML Page) ] Syntax:

HTML Coding Page Editor:
=>     < Link   rel="stylesheet"   href="filename.css"   media= "All" />
=>     < Link   rel="stylesheet"   href="filename.css"   media= "Screen" />
=>     < Link   rel="stylesheet"   href="filename.css"   media= "Print" />
Media Control 2 [ Css Method ]

Media Control [ Method 2 (In CSS Page) ] Concept:

Main Concept of Media Control by CSS method is specifing the Media type for partial CSS properties of some elements in webpage with in the CSS pagecode.

This can be done through the CSS Coding page.

Media Control [ Method 2 (In CSS Page) ] Syntax:

CSS Coding Page Editor:
=>     @Media   All {     Element Name { Css Properties : Property Value }     }
=>     @Media   Screen {     Element Name { Css Properties : Property Value }     }
=>     @Media   Print {     Element Name { Css Properties : Property Value }     }
Media Type : [ All ] (Default)

Media Type [ All ] (Default) Concept:

Main Concept of Media Type [All] is that the Css properties completly or partially will be applicable to ALL kinds of devices and modes.

This can be applied completly on the whole page by the <Link> Tag in [Media] Attribute, or partially on some page elements by Css coding page.

Media Type [ All ] Syntax:

HTML Coding Page Editor:
Completly (HTML Method) =>     < Link   rel="stylesheet"   href="filename.css"   media= "All" />
CSS Coding Page Editor:
Partially (Css Method) =>     @Media   All {     Element Name { Css Properties : Property Value }     }
Media Type : [ Screen ] (Default)

Media Type [ Screen ] Concept:

Main Concept of Media Type [ Screen ] is that the Css properties completly or partially will be applicable to Screen Devices ONLY.

This can be applied completly on the whole page by the <Link> Tag in [Media] Attribute, or partially on some page elements by Css coding page.

Media Type [ Screen ] Syntax:

HTML Coding Page Editor:
Completly (HTML Method) =>     < Link   rel="stylesheet"   href="filename.css"   media= "Screen" />
CSS Coding Page Editor:
Partially (Css Method) =>     @Media   Screen {     Element Name { Css Properties : Property Value }     }
