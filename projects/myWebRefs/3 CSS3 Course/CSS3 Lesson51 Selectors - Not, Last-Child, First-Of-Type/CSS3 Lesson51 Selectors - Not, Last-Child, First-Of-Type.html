[CSS3 Lesson51 Selectors - Not || Last-Child || First-Of-Type || Last-Of-Type ] Concept:

{Selector- [:Not()] {Not()} =>

  • This selector to select any children Elements in certain parent Except a cetain chlid, according to one it's property [Tag name || class name || Other selector ]
  • This selector needs to write the Creteria inside the selector bracketes()

{Selector- [:Last-Child] {Last-Child}} =>

  • This selector to select the last child inside the certain parent container {regardless of its type}

{Selector- [:First-Of-Type] {First-Of-Type}} =>

  • This selector to select the first child of certain type inside [regardless of it's overall position or place inside it's parent]

{Selector- [:Last-Of-Type] {Last-Of-Type}} =>

  • This selector to select the last child of certain inside [regardless of it's overall position or place inside it's parent]

[CSS3 Selectors - Not || Last-Child || First-Of-Type || Last-Of-Type] Importance & Benefits:

=> [ CSS3 Selectors - Not || Last-Child || First-Of-Type || Last-Of-Type] : This Advanced Selectors providing the following features & imporatance :

  • High Accessing ability to any element inside webpage document {according to different Creterias }
  • These advanced Selectors will limit using of J-Query & JS scripting , and using pure CSS codes instead
