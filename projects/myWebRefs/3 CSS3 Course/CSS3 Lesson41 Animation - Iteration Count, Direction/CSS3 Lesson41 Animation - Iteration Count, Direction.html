[CSS3 Animation - [Iteration Count || Direction] ] Concept:

{Animation- [Iteration-Count || Direction]} => Is Needing for using [prefixes] before the animation property and it's branshed properties

{Animation - [Iteration Count || Direction]}: These are the first and second child perperites of Animation syntax , with following details :

  1. { -webkit-animation-iteration-count || animation-iteration-count} =>

    Which is the Animation repeating count

    Default value of Animation-iteration-count : 1

    Animation-iteration-count => can be set by : {# number} || {infinite}: (Which ins meaning to repeate the Animation forever )

  2. { -webkit-animation-direction || animation-direction} =>

    Which is the Animation Direction Mode to be displayed with

    [Default value] : normal

    Can be assigned by following values :

    1. [normal] => To Start the animation funcion from the first step to the last step
    2. [reverse] => To Start the animation funcion from the last step to the first step
    3. [alternate] => To Start the animation funcion from the [last step -> the first step] , and reverse the animation back step again
    4. [alternate-reverse] => To start the animation funcion from the [Last step -> First step] , and reverse the animation back step again

[CSS3 / Animation - [Iteration-Count || Direction] ] Importance & Benefits:

=> [CSS3 / Animation - [Iteration-Count || Direction] ] : This Advanced sub properties enabling the following features & imporatance :

  • [animation-iteration-count] : Give a ability to assign a certain count of repeating the Declared Animation, with ability to set it (forever by infinite) to a specific Element [can be set diffrerntly for each element]
  • [animation-direction] : Give a ability to assign a certain direction mode to Declared Animation for to a specific Element [can be set diffrerntly for each element]
