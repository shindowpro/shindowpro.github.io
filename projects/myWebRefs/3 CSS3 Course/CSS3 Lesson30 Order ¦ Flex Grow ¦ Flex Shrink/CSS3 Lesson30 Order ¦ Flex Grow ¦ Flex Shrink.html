[CSS3 Lesson30 Order ¦ Flex Grow ¦ Flex ShrinkContent] Concept:

{Order ¦ Flex Grow ¦ Flex ShrinkContent}: These properties are allocated to the parent Container (Children Element => part:1)

  • First (Order) :

    [Order] => This property to set cdrtain order of arranging children inside the parent container

    [Order] => This property is being for each single element by value of desired arrange / order

    [Order] => This property not needs to arrange items in HTML script

    [Order] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
  • Second (Flex-Grow) :

    [Flex Grow] => Is one of the (Children Elements => part:1)

    [Flex Grow] => This property to control growing ratio of the certian child Element {incase of parent container width changed}

    [Flex Grow] => The default value is : {0}: Which means not to grow when parent container's width changed & {use auto Total width dividing }

    [Flex Grow] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Changing in parent container width .

    [Align-Items] => This property can be assinged wtih one following values :

    1. {0} : The Disabling Value (not to activate Child grow if parent conatiner changed )
    2. {1} : The Default Value Activate the flex grow by ratio of [1 time of setted child element width] {if parent conatiner changed & there enough space for one/more child elements}
    3. {2 -> ...} : Activate the flex grow by ratio of [2 -> ... time of setted child element width] {if parent conatiner changed & there enough space for one/more child elements}
  • Third (Flex-shrink) :

    [Flex-shrink] => Is one of the (Children Elements => part:1)

    [Flex-shrink] => This property to control shrinking ratio of the certian child Element {Decrease of parent container width changed}

    [Flex-shrink] => The default value is : {1}: Which means not to shrink when parent container's width changed & {use auto width dividing }

    [Flex-shrink] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Changing in parent container width .

    [Flex-shrink] => This property can be assinged with one following values :

    1. {0} : The Disabling Value (not to activate Child shrink if parent container's width changed )
    2. {1} : The Defuault value to Activate the flex shrink by ratio of [1 time of setted child element width] {if parent conatiner changed}
    3. {2 -> ...} : Activate the flex shrinking by ratio of [2 -> ... time of setted child element width] {if parent conatiner changed }
