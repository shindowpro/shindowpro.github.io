[2D Transform - Matrix()] Concept:

=> [2D Transform - Matrix()] : This is one of CSS3 advancd Properties to control the Element all transform Properties Matrix (by controoloing speicfic Element Value) .

=> [2D Transform - Matrix()] : This is one of CSS3 {2D Transform} Properties Functions, which has the same in 3D Transform .

=> [2D Transform - Matrix(a, b, c, d, e , x , y ) ] => It is containing following {6} aspects/values/sub propeties .

  • [a] => [ ScaleX : Horizontal Scaling value]
  • [b] => [ SkewY : Vertical Skewing Angel ]
  • [c] => [ SkewX : Horizontal Skewing Angel ]
  • [d] => [ ScaleY : Vertical Scaling ]
  • [x] => [ TranslateX : Horizontal tanslating value ]
  • [y] => [ TranslateY : Vertical tanslating value ]

=> Each 2D-tranform Property can be controled by the Matrix() property as following :

  • {transform : translate()} => [ transform: Matrix(1 , 0 , 0 , 1 , +/-x , +/-y) ] => Can be controled by changing [x & Y] of tranlate value + beside the zeroing other value properties

  • {transform : scale()} => [ transform: Matrix(+/-a , 0 , 0 , +/-b , 0 , 0) ] => Can be controled by changing [a & d] of scale value + beside the zeroing other value properties

  • {transform : rotate()} => [ transform: Matrix(Calc value of degree , .5 , -.5 , Calc value of degree , 0 , 0) ] => Can be controled by changing [a & d] of scale with the same value as a circle cos (degree) + beside the (b , c) of skewing values with [.5 , -.5] + other value can be zero

  • {transform : All Combined()} => [ transform: Matrix(.86 , .5 , -.5 , .86 , value , value) ] => This will control all propeites of 2d-tranform

  • {transform : Customized() [ X-Fliping element ]} => [ transform: Matrix(-1 , 0 , 0 , 1 , 0 , 0) ] => This will Flip the Element Horizontally

  • {transform : Customized() [ Y-Fliping element ]} => [ transform: Matrix(1 , 0 , 0 , -1 , 0 , 0) ] => This will Flip the Element Vertically

=> This Matrix property's live changing can be acheived by [inspect element] option in browser {as all other properties}

=> This Matrix property's will gives great effect [sensivity] when using with transition property

=> This Matrix property's will gives great effect [great animation effects ] when using with animation property
