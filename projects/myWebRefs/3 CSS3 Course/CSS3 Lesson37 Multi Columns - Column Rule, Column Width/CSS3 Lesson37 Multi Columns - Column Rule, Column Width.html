[CSS3 Multi Columns [Column Rule, Column Width] ] Concept:

{Multi Columns [Column Rule, Column Width]}: These properties and all of it's proerties need to add web prefixes to work correctly

{Multi Columns [Column Rule, Column Width]}: Is providing the best solution for separeted desiging Templates (Textual) as Columns by simple way with out further old CSS perperties

{Multi Columns [Column Rule, Column Width]}: This property is recommended for [Texutal content] only, and not for other elements types

{Multi Columns [Column Rule, Column Width]}: These (2) new properites of the Multi Columns is the {part2}

  1. First property => {Column-Rule} : Which is the between Border which is lies in the Gap space
  2. Second property => {Column-Width} : Which is the Column width itself (not the Border Between[colomn-rule])
