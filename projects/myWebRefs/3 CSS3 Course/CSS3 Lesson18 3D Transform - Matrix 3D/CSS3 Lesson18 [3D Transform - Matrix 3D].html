[3D Transform - Matrix3D()] Concept:

=> [3D Transform - Matrix3D()] : This is one of CSS3 advancd Properties to control all 3d-Transform Properties of the Element by assinging all 3 dimensions of all properties in one function inside Matrix() .

=> [3D Transform - Matrix3D()] => It is containing following {16} aspects/values/Parameters :

  • [1] => [ ScaleX : Horizontal Scaling value => (+/-): Controling Width of the Element ]
  • [2] => [ SkewX : Horizontal Skewing value => (+/-): Controling Horizontal Skewing Effect of the Element ]
  • [3] => [ SkewY : Veritcal Skewing value => (+/-): Controling Veritcal Skewing Effect of the Element ]
  • [4] => [ => (+/-): ]
  • [5] => [ ScaleY : Veritcal Scaling value => (+/-): Controling Height of the Element ]
  • [6] => [ Rotate Value Positive Deg : Horizontal/Vertical Rotating Cosine value => (+/-): Controling Horizontal/Vertical Rotating Effect of the Element ]
  • [7] => [ => (+/-): ]
  • [8] => [ => (+/-): ]
  • [9] => [ Rotate Value Negative Deg : Horizontal/Vertical Rotating -Cosine value => (+/-): Controling Horizontal/Vertical Rotating Effect of the Element ]
  • [10] => [ 1 ]
  • [11] => [ TranslateY : Vertical Translating value => (+/-): Controling Up-down Moving of the Element ]
  • [12] => [ TranslateX : Horizontal Translating value => (+/-): Controling right-left Moving of the Element ]
  • [13] => [ TranslateZ : Zooming Translating value => (+/-): Controling ZoomingIn-ZoominOut Moving of the Element ]
  • [14] => [ ScaleZ : Stretching Scaling value => (+/-): Controling StretchingIn-StrectchingOut Scale of the Element ]
  • [15] => [ 1 ]
  • [16] => [ 1 ]

=> {3D Matrix property can handle any 3d Transform property's value, more or All of them at once code }

=> [3D Transform - Matrix3D() ] : This Advanced property enabling the following features :

  • This Matrix3D property's will gives great effect [sensivity] when using with another Transition property
  • This Matrix3D property's will gives great effect [great animation effects- stretching Effect ] when using with animation property
  • Using the [3D property() will gives a control of all Dimensions of any Property at the same time
  • 3D Scaling property will provide unique effects specilaly when merged with other 3D Transform properties animations effects {which will be studied will later lessons}

[3D Transform - Matrix3D()] General Practical Applications :

=> Following is the Applied example of [3D-Transform: MatrixX,Y,Z(+/-Value px)] syntax : >>

  • (3) Fieldsets contain (2-6) section (each section is by Certain by animation Methods) for each as following:

      • A) Rotating Effect :

    1. First Fieldset [Matrix3D(+/-X , 0 , 0)] + [Rotating +60 deg] :
      • First Section => - Value Deg {Horizontal Rotating Over X Aixe [left -> Right] }
      • Second Section => + Value Deg {Horizontal Rotating Over X Aixe [Rigtht -> Left]}
    2. Second Fieldset [Matrix3D(0, +/-Y , 0)] + [Rotating +60 deg] :
      • First Section => + Value deg {Vertical Rotating Over Y Aixe [Down -> up] }
      • Second Section => - Value deg {Vertical Rotating Over Y Aixe [Up - Down]}
    3. Third Fieldset [Matrix3D(0 , 0 , +/-Z)] + [Rotating +60 deg] :
      • First Section => - Value deg {2D Rotating Over Z Aixe [Left -> Right] }
      • Second Section => + Value deg {2D Rotating Over Z Aixe [Right -> Left]}

      B) Scaling Effect :

    1. First Fieldset [Matrix3D(+/-X , 0 , 0)] + [Scaling 100px] :
      • First Section => - Value {Horizontal Scalings by X Aixe Reducing size [Right , Left] }
      • Second Section => + Value {Horizontal Scalings by X Aixe Raising size [Right , Left] }
    2. Second Fieldset [Matrix3D(0, +/-Y , 0)] + [Scaling 100px] :
      • First Section => - Value {{Veritcal by Y Aixe Reducing size [Up , Down ] }
      • Second Section => + Value {Veritcal Scalings by Y Aixe Raising size [Up , Down ] }
    3. Third Fieldset [Matrix3D(0, 0 , +/-Z)] + [Scaling 100px] :
      • First Section => - Value {Zooming Scalings by Z Aixe Reducing size [Strectching-In , Strectching-Out ] }
      • Second Section => + Value {Zooming Scalings by Z Aixe Raising size [Strectching-In , Stretching-Out ] }

      C) Translating Effect :

    1. First Fieldset [Matrix3D(+/-X , 0 , 0)] + [Translating 100px] :
      • First Section => - Value px {Horizontal Translating by X Aixe Moving [ Right -> Left ] }
      • Second Section => + Value px {Horizontal Translating by X Aixe Moving [ Left -> Right ] }
    2. Second Fieldset [Matrix3D(0 , +/-Y , 0)] + [Translating 100px] :
      • First Section => - Value px {Veritcal Translating by Y Aixe Moving [Bottom -> Top] }
      • Second Section => + Value px {Veritcal Translating by Y Aixe Moving [Top -> Bottom ] }
    3. Third Fieldset [Matrix3D(0, 0 , +/-Z )] + [Translating 100px] :
      • First Section => - Value px {Zooming Translating by Z Aixe Moving [Zooming-Out] }
      • Second Section => + Value px {Zooming Translating by Z Aixe Moving [Zooming-In] }
