Selectors - Nth-Child, Nth-Of-Type ] {Part 1} Main Topics :

{ These Selector => These type of selectors [nth] are commonly to use in accessing to Elements + addional features


{Selector[Nth()] - group ] => Has some additional Features in comparison to the types of Selectors, such as :

  • Can be used to access According to Child Index Or Types Index , by adding {-child} || {-Of-Type} prefixes to [Nth]
  • Can be added with (3) types of accessing creterias inside the brackets() such as :
    • [Number] = > which is the required number of index
    • [Keyword] = > which is a certain Accessing Creteria to the element, such as
      1. {Odd: the Odd number in arrangment }
      2. {Even: the even number in arrangment }
    • [Formula] = > which is more complicated certia to accesss the element {will be teached in the next }
  • Can be added with more Complicated Frefixes to gain the reverese Aceessing Creteria, wether in using the {-Child} prefix || {-Of-Type} prefix, as followikng :
    1. [:Nth-Last-Child()] => To reach the [assigned number] child in the Parent Conatiner starting counting with the last child
    2. [:Nth-Last-Of-Type()] => To reach the [assigned number] child in the Parent Conatiner starting counting with the last Type of Child
    3. [:Nth-Last-Child(even)] => To reach the [assigned number] child in the Parent Conatiner starting counting Eventualy (from second child) with the last child
    4. [:Nth-Last-Child(odd)] => To reach the [assigned number] child in the Parent Conatiner starting counting Oddinaly (from second child) with the last child
    5. [:Nth-Last-Of-Type(even)] => To reach the [assigned number] child in the Parent Conatiner starting counting Eventualy (from first type child) with the last Type of Child
    6. [:Nth-Last-Of-Type(Odd)] => To reach the [assigned number] child in the Parent Conatiner starting counting Oddinaly (from first type child) with the last Type of Child
