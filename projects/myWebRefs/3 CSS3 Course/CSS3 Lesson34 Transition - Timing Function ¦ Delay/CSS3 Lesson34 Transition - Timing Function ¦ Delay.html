[CSS3 Transition Timing Function ¦ Delay ] Concept:

{Transition Timing Function ¦ Delay}: These properties are the Third & fourth parts of Transition Function

{Transition Timing Function ¦ Delay}: These Two Basic properties are optional in shorthand function to be activate tarnstion Function proprly

{Transition Timing Function ¦ Delay}: These properties in details (Syntax) as following :

  1. First: {transition-Timing function} => This is property is responsible for the type Transition speed curve of the Element which the transition will be applied on
    • {transition-Timing function} => Default Value is [ease] : which means to set curve speed to be easy on start & easy in End
    • {transition-Timing function} => Can be set by one single property name OR Multiple properties names (Seperated by ,) in order
    • {transition-Timing function} => Has (2) major Sections :
      1. Frist Section [Built-in] Timing Functions (6) type:
        1. {ease} => To start the transtion easy & end it easy also
        2. {ease-in} => To start the transition easy only
        3. {ease-out} => To end the transition easy only
        4. {ease-in-out} => To start the transtion easy & end it easy also
        5. {linear} => To start the transition end it by the same ratio 1
      2. Second Section [Customized] Timing Fucntion (2) types :
        1. {steps} => This is a function to seperate the transition to several steps,
          with the ability to assign certain mode of assensrunizing the transition effect and step, from following :
          • [start] : To start the transtion immediatly with from the first step
          • [end] : To start the first step first then start the transition's first step
        2. {cubic-bezier} => This is an advanced function to seperate the transition to several steps,
          with the ability to assign certain mode of assensrunizing the transition effect and step, with following tips
          • This method can assign several numbers of steps,
          • Each number value is a range of step speed curve [0 -> 1]
  2. Second: {transition-Delay} => This is Transition Delaying Time that the Element will spend to start the Transition
    • {transition-Delay} => Default Value is [0s] : which means to transate the Element's property(s) instantely with out any delaying .
    • {transition-Delay} => Can be set by one single duration time OR Multiple duration times (Seperated by ,) in order
    • {transition-Delay} => Can be set by both (s -> Second || ms-> MilliSecond)

[CSS3 Lesson 34 Transition Timing Function ¦ Delay ] Importance & Benefits:

=> [CSS3 Lesson 34 Transition Timing Function ¦ Delay ] : This Advanced property enabling the following features & imporatance :

  • [transition-Timing function] & [transition-Delay] => Add much more wonderful effects to animating elements
  • [transition-Timing function] & [transition-Delay] => Gives a great specific Tool to reach varaiety of Transtion-timing-function built-in of the speed curves
  • [transition-Timing function] & [transition-Delay] => Gives a great amazing Tool to customize certain speed curves , with ability to merge certain steps with certain type of transition
