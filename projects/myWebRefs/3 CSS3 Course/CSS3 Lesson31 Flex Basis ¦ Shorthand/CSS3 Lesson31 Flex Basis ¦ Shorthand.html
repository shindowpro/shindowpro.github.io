[CSS3 Flex Basis ¦ flex Shorthand ] Concept:

{Flex Basis ¦ Flex Shorthand}: These properties are allocated to the parent Container (Children Element => part:2)

  • First (Flex-Basis) :

    [Flex-Basis] => This property to set certain Flexible Width to a certain Child(s) element(s)

    [Flex-Basis] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Changing in Parent Container's Width .
  • Second (flex Shorthand code) :

    [flex Shorthand] => Is a shortcut overall Code of all Children Flex properties codes

    [flex Shorthand] => It is as default => {flex: 0 1 auto}

    [flex Shorthand] => It is contain Following items/ properties

    1. {flex-grow} : First value is flex-grow property value [0 - ...] / default: [0]
    2. {flex-shrink} : Second value is flex-shrink property value [0 - ...] / default: [1]
    3. {flex-basis} : Third value is flex-basis property value [# value px] / default: [auto]

    [flex Shorthands] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Changing in parent container width: {larger : to display flex-grow Effect} & {smaller: to display flex-shrink Effect}

[CSS3 Lesson 31 Flex Basis ¦ Shorthand ] Importance & Benefits:

=> [CSS3 Lesson 31 Flex Basis ¦ Shorthand ] : This Advanced property enabling the following features & imporatance :

  • Providing better, easier, practical, shortcut and profficent Renposive Desiging [Specially Mobile Scalls], Much more easier way than old methods
  • Using {Flex-Basis} => Provides useful methods for specifing certian element child by flexiable width
  • Using {Flex Short hand} => Provides very easy method/tool to write all Flex properties of children codes in one line
