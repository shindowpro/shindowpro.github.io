[Box Shadow] Concept:

=> [Box Shadow] : It is a property of desiging a Shadow Arrount the Element according to multiple properties.

=> Main Properties of Box shadow as following:

  • Horizonatl Value [Positive, Negative] => which determine whether : Bottom Box-shadow{positive} / the upper Box-Shadow {Negative}
  • Vertical Value [Positive, Negative] => which determine whether : Right Box-shadow{positive} / the Left Box-Shadow {Negative}
  • Blur Value [Positive only] => which determine the value of blur (the Fog Effection) for the box Shadow
  • Spread Value [Positive only] => which determine the value of spread (the Shadow Extension Effection) for the box Shadow
  • color Value [ rgb - argb - hexa - Name ] => which determine the color of the box Shadow
  • inset-outset Value [inside/outside] shadow => which determine the direction of the box Shadow {inset: inside the Element} / {without writing(Default): outside the Element }

=> { Multiple Box Shadows}: It is the property of setting more than one box-shadow for the same Element

=> {One General Box-shadow full Syntax}: This is the Full Syntax of the Box-Shadow (According the priority of reading) :
[box-shadow: Horizontal Value | Vertical Value | Blur Value | Spread Value | Color | inset-outset]

Box Shadow [Box Shadow ] Importance & Benefits:

[Box Shadow] gives us the Following Features :

  • It provides designing beautiful Shadows of a cetain Elements with out using a spicific Applications
  • It provide high control of Element's Box Shadow by multiple tools & properties.
  • This Property is now Supportes by all types of kinds of Browsers [new versions] .

[ Box Shadow ] Syntax & Methods:

// *) [Box-shadow] Syntax as following :

//1) [Box-shadow] General Full properties Syntax
//a) [Box-shadow] General Full properties Syntax [Positive Values]
.ElementClassName {
box-shadow :10px 10px 8px 5px #DDD;
}
// ------------------------------------------------------------------------------

//b) [Box-shadow] General Full properties Syntax [Negative Values]
.ElementClassName {
box-shadow :-10px -10px 8px 5px #DDD;
}
// ------------------------------------------------------------------------------

//c) [Box-shadow] General Full properties Syntax [inset]
.ElementClassName {
box-shadow :-10px -10px 8px 5px #DDD inset;
}

.ElementClassName {
box-shadow :-10px -10px 8px 5px #DDD inset;
}
// ------------------------------------------------------------------------------

//2) [Box-shadow] Shorten properties Syntax (most Public)
.ElementClassName {
box-shadow :-10px -10px 8px #DDD;
Value1 : The Horizontal Value
Value2 : The Vertical Value
Value3 : The blur Value
Value4 : Color
}

.ElementClassName {
box-shadow :-10px -10px 8px #DDD inset;
Value1 : The Horizontal Value
Value2 : The Vertical Value
Value3 : The blur Value
Value4 : Color
Value5 : box-shadow Direction
}

//===========================================================

//3) [Box-shadow] Multiple box-shadows or the same Element :
//a) [Box-shadow] Multiple box-shadows (with the shorten Form):
.ElementClassName {
box-shadow :-5px -5px 6px #00f; {first box-shadow}
box-shadow : 5px 5px 6px #f00; {Second box-shadow}
box-shadow : 10px 10px 8px #080; {Third box-shadow}
}

//b) [Box-shadow] Multiple box-shadows (with the Full General Form):
.ElementClassName {
box-shadow :-5px -5px 6px 5px #00f; {first box-shadow}
box-shadow : 5px 5px 6px 5px #f00; {Second box-shadow}
box-shadow : 10px 10px 8px 5px #080; {Third box-shadow}
}
