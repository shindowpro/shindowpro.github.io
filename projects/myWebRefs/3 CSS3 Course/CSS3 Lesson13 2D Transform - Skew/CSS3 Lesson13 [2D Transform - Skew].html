[2D Transform - Skew()] Concept:

=> [2D Transform - Skew()] : This is one of CSS3 Properties to control the Element Angular Skew Amount property (Horizontally, or Vertically) .

=> [2D Transform - Skew()] : The one Value of Skew function => will only gives Angle value to the Horizontal Axe and vertical axe with 0 value

=> [2D Transform - Skew()] : This is one of CSS3 {2D Transform} Properties Functions, which has the same in 3D Transform .

=> There are (2) Methods to write this property as following :

  • [transform: skew(X: Horz Deg value, Y: Vert Deg Value) ] => This Can Hold one angluar Value to each Direction

  • [transform: skewX (Horz Deg value) / : transformY: skewY (Vert Deg Value) ] => This Can control only detected Direction

=> This Skew Value is about amount of skewing angle the Element toward a certain Dimension(s) as following examples :

  • [X] => Skew Element to the Left Direction
  • [-X] => Skew Element to the Right Direction
  • [Y] => Skew Element to the Bottom Direction
  • [-Y] => Skew Element to the Top Direction

=> To activate skew() Property properly with a clear Difference from the rotate() => It is recomended to following following procedures :

  • [One Value] => It will be clear differ when write only Value of one Direction & and the other Direction [0] value, as following :

    transform: skew(+-value deg); => This will set only the Horizontal Axe

    transform: skew(+-value deg, 0);

    transform: skew(0, +-value deg);

    transform: skewX(+-value deg);

    transform: skewY(+-value deg);

  • [Two Values] => It will be clear differ when write two Different value when writing 2 values (different amounts, regardless of the - or + values)

    transform: skewX(+-value deg, +-value deg);
