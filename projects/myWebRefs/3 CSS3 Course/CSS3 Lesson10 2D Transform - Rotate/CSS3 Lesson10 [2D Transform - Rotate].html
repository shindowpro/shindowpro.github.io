> Main new Methods in this lesson are as following :

  • Differnce between [Positive Value] & [Negative Value] in controling Rotating of Element

[2D Transform - Rotate()] Concept:

=> [2D Transform - Rotate()] : This is one of CSS3 Properties to control the Element Display Angel .

=> [2D Transform - Rotate()] : This is one of CSS3 {2D Transform} Properties Functions, which has the same in 3D Transform .

=> Differnce between [Positive Value] & [Negative Value] of {Angel} in controling Element Rotating :

  • [Positive-Value] => To Rotate The Element toword the Right Side

  • [Negative-Value] => To Rotate The Element toword the Left Side

[2D Transform - Rotate() ] Importance & Benefits:

=> [2D Transform - Rotate() ] : This Advanced property enabling the following features :

  • The ability to rotate the Element toword certian direction or Angle
  • This Property is will be great in Designing nice Templates specilaly when using with (Transition Property- will be taken in future)

// ------------------------------------------------------------------------------

[2D Transform - Rotate] General Practical Applications :

=> Following is the Applied example of [2D-Transform: rotate(deg)] syntax : >>

  • (2) Fieldsets contain (6) section (each section is rotated by Certain defferent Angle) for each as following:
    1. First Fieldset [Fixed Mode] :
      • First Section => Rotated by {30deg} (to right) : [by Fixed Mode]
      • Second Section => Rotated by {-30deg} (to left) : [by Fixed Mode]
      • Third Section => Rotated by {90deg} (to right) : [by Fixed Mode]
      • Forth Section => Rotated by {-90deg} (to left) : [by Fixed Mode]

      • Fifth Section => Rotated by {180deg} (to right) : [by Fixed Mode]
