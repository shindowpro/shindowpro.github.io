[3D Transform - Translate3D()] Concept:

=> [3D Transform - Translate3D()] : This is one of CSS3 advancd Properties to control the translating Element by assinging all 3 dimension of with X , Y , Z .

=> [3D Transform - Translate3D()] => It is containing following {3} aspects/values/sub propeties [one each Time / two each / All] .

  • [X] => [ translateX : Horizontal translating value => (+/-): translating over the Horizontal Axie (X left to Right / vise versa) ]
  • [Y] => [ translateY : Vertical translating value => (+/-): translating over the Vertical Axie (Y Up to down or down to Up) ]
  • [Z] => [ translateZ : 3D (Zooming) translating value => (+/-): translating over the the Third Axie (Z Zooming in / Zooming out ): Need Perspective Property Effect ]

=> {3D translating proepty can be assing by one of the translating value or more at once }

=> {3D translating proepty {X,Y,Z} for properly activation => it needs (Perspective property in the parent element) }

[3D Transform - Translate3D() ] Importance & Benefits:

=> [3D Transform - Translate3D() ] : This Advanced property enabling the following features :

  • This Matrix property's will gives great effect [sensivity] when using with transition property
  • This Matrix property's will gives great effect [great animation effects- Zooming Effect ] when using with animation property
  • Using the [3D property() will gives a control of all Dimensions of translating at the same time , so it is better than the ordinary 2D translating
  • 3D translating property will provide unique effects specilaly when merged with other 3D Transform properties animations effects {which will be studied will later lessons}
  • This Property is will be great in Designing nice Templates specilaly when using with ( Other 2d Properties - Animation Property- will be taken in future)

[3D Transform - Translate3D()] General Practical Applications :

=> Following is the Applied example of [3D-Transform: translateX,Y,Z(+/-Value px)] syntax : >>

  • (3) Fieldsets contain (2-6) section (each section is by Certain by animation Methods) for each as following:
    1. First Fieldset [translate3d(+/-X , 0 , 0)] :
      • First Section => +Value px {Horizontal translating Left to Right}
      • Second Section => -Value px {Horizontal 3D translating Right to Left}
    2. Second Fieldset [translate3d(0, +/-Y , 0)] :
      • First Section => +Value px {Vertical translating Up to bottom }
      • Second Section => -Value Degree {Veritcal 3D translating bottom to top}

      • Fourth Section => 3D- translating { X+, Y-, Z-}
      • Fifth Section => 3D- translating { X-, Y-, Z+}
      • Sixth Section => 3D- translating { X-, Y+, Z+}
