[CSS3 Lesson25 Resize] Concept:

  • A) {CSS3 Resize} main Concept :

    • This is a new property in CSS3 , which providing the ability to [Resize] the Element by different Direction

    This is property in CSS3 is reltated to overflow property

    This property is not supported by Ineternet Explorer & old Verions of Opera

    In CSS3 default Values as following :


    • [none] => To Reset or Disabling the Resize property of the Element
    • [Horizontal] => To Activating the resize property only in Horizontal diection
    • [Vertical] => To Activating the resize property only in Horizontal diection
    • [Both] => To Activating the resize property in the Both directions

[CSS3 Lesson25 Resize ] Importance & Benefits:

=> [CSS3 Lesson25 Resize ] : This Advanced property enabling the following features & imporatance :

  • Providing the ability to resie the Element , which was not aviable in CSS2
  • The Abiltiy to Reset/Disable Resize property in certain Elements which is assigned by Default

[ CSS3 Resize] Syntax & Methods:

// *) [CSS3 Resize] Syntax as following :

//a) [HTML] file :
< textarea class='Resize-None' >
This is a default text area
< /textarea >

// ------------------------------------------------------------------------------

< div class='Resize-Hor' >
This div is able to be resized Horizontally
< /div >


// ------------------------------------------------------------------------------

< div class='Resize-Ver' >
This div is able to be resized Vertically
< /div >


// ------------------------------------------------------------------------------

< div class='Resize-Both' >
This div is able to be resized Both Directions
< /div >


// ------------------------------------------------------------------------------

//b) [CSS3] file :
// A) {Resize-None} Syntaxes:

body {
box-sizing: border-box ;
}
