[3D Transform - Rotate X, Y, Z()] Concept:

=> [3D Transform - Rotate X, Y, Z()] : This is one of CSS3 advancd Properties to control the Rotating Element by assinging all 3 dimension of with X , Y , Z .

=> [3D Transform - Rotate X, Y, Z()] => It is containing following {3} aspects/values/sub propeties [one each Time / two each / All] .

  • [X] => [ rotateX : Horizontal Rotating value => (+/-): Rotating over the Horizontal Axie (X Up to down or down to Up) ]
  • [Y] => [ rotateY : Vertical Rotating value => (+/-): Rotating over the Vertical Axie (Y Up to down or down to Up) ]
  • [Z] => [ rotateZ : 2D Rotating value => (+/-): Rotating over the the Third Axie (Y over clock / over Vise Clock ): Same to the 2D rotating Effect ]

=> {3D Rotating proepty can be assing by one of the rotating value or more at once }

=> {3D Rotating proepty {X,Y,Z} there are no (-360: by negative value) degree can be assing 3D-rotating Function value or more at once }

=> [3D Transform - Rotate X, Y, Z() ] : This Advanced property enabling the following features :

  • This Matrix property's will gives great effect [sensivity] when using with transition property
  • This Matrix property's will gives great effect [great animation effects ] when using with animation property
  • Using the [3D property() will gives a co9ntrol of all Dimensions of rotating at the same time , so it is better than the ordinary 2D Rotating
  • 3D Rotating property will provide unique effects specilaly when merged with other 3D Transform properties animations effects {which will be studied will later lessons}
  • This Property is will be great in Designing nice Templates specilaly when using with (Animation Property- will be taken in future)

[3D Transform - Rotate X, Y, Z()] General Practical Applications :

=> Following is the Applied example of [3D-Transform: rotateX,Y,Z(+/-ValueDeg)] syntax : >>

  • (3) Fieldsets contain (2) section (each section is by Certain by animation Methods) for each as following:
    1. First Fieldset [rotateX()] :
      • First Section => +Value Degree {Horizontal Rotating Upper - Lower}
      • Second Section => -Value Degree {Horizontal 3D Rotating Lower - Upper}

      • Second Section => -Value Degree {2D- Rotating left - right}
