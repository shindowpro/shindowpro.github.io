[CSS3 Lesson27 Flex Direction ¦ Flex Wrap] Concept:

These properties are allocated to the parent Container Element.

This property is requiring from (4-5) lessons to covered

This property is seperated to main (2) sectors :

  • Parent => There are Flex properties specialized for parent Container such as {flex-direction} || {flex-wrap}
  • Children => There are Flex properties specialized for Children Elements inside the Parent Container {will be taken in later lessons}

This property is required {display: flex} first to be activated to the parent Element .

  1. {CSS3 Flex Direction } :

    2. It is the property to make children element sorted in order flexilbe (according to certain mode & direction)

    This property is providing the Abilty to assign a certain direction of Flexing-sorting Children Elements)

    This property has the following Values :

    • [flex-direcion: row] => To sort Children Elements in row
    • [flex-direcion: row-reverse] => To sort Children Elements in row, but in opposite directions of the page direction (rtl- ltr)
    • [flex-direcion: coloumn] => To sort Children Elements in coloumn
    • [flex-direcion: coloumn-reverse] => To sort Children Elements in coloumn, but in opposite directions of the page direction (rtl- ltr)
  2. {CSS3 Flex Wrap } :

    3. It is the property to control the wraping property inside the Flexable parent container

    It is the property to make children element sorted in order flexilbe (according to certain mode & direction)

    This property has the following Values :

    • [flex-wrap: nowrap] => This is the default value of this property, and it will make no wraping childre Wraping inside the parent container
    • [flex-: wrap] => This will make the Children Elements wraping inside the parent container (according to the flex-direction & page direction
    • [flex-wrap: wrap-reverse] => This will make the Children Elements wraping inside the parent container (Oppositally to the flex-direction & page direction
