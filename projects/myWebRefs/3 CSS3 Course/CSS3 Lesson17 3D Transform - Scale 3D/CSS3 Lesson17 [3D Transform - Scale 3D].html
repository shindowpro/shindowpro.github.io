[3D Transform - Scale3D()] Concept:

=> [3D Transform - Scale3D()] : This is one of CSS3 advancd Properties to control the Scaling Element by assinging all 3 dimension of with X , Y , Z .

=> [3D Transform - Scale3D()] => It is containing following {3} aspects/values/sub propeties [one each Time / two each / All] .

  • [X] => [ ScaleX : Horizontal Scaling value => (+/-): Scaling over the Horizontal Axie ( ) ]
  • [Y] => [ ScaleY : Vertical Scaling value => (+/-): Scaling over the Vertical Axie ( ) ]
  • [Z] => [ ScaleZ : 3D (stretching ) Scaling value => (+/-): Scaling over the the Third Axie ( ) ]

=> {3D Scaling proepty can be assing by one of the Scaling value or more at once }

=> {3D Scaling property {X,Y,Z} for properly activation => needs following cosiders }

  1. Parent Element Container : Perspective property
  2. Another Transform property {Rotatng , Transition}

[3D Transform - Scale3D() ] Importance & Benefits:

=> [3D Transform - Scale3D() ] : This Advanced property enabling the following features :

  • This Scale3D property's will gives great effect [sensivity] when using with another Transforming properties
  • This Scale3D property's will gives great effect [great animation effects- stretching Effect ] when using with animation property
  • Using the [3D property() will gives a control of all Dimensions of Scaling at the same time , so it is better than the ordinary 2D Scaling
  • 3D Scaling property will provide unique effects specilaly when merged with other 3D Transform properties animations effects {which will be studied will later lessons}
  • This Property is will be great in Designing nice Animated Templates specilaly when using with ( Other 2d Properties - Animation Property- will be taken in future)

// A) [3D - Scale3D()] with Rotating Effect {+} => [One X-dimension only] Syntaxes:
// A1) [ ScaleX(+Value] : {Example1} / [Scaling over Horizontal (X) by 1.5 times + Rotating 60deg ]=> Syntax:
.container {
margin: 40px auto;
border: 1px solid #CCC;
dispaly: block;
Perspective: 600px ;
}

margin: 40px;
padding: 3px;
width: 400px;
border: 1px solid #CCC;

img:hover {

[3D Transform - Scale3D()] General Practical Applications :

=> Following is the Applied example of [3D-Transform: ScaleX,Y,Z(+/-Value px)] syntax : >>

  • (3) Fieldsets contain (2-6) section (each section is by Certain by animation Methods) for each as following:
    1. First Fieldset [Scale3D(+/-X , 0 , 0)] + [Rotating +60 deg] :
      • First Section => +Value px {Horizontal Scaling [+Width] }
      • Second Section => -Value px {Horizontal Scaling [-Width]}
    2. Second Fieldset [Scale3D(0, +/-Y , 0)] + [Rotating +60 deg] :
      • First Section => +Value px {Vertical Scaling [+Height] }
      • Second Section => -Value px {Vertical Scaling [-Height]}
    3. Third Fieldset [Scale3D(0 , 0 , +/-Z)] + [Rotating +60 deg] :
      • First Section => +Value px {3D- Scaling - Stretching in }
      • Second Section => -Value Degree {3D- Scaling - Stretching out }
    4. Fourth Fieldset [Scale3D(+/-X , +/-Y , +/-Z )] + [Rotating +60 deg] :
      • First Section => 3D-Scaling {All + Values}
      • Second Section => 3D-Scaling { All - Values}
      • Third Section => 3D- Scaling { X+, Y+, Z-}
      • Fourth Section => 3D- Scaling { X+, Y-, Z-}
      • Fifth Section => 3D- Scaling { X-, Y-, Z+}
      • Sixth Section => 3D- Scaling { X-, Y+, Z+}

3D-Transform: Scale3D(+/-X , 0 , 0) => First Mode {Scaling by only X dimenison Mode} + [Rotating: 60 deg ]

Following sections will be X-Scaled property by [positive value Degree] + [Rotating: 60 deg ]

This Section will be {Scaled} into Horizontal dimension {Width} [{Scale:1.5X, 0Y , 0Z} + rotate(60deg)]


This Section will be {Scaled} into Horizontal dimension {Width} by [{Scale:1.7X, 0Y , 0Z} + rotate(-60deg)]




Following sections will be X-Scaled property by [Negative value Degree] + [Rotating: 60 deg ]

This Section will be {Scaled} into Horizontal dimension { -Width } by [{Scale:-1.5X, 0Y , 0Z} + rotate(60deg)]


This Section will be {Scaled} into Horizontal dimension { -Width } by [{Scale:-1.7X, 0Y , 0Z} + rotate(-60deg)]


3D-Transform: Scale3D(0 , +/-Y , 0 ) => Second Mode {Scaling by only Y dimenison Mode} + [Rotating: 60 deg ]

Following sections will be Y-Scaled property by [positive value Degree] + [Rotating: 60 deg ]

This Section will be {Scaled} into Veritcal dimension { +Height } by [{Scale:0X, 1.5Y , 0Z} + rotate(60deg)]


This Section will be {Scaled} into Veritcal dimension { +Height } by [{Scale:0X, 1.7Y , 0Z} + rotate(-60deg)]




Following sections will be Y-Scaled property by [Negative value Degree] + [Rotating: 60 deg ]

This Section will be {Scaled} into Veritcal dimension { -Height } by [{Scale:0X, -1.5Y , 0Z} + rotate(60deg)]


This Section will be {Scaled} into Veritcal dimension { -Height } by [{Scale:0X, -1.7Y , 0Z} + rotate(-60deg)]


3D-Transform: Scale3D(0 , 0 , +/-Z ) => Third Mode {Scaling by only Z dimenison Mode} + [Rotating: 60 deg ]

Following sections will be Z-Scaled 3D property by [positive value Degree]

This Section will be {Scaled 3D} into Stretching dimension {Stretching In} by [{Scale:0X, 0Y , 1.5Z} + rotate(60deg)]


This Section will be {Scaled 3D} into Stretching dimension { Stretching In } by [{Scale:0X, 0Y , 1.7Z} + rotate(-60deg)]




Following sections will be Z-Scaled property by [Negative value Degree]

This Section will be {Scaled 3D} into Stretching dimension {Stretching out} by [{Scale:0X, 0Y , -1.5Z} + rotate(60deg)]


This Section will be {Scaled 3D} into Stretching dimension {Stretching Out} by [{Scale:0X, 0Y , -1.7Z} + rotate(-60deg)]


3D-Transform: Scale3D(+/-X , +/-Y , +/-Z ) => Fourth Mode {Scaling by only 3D dimenison Mode}

Following sections will be Scaled - 3d property by [all dimensions values +]

This Section will be {Scaled 3d} into [+Width +height +Stretching in] dimension [{Scale:1.5X, 1.5Y , 1.5Z} + rotate(60deg)]


Following sections will be Scaled - 3d property by [all dimensions values -]

This Section will be {Scaled 3D} into [-Width -height -Stretching out] dimension by [{Scale:-1.5X, -1.5Y , -1.5Z} + rotate(60deg)]



Following sections will be Scaled - 3d property by [+X, -Y, +Z]

This Section will be {Scaled 3D} into [+Width -height +Stretching in] by dimension [{Scale:+1.5X, -1.5Y , +1.5Z} + rotate(60deg)]




Following sections will be Scaled - 3d property by [-X, +Y, -Z]

This Section will be {Scaled 3D} into [-Width +height -Stretching Out] by dimension [{Scale:-1.5X, +1.5Y , -1.5Z} + rotate(60deg)]




Following sections will be Scaled - 3d property by [+X, +Y, -Z]

This Section will be {Scaled 3D} into [+Width +height -Stretching out] by dimension [{Scale:+1.5X, +1.5Y , -1.5Z} + rotate(60deg)]




Following sections will be Scaled - 3d property by [-X, -Y, +Z]

This Section will be {Scaled 3D} into [-Width -height +Stretching in] by dimension [{Scale:-1.5X, -1.5Y , +1.5Z} + rotate(60deg)]


Following sections will be Scaled - 3d property by [+X, -Y, -Z]
