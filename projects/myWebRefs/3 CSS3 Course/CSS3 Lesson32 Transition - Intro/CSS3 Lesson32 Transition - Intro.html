[CSS3 Transition - Intro ] Concept:

{Transition - Intro}: This property is very important one and the base for CSS3 animation

{Transition - Intro}: This property is about setting how the element Transate within Changing Event

{Transition - Intro}: This property default Function (Syntax) components from following elements (in-order) :

  1. First: {Property Name} => This is the Property of the Element which the Transition will be applied on it [Will be studied in details later]
  2. Second: {Duration} => This is Transition Duration Time , which the Transtion will be appllied in [Will be studied in details later]
  3. Third: {Timing Function } => This is one of Transition Function which will be studied later
  4. Fourth: {Delay} => This is one of Transition Function which will be studied later

[CSS3 Lesson 32 Transition - Intro ] Importance & Benefits:

=> [CSS3 Lesson 32 Transition - Intro ] : This Advanced property enabling the following features & imporatance :

  • Transition is the soul of Animation in CSS3
  • Gives a great specific Tool to control the Timing and kind of property to apply the transition on
  • Using {Transition ShortHand} => Is just the Gate to in-detail Transition Function (which will be studied in later lessons)

[ CSS3 Transition - Intro] Syntax & Methods:

// (*) [CSS3 Transition - Intro] Syntax as following :

//a) [HTML] file :
< section class='Transition' >
< div class='Width-Change' > This element's Width will be changed with Transition (when hover) < /div >
< div class='Color-Change' > This element's Background-Color will be changed with Transition (when hover) < /div >
< div class='Width-Color-Change' > This element's Width + Background-Color will be changed with Transition (when hover) < /div >
< /section >

//b) [CSS3] file :
.Transition {
margin: 50px auto ;
position: relative ;
width: 100%;
height: 400px;
border: 3px solid #EEE;
border-radius: 20px ;
}

.Transition div {
position: absolute ;
left: 10px ;
top: 10px ;
margin: 50px auto ;
width: 200px;
height: 200px;
background-color: blue;
border: 1px solid black;
border-radius: 20px ;
}

.Transition .Width-Change {
transition: width 1.5s;
}

.Transition .Width-Change:hover {
width: 500px;
}

.Transition .Color-Change {
transition: background-color 2s;
}

.Transition .Color-Change:hover {
background-color: red;
}

.Transition .Width-Color-Change {
transition: background-color 2s, width 1.5s;
}

.Transition .Width-Color-Change:hover {
background-color: red;
width: 500px;
