Selectors - [Nth-Child || Nth-Of-Type] {Part 2} Main Topics :

> Main Subjects in this lesson are as following :

Selectors with different types following:

  1. [Nth-Child()] Selector => [:Nth-Child(Furmula)]
  2. [Nth-Of-Type()] Selector => [:Nth-Of-Type(Furmula)]
  3. [Nth-Last-Child()] Selector => [:Nth-Last-Child(Furmula)]
  4. [Nth-Last-Of-Type()] Selector => [:Nth-Of-Type(Furmula)]
  5. [(Formula)] => diffrent types :
    1. (n+Number) => {Select from a certain element number -> The end of Children || Types }
    2. (-n+Number) => {Select from Start of Parent container's Children || Types -> certain element number }
    3. [nth-child(Number X n + Number){}] => Select customized Adding Sequance accroding to customized Function/Formula from Start Parent Container to a certain Child number
    4. (n+Number):(-n+Number) => {Select from Certain element number (n) -> Certain element number (n) }
Selectors - Nth-Child(F) || Nth-Of-Type(F) ] {Part 2} Main Topics :

{ These Selector => These type of selectors [nth] are commonly to use in accessing to Elements & addional features


{Selector[Nth(formula)] - group ] => Has some additional Features in comparison to the types of Selectors, such as :

  • Can be used to access According to Child Index Or Types Index , by adding {Customized Function} || {Adding Sequace} || {Certain Range}, as following :
    • [nth-child(F)] :
      1. [nth-child(n+Number){}] => Select Range from certain Child number until to end of container Children
      2. [nth-child(-n+Number){}] => Select Range from Start Parent Container to a certain Child number
      3. [nth-child(Number X n + Number){}] => Select customized Adding Sequance accroding to customized Function/Formula from Start Parent Container to a certain Child number
      4. [nth-Of-type(n+Number):nth-child(-n+Number){}] => Select Range from certain type index to a certain type index
    • [nth-Of-type(F) ] :
      1. [nth-Of-type(n+Number){}] => Select Range from certain type index until to end of container end of this Type
      2. [nth-Of-type(-n+Number){}] => Select Range from Start Parent Container of certain type to a certain type index
      3. [nth-Of-type(Number X n + Number){}] => Select customized Adding Sequance accroding to customized Function/Formula from Start Parent Container to a certain Child number
      4. [nth-Of-type(n+Number):nth-child(-n+Number){}] => Select Range from certain type index to a certain type index
    • [nth-last-child(F)] :
      1. [nth-last-child(n+Number){}] => Select Range from certain Child number until to end of container Children {starting counting reversely}
      2. [nth-last-child(-n+Number){}] => Select Range from Start Parent Container to a certain Child number {starting counting reversely}
      3. [nth-last-child(Number X n + Number){}] => Select customized Adding Sequance accroding to customized Function/Formula from Start Parent Container to a certain Child number
      4. [nth-last-child(n+Number):nth-child(-n+Number){}] => Select Range from Certain Child number to a Certain Child number {starting counting reversely}
    • [nth-last-Of-type(F) ] :
      1. [nth-last-Of-type(n+Number){}] => Select Range from certain type index until to end of container end of this Type {starting counting reversely}
      2. [nth-last-Of-type(-n+Number){}] => Select Range from Start Parent Container of certain type to a certain type index {starting counting reversely}
      3. [nth-last-Of-type(Number X n + Number){}] => Select customized Adding Sequance accroding to customized Function/Formula from Start Parent Container to a certain Child number
      4. [nth-last-Of-type(n+Number):nth-child(-n+Number){}] => Select Range from certain type index to a certain type index {starting counting reversely}

[Nth-Child(F) || Nth-Of-Type(F) ] {Part 2} Importance & Benefits:

=> [ Nth-Child || Nth-Of-Type ] : This Advanced Selectors providing the following features & imporatance :

  • Using advanced functions / Formulas to access to certain or multiple elements
  • Can use Customized functions or built-in functions
  • advnanced Formulas can be used with reversely selectors (Last)
