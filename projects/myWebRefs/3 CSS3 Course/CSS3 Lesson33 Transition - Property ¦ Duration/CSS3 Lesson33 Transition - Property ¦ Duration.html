[CSS3 Transition - Property ¦ Duration ] Concept:

{Transition - Property ¦ Duration}: These properties are the first & second parts of Transition Function

{Transition - Property ¦ Duration}: These Two Basic properties are Must to be activate Transition Function

{Transition - Property ¦ Duration}: These properties in details (Syntax) as following :

  1. First: {transition-Timing function} => This is the Property Name of the Element which the transition will be applied on
    • {transition-Timing function} => Default Value is [all] : which means to set transition to all Element's properties.
    • {transition-Timing function} => Can be set by one single property name OR Multiple properties names (Seperated by ,) in order
    • {transition-Timing function} => Disabling Code is -> [none] : which means to Cancel transition for this property Name
  2. Second: {transition-Delay} => This is Transition Duration Time that will be spent with in Transated element
    • {transition-Delay} => Default Value is [0s] : which means to transate the Element's property(s) instantely.
    • {transition-Delay} => Can be set by one single duration time OR Multiple duration times (Seperated by ,) in order
    • {transition-Delay} => Can be set by both (s -> Second || ms-> MilliSecond)

[CSS3 Lesson 33 Transition - Property ¦ Duration ] Importance & Benefits:

=> [CSS3 Lesson 33 Transition - Property ¦ Duration ] : This Advanced property enabling the following features & imporatance :

  • [transition-Timing function] & [transition-Delay] => provide much more accurate-specific controling tools in Transition
  • [transition-Timing function] & [transition-Delay] => Gives a great specific Tool to reach certain element's property with in transition
  • [transition-Delay] => provides controling duration by both (second) & (MilliSecond) for more Advanced tool
