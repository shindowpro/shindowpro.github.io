[CSS3 Multi Columns [Column-Span, Shorthand] ] Concept:

{Multi Columns => [Column-Span] & [Columns] }: These properties and all of it's proerties need to add web prefixes to work correctly

{Multi Columns [Column-Span, Shorthand]}: Is providing the best solution for separeted desiging Templates (Textual) as Columns by simple way with out further old CSS perperties

{Multi Columns [Column-Span, Shorthand]}: This property is recommended for [Texutal content] only, and not for other elements types

{Multi Columns [Column-Span, Shorthand]}: These (2) new properites of the Multi Columns is the {part3} as following :

  1. First property [Column-Span]: This new feature of the Multi-Column is to control the overcrossing of heading element over the Columns

    The Default value is => {1}: Which is meaning to overcross the Inner Heading over first Column only

    The value is => {all}: Which is meaning to overcross the Inner Heading over all Columns

  2. Second property => {Columns} : Which is the ShortHand of most important propeties of the Multi-Coloums

    The Default value is => {auto auto}: Which is meaning to Make the webbrowser Take the auto Arranging Multi-Coloums properties

    The value is => {auto auto}: Prefering to each {Column-Width || Columns-count}

[CSS3 / Multi Columns [Column-Span, Shorthand] ] Importance & Benefits:

=> [CSS3 / Multi Columns [Column-Span, Shorthand] ] : This Advanced property enabling the following features & imporatance :

  • [column-span] : Give a great Tool to control Heading OverCrossing over Multi-Columns
  • [Columns || -webkit-Columns] : Providing useful & shortHand method to contorl most important properites of the Multi-Coloums
