[Border Radius] Concept:

=> [Border Radius] : It is a property of desiging Rounding Shapes of the Elements or Boxes with cotroling the Percentage of Curvies.

=> {(One side) Border Radius}: It has (4) values of each Side of the Element of Shape, as following:

  • border-top-left-radius => Which is the first Value
  • border-top-right-radius => Which is the second Value
  • border-bottom-right-radius => Which is the third Value
  • border-bottom-left-radius => Which is the fourth Value

=> {(Dual Side) Border Radius}: It has (2) values of each Side of the Element of Shape, as following:

  • border-top-radius => Which is the first Value
  • border-bottom-radius => Which is the first Value

Border Radius [Border Radius ] Importance & Benefits:

[Border Radius] gives us the Following Features :

  • It provides design certian Curvy shapes such as (Circles, Cylinders)
  • It provide high control of percentage of Curvnessj of each side.
  • This Property is now Supportes by all types of kinds of Browsers [new versions] .

[ Border Radius ] Syntax & Methods:

// *) [Border-radius] property Syntax as following :

//1) [Border-Radius] Syntax {without Vendor Prefix}
a) ( 1 Value for All sides ):
border-radius : #Value px;
// ------------------------------------------------------------------------------

b) ( 2 Values for All sides ):
border-radius : #Value1 px #Value2 px ;
// value1 => {for each (top-left , bottom-right) sides}
// value2 => {for each (top-right, bottom-left) sides}
// ------------------------------------------------------------------------------

c) ( 3 Values for All sides ):
border-radius : #Value1 px #Value2 px #Value3 px ;
// value1 => {for (top-left) side}
// value2 => {for (top-right , bottom-right) following sides}
// value3 => {for each (bottom-left) last side}
// ------------------------------------------------------------------------------

d) ( 4 Values for All sides ):
border-radius : #Value1 px #Value2 px #Value3 px #Value4 px;
// value1 => {for (top-left) side}
// value2 => {for (top-right) side}
// value3 => {for (bottom-right) side}
// value4 => {for (bottom-left) side}
//===========================================================
