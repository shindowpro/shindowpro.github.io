[Overflow New Features] Concept:

  • A) {Overflow New Features} main Concept :

    • In CSS2 there was only the ability to control the Item overflow property as over all one property

    In CSS2 default Properites as following :


    • [Visible] => To make the overflow content visible as overflowed
    • [Hidden] => To make the overflow content Hidden as Cutted
    • [Scroll] => To make the overflow content visible with Scroll Bar

    But in CSS3 there are much specification by controling Horzontal & Vertical Overflow directions as following:

    1. {overflow-X}:

      2. Default value of {overflow-X} is : [visible]

      This new feature to control {overflow in Horizontal} with all values in old CSS2 [visible, hidden, scroll] as following :

    2. {overflow-Y}:

      3. Default value of {overflow-Y} is : [visible]

      This new feature to control {overflow in Vertical} with all values in old CSS2 [visible, hidden, scroll]

[Overflow New Features ] Importance & Benefits:

=> [Overflow New Features ] : This Advanced property enabling the following features & imporatance :

  • Providing much more accurate tool to control element overflow property in certain direction
