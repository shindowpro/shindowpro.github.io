CSS3 Lesson 49 Selectors - Next, Start With, End With, Contains Property Main Topics :

> Main Subjects in this lesson are as following :

Selectors with different types following:

  1. [Next] Selector => [~]
  2. [Start with] Selector => [^]
  3. [End with] Selector => [$]
  4. [contains] Selector => [*]

[CSS3 Selectors - Next, Start With, End With, Contains ] Concept:

{Selector} Basic concept => It is a new feature in CSS3 to access to certian accurate element(s) with out using any of [J-Qeurry || JS || Bootstrap]

{Selector- [~] {Next}} =>

  • It is about Selecting the Next of the current element [in the same Parent]
  • [Element/Tag Name ~ Element/Tag Name(desired Element/Tag) ]

{Selector- [^] {Start with}} =>

  • {Selector- [^] {Start with}} => It is about Selecting the element that has an Attribute start with certain value
  • [Attribute Name^='Start with Value' ]

{Selector- [$] {End with}} =>

  • {Selector- [$] {End with}} => It is about Selecting the element that has an Attribute end with certain value
  • [Attribute Name$='End with Value' ]

{Selector- [*] {Contain} } =>

    {Selector- [*] {contain}} => It is about Selecting the any element contain all of assigned value [not nessesry to be in order]

  • [Attribute Name*='containing Value' ]
