[Word Wrap, White Space] Concept:

=> [Text-Overflow] : This Advanced Tool to control Breaking the Text in it's parent Container.

=> Differnces between [word-wrap] & [white-space] :

  • [white-space] => To control the Wrapping property of the Text with mainly following options :

    • {white-space: normal} => The Normal Value of this Property, which is meaning to Allow text Wrapping {will not break if the Text is one word}
    • {white-space: nowrap} => To prevent text wrapping
  • [word-wrap] => More advanced tool To Control the Breaking Text over it's container by following properties:

    • {word-wrap: normal} : It is the Default Value of text breaking => which will normally [NOT] break the text
    • {word-wrap: break-word} : Will break the Text (even it was one Word) to be contained in it's parent Container.

[Word Wrap, White Space ] Importance & Benefits:

=> [Word Wrap] : This Advanced applying will give us advanced & unique & assurd Breaking Features for the Text ) with following features :

  • Assured Text breaking even the Text is only one Word (text with out spacing)
  • It is more better than [white-spacing] breaking Tool (due to it will not break if the text is one Word)

[ Word Wrap, White Space ] Syntax & Methods:

// *) [Word Wrap, White Space] Syntax as following :

//1) [ Not breaking text property using both of {Word Wrap} , {White Space}] Syntax:
//a) [HTML] file :
< div > This is A normal Div with long sentence text with will be NOT breaken using [word-wrap:normal + white-space:nowrap]
</div>
// ------------------------------------------------------------------------------

// {Exmple 1}/a: [CSS3] file [Disable breaking Property by {word-wrap:normal}] :
div {
margin: 40px;
padding: 3px;
width: 400px;
border: 1px solid #CCC;
backgournd-color: #FFF;
word-wrap: normal;
}
// ------------------------------------------------------------------------------

// {Exmple 1}/b: [Enabling text breaking property when (hovering) by {word-wrap: break-word} property] Syntax:
div: hover {
word-wrap: break-word;
}
