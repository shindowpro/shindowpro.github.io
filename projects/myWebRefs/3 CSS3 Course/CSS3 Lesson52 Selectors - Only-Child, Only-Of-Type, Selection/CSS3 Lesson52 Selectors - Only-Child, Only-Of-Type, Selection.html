CSS3 Lesson52 Selectors - [Only-Child || Only-Of-Type || Selection ] Main Topics :

{ These Selector => is rarely to use in the CSS3 , but we might need it some time ] =>


{Selector- [:Only-Child()] =>

  • This selector to select a Certain Child that is existed alone in it's Parent container

{Selector- [:Only-Of-Type()] =>

  • This selector to select a Certain Type Child that is one of it's type existed in it's Parent container

{Selector- [::Selection] =>

  • This selector to control the text Selection in the browser [color , backgrond-color, text-shadow ]
  • This selector will not respond to non textual properties [margin, pading ]

[Only-Child, Only-Of-Type, Selection] Importance & Benefits:

=> [ Only-Child, Only-Of-Type, Selection] : This Advanced Selectors providing the following features & imporatance :

  • It can access only a certain element(s) in some cases
  • It is rarely to use but might be useful in some cases
  • [::Selection] => It is the only property to control the browser Selection
