[Box Sizing] Concept:

=> [Box Sizing] : It is one of most important properties of desiging Elements in CSS3, due to it's Critical Features.

=> Main Properties of {Box Sizing} as following:

  • Only Google Chorme & Modziela Web Browsers are Supporting , and it not supportedi by each Modzella and opera
  • This Property has Solven Many Deigning problems specaily in Percetntage Equilibrium between Childen Elemenets and their Parent Element
  • This Property has (2) main Modes [contain-box] {Default Mode} & [border-box]

=> {box-sizing} Main Concept: It is a unique tool to control the width sizing of each Chiledren elements and their Percentage between there and their Parent Width

=> {box-sizing : contain-box} [Default Mode]: The Most properties of this mode as following :

  • It is the Default mode of {box-sizing} of Controling the Percentage of Children Elements & Their Parent/container.
  • this Mode is counting on the total dimensions of the intenal children Elements's Wiidth of any kind (Width, height, padding, border), and that is way it doest Not do Nothing to Percenatge adjusting.
  • This Mode does not Solve any problems practically.

=> {box-sizing : border-box} : The Most properties of this mode as following :

  • It is the Magic mode of Controling the Percentage of Children Elements & Their Parent/container, and this Mode Solves Manu problems in percentage Desinging practically , due it is Cancel any considerations of addiontional Children Elements width like (padding , border , width) .
  • this Mode is counting on the overall Parent's dimension regadless of any of intenal children Elements's Width of any kind (Width, height, padding, border) .
  • This Mode has Solven many problems practically, specially in Percenatge adjusting.

Box Sizing [Box Sizing ] Importance & Benefits:

[Box Sizing] gives us the Following Features :

  • It provides designing beautiful percentage Harmony between Parent Element's Dimension and it's Childeren Elements 's Dimensions
  • It provide higher control of Element's Sizing via (2) Modes [border-box] [content-box]
  • New And Advanced Desings is using this property as basically in their Preifxex {* {} / *.:before{} / *.:after{} }

[ Box Sizing ] Syntax & Methods:

// *) [Box-sizing] Syntax as following :

//1) {Default Mode} [content-box] Mode Syntax:
//a) [content-box] Mode with available Vendor prefixes :
-webkit-box-sizing: content-box
-moz-box-sizing: content-box
// ------------------------------------------------------------------------------

//b) [content-box] Mode as all of Page Contents Properties :
*{
box-sizing: content-box;
}
// ------------------------------------------------------------------------------

//c) [content-box] Mode as all of certain Parent Element property:
.parentElementClassName {
box-sizing: content-box;
}
// ------------------------------------------------------------------------------

// ------------------------------------------------------------------------------

//2) [border-box] Mode Syntax:
//a) [border-box] Mode with available Vendor prefixes :
-webkit-box-sizing: border-box
-moz-box-sizing: border-box
// ------------------------------------------------------------------------------

//b) [content-box] Mode as all of Page Contents Properties :
*{
box-sizing: border-box;
}
// ------------------------------------------------------------------------------

//c) [content-box] Mode as all of certain Parent Element property:
.parentElementClassName {
box-sizing: border-box;
}
// ------------------------------------------------------------------------------
