[2D Transform - Scale()] Concept:

=> [2D Transform - Scale()] : This is one of CSS3 Properties to control the Element Scale Amount (by each width And Height, or one of them only) .

=> [2D Transform - Scale()] : The Default value of this property (and all Transform) is (none) .

=> [2D Transform - Scale()] : This is one of CSS3 {2D Transform} Properties Functions, which has the same in 3D Transform .

=> There are (2) Methods to write this property as following :

  • [transform: scale(width value,height Value) ] => This Can Hold two values of each dimenstions both , or one Value for Each Dimensions at once

  • [transform: scaleX(Width Value)] ,
    [transform: scaleY(Height Value)]
    => This method to control one Dimension only

=> This property 's Value is about amount of Duplicating of Original's Element Dimensions as following examples :

  • [none] => Reset this Element's Property to none {Default Value}
  • [1] => keep the same dimensions as it , with out changes
  • [.5] => Change the Current Dimensions by half
  • [2] => Change the Current Dimensions by Double
  • [3] => Change the Current Dimensions by Trible

[2D Transform - Scale] General Practical Applications :

=> Following is the Applied example of [2D-Transform: Scale(Value)] syntax : >>

  • (2) Fieldsets contain (3) section (each section is Scaled by Certain defferent (2) Methods) for each as following:
    1. First Fieldset [Fixed Mode] :
      • First Section => Scaled by {First Method (2 values for Each dimension)}
      • Second Section => Scaled by {First Method (1 value for both dimensions)}
      • Third Section => Scaled by {Second Method (2 values for Each dimension)}
      • Fourth Section => Scaled by {First Method (1 value for both dimensions)}
    2. Second Fieldset [Dynamic Mode] :
      • First Section => Scaled by {First Method (2 values for Each dimension)}
      • Second Section => Scaled by {First Method (1 value for both dimensions)}
      • Third Section => Scaled by {Second Method (2 values for Each dimension)}
      • Fourth Section => Scaled by {First Method (1 value for both dimensions)}

Scale() => First Mode {Fixed Mode}

This Section will be Scaled by First Method (1.5 values for Each dimension)

This Section will be Scaled by First Method (1.5 value for both dimensions)

This Section will be Scaled by Second Method (1.5 values for Each dimension)

Scale() => Second Mode {Dynamic Mode}

This Section will be Scaled by First Method (1.5 values for Each dimension)

This Section will be Scaled by First Method (1.5 value for both dimensions)

This Section will be Scaled by Second Method (1.5 values for Each dimension)
