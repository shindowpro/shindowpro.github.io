[2D Transform - Translate()] Concept:

=> [2D Transform - Translate()] : This is one of CSS3 Properties to control the Element Translate Amount (by each width And Height, or one of them only) .

=> [2D Transform - Translate()] : The Default value of this property (and all Transform) is (none) .

=> [2D Transform - Translate()] : This is one of CSS3 {2D Transform} Properties Functions, which has the same in 3D Transform .

=> There are (2) Methods to write this property as following :

  • [transform: Translate(width value,height Value) ] => This Can Hold two values of each dimenstions both , or one Value for Each Dimensions at once

  • [transform: TranslateX(Width Value)] ,
    [transform: TranslateY(Height Value)]
    => This method to control one Dimension only

=> This property 's Value is about amount of Duplicating of Original's Element Dimensions as following examples :

  • [none] => Reset this Element's Property to none {Default Value}
  • [1] => keep the same dimensions as it , with out changes
  • [.5] => Change the Current Dimensions by half
  • [2] => Change the Current Dimensions by Double
  • [3] => Change the Current Dimensions by Trible

[2D Transform - Translate() ] Importance & Benefits:

[2D Transform - Translate] General Practical Applications :

=> Following is the Applied example of [2D-Transform: Translate(Value)] syntax : >>

  • (2) Fieldsets contain (3) section (each section is Translated by Certain defferent (2) Methods) for each as following:
    1. First Fieldset [Fixed Mode] :
      • First Section => Translated by {First Method (2 values for Each dimension)}
      • Second Section => Translated by {First Method (1 value for both dimensions)}
      • Third Section => Translated by {Second Method (2 values for Each dimension)}
      • Fourth Section => Translated by {First Method (1 value for both dimensions)}
    2. Second Fieldset [Dynamic Mode] :
      • First Section => Translated by {First Method (2 values for Each dimension)}
      • Second Section => Translated by {First Method (1 value for both dimensions)}
      • Third Section => Translated by {Second Method (2 values for Each dimension)}
      • Fourth Section => Translated by {First Method (1 value for both dimensions)}

Translate() => First Mode {Fixed Mode}

Move to [Right] by 20px {by 2 values}

This Section will be Translated => {Right:20px} by [First Method : two Values (x = 20, y = 0)] with in First Method


Move to [Left] by 20px {by 2 values}

This Section will be Translated => {Left:20px} by [First Method : two Values (x = -20, y = 0)] with in First Method


Move to [Top] by 20px {by 2 values}

This Section will be Translated => {Top:20px} by [First Method : two Values (x = 0, y = -20)] with in First Method


Move to [Bottom] by 20px {by 2 values}

This Section will be Translated => {Bottom:20px} by [First Method : two Values (x = 0, y = 20)] with in First Method


Move to [Right + Top] by 20px {by 2 values}

This Section will be Translated => {Right:20px + Top:20px} by [First Method : two Values (x = 20, y = -20)] with in First Method


Move to [Right + Bottom] by 20px {by 2 values}

This Section will be Translated => {Right:20px + Bottom:20px} by [First Method : two Values (x = 20, y = 20)] with in First Method


Move to [Left + Top] by 20px {by 2 values}

This Section will be Translated => {Left:20px + Top:20px} by [First Method : two Values (x = -20, y = 20)] with in First Method


Move to [Left + Bottom] by 20px {by 2 values}

This Section will be Translated => {Left:20px + Bottom:20px} by [First Method : two Values (x = -20, y = 20)] with in First Method


Translate() => [One Value Method] => will only read one Dimension {X : Horizontal}
So There will be no second dimension {Y : Vertical}

This Section will be Translated => {Right:20px} by [Second Method : One Values (20px)] with in First Method

Translate() => Third Mode {Multiple Mode}

A) Rotation + Scaling


1) {Rotate:45 deg => Right + Scale:1.5} :

This Section will be {Rotated} by 45 deg into the Right Direction + {Scaled:1.5} times


2) {Rotate:45 deg => Left + Scale:1.5} :

This Section will be {Rotated} by 45 deg into the Left Direction +
{Scaled:1.5} times

This Section will be {Rotated} by 45 deg into the Right Direction + {Moved:20px} into the left Direction
