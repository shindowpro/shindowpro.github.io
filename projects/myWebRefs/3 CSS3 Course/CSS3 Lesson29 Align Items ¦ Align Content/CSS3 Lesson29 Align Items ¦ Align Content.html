[CSS3 Lesson29 Align-Items ¦ Align-Content] Concept:

{Align-Items ¦ Align-Content}: These properties are allocated to the parent Container (Parent container => part:3)

  • First (Align-Items) :

    [Align-Items] => This property to align the elements inside the parent container {Vertically}, with taking consideration of {child element max-Height }

    [Align-Items] => The default value is : {stretch}

    [Align-Items] => This property will make childern show as their content height {different in height}

    [Align-Items] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Child element => {max-height: #Value} To not filling the whole Container height .

    [Align-Items] => This property can be assinged wtih one following values :

    1. {Stretch} : To make children Elements Stretching thrugh it's entire height
    2. {flex-start} : To make children Elements Stretching thrugh it's entire height
    3. {flex-end} : To make children Elements Stretching thrugh it's entire height
    4. {center} : To make children Elements Stretching through it's entire height
    5. {baseline} : To make children Elements Stretching according (Items baseline)
  • Second (Align-Content) :

    [Align-Content] => Is one of the (Parent container => part:3)

    [Align-Content] => This property to align the elements inside the parent container {Vertically}, with taking consideration of {child element Height + wraping Effect }

    [Align-Content] => The default value is : {stretch}

    [Align-Content] => This property is Requiring following conditions:

    1. Parent Container => {display: flex} to be activated .
    2. Parent Container => {flex-wrap: warp} To make children element wraping to display the aligning content effect.
    3. Max children Elements Width => must less (<) than the parent container width .

    [Align-Items] => This property can be assinged wtih one following values :

    1. {Stretch} : To make children Elements Stretching thrugh it's entire height
    2. {flex-start} : To make children Elements Stretching thrugh it's entire height
    3. {flex-end} : To make children Elements Stretching thrugh it's entire height
    4. {center} : To make children Elements Stretching thrugh it's entire height
    5. {space-between} : To make Vertical equaled spaces between the children Elements inside the Parent container
    6. {space-around} : To make Vertical equaled spaces between + before + After the children Elements inside the Parent container

Differences between {Justify-Content ¦ Align-Content}:

  1. [Justify-content] => Taking in consideration Following conditions :
    • Horizontal Aligning is the main focus of this property
    • Page direction & items direction
  2. [Align-content] => Taking in consideration Vertical Aligning
    • Vertical Aligning is the main focus of this property
    • Wraping Activating property is a Must
    • Total Width of Children must be Less than Overall parent container => to make assuranace of wrap property Activation
