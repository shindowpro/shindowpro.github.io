[3D Transform - Create Flip Card] Concept:

  • The following concepts will be Applied in this Application :
    1. {Persective} => To assign 3d persective Val3ue
    2. {Transform-Style} => Determining the Transform Style (3d-preserve / flat)
    3. {Transform-Origin} => Determining the Transform Origin/Direction
    4. {Transition} => Determining Duration of Transform
    5. {backface-visibiltiy} => Deteriming the visibility of Card face (After performing Transforming )
    6. {z-index} => To assign Which face of card will be the Front & Which will be the Back
