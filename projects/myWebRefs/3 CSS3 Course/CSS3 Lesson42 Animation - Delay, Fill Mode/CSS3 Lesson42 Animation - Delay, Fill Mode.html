[CSS3 Animation - [Delay, Fill Mode] ] Concept:

{Animation- [Delay || Fill-mode]} => Is Needing for using [prefixes] before the animation property and it's branshed properties

{Animation - [Delay, Fill Mode]}: These are the fifth and Sixth child porperties of Animation syntax , with following details :

  1. { -webkit-animation-delay || animation-delay} =>

    It is the delaying period before from starting the Animation

    [Default value] of Animation-delay : 0s (which is meaning to start the animation immediatly )

    Animation-delay => can be set by : {# Second} || {milli Second}

  2. { -webkit-animation-fill-mode || animation-fill-mode} =>

    Which is the Animation Filling keyframe Mode to be start with / end at

    [Default value] : none

    Can be assigned by one of following values :

    1. [none] => To make the animating element start the animation with out any of fill-mode
    2. [forwards] => To make the animating element End at the [End] keyframe of the animation function, after finishing the whole animation function
    3. [Backwards] => To make the animating element start with the [first] keyframe of the animation function, Before starting the animation function (even before the inimation-delay)
    4. [both] => To merge between both [forwards] & [backwards]

[CSS3 / Animation - [Delay || fill-mode] ] Importance & Benefits:

=> [CSS3 / Animation - [Delay || fill-mode] ] : This Advanced Animation sub properties enabling the following features & imporatance :

  • [animation-delay] : Give a ability to set a certain amount of delaying time of the Declared Animation even before
  • [animation-fill-mode] : Give a ability to assign a certain filling mode of Declared Animation for to a specific Element, to assign certain way (Step/keyfarame) to start / end the animation [can be set diffrerntly for each element]
