[CSS3 Lesson28 Flex-Flow - Justify-Content] Concept:

These properties are allocated to the parent Container Element.

These properties are (Parent container => part:2) required {display: flex} first to be activated to the parent Element .

  1. {CSS3 Flex-Flow } :

    2. It is the property is the shorthand of both (Flex-direction - Flex-wrap) in one property

    This property can accept all following Values :

    • [flex-flow: ( row || coloumn || row-reverse || coloumn-reverse ) , ( nowrap || wrap || warp-reverse ) ]
  2. {CSS3 Justify-Content} :

    3. It is the property to control Justifying property of Flex Parent Element's children inside it

    This property is allocated for the Parent Element CSS properties , but cont4oling with the parent container' Children

    This property has the following Values :

    • [Justify-Content: flex-start] => This is the default value of this property, and it will the parent container start flexing it's Children from the first one
    • [Justify-Content: flex-end] => This will make the parent container start flexing it's Children from the first one
    • [Justify-Content: center] => This will make the parent container positioning it's children in center of it's Flex-box
    • [Justify-Content: space-between] => This will make the parent container assigning between equalled spaces between children
    • [Justify-Content: space-around] => This will make the parent container assigning around equalled spaces around children [before, between , after]

[CSS3 Lesson28 FlexFlow - Justify Content ] Importance & Benefits:

=> [CSS3 Lesson28 FlexFlow - Justify Content ] : This Advanced property enabling the following features & imporatance :

  • Providing better, easier, practical, shortcut and profficent Renposive Desiging , Much more easier way than old methods
  • {Flex-Flow} => Providing shortHand method for both (Flex-Direction || Flex-Wrap) properties in one properties with all of it's values
  • {Justify-content} => Providing very easy method & tools for controling children inside the parent contaier in comparison with the old methods
