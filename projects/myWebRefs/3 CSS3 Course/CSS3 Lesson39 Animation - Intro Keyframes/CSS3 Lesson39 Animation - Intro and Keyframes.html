{Animation - Intro and Keyframes}: Is containing following components :

  1. [Animation Declaring Function]: And it contains Items below:

    [@keyframes] key word => Which is the keyword used for declaring animation function

    [from / 0%] => Which the Starting point to start animation from

    [to / 100%] => Which the Ending point to End animation at

    [0->100%] => Which is the between steps (if the developer desire) after the starting point & ending point [one between percetage for each one]

  2. [Calling Animation function - Animation using property ] => Which is the calling/using animation inside an element properites, and contains following :

    [Animation Name] : Which is the name of Declared Animation Function

    [Animation Duration (for once)] : Which is the Time period of the single Animation for (Once)

    [Animation Count] : Which is the Number of repeating the Declared Animation Function

    [Animation Type] : Which is the Display type of Declared Animation Function
