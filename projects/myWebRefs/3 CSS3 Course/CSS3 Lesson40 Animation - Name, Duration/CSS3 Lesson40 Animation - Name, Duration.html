[CSS3 Animation - [Name || Duration] ] Concept:

{Animation [Name || Duration]} => Is Needing for using [prefixes] before the animation property and it's branshed properties

{Animation - [keyframes]}: It's syntax containing following components :

  1. [@ keyframes]: which is the main keyword of declaring new animation function => {Required}
  3. [Animation Name]: which is the Name of declaring new animation function, and will be declared later with in desired element => {Required}
  5. [Animation Selectors]: which are the Animation function Steps / Milestones, and will they can be assigned by using one of (2) options => {Required}
    1. Using From-> To keyframes : [From (0%)] [to (100%)]:
    2. Using Percentages keyframes : [(0%) (10%) (20%) (30%)-> (100%)]:
  7. [Animation Selector's CSS Attributes]: which are the CSS3 propeties will be assigned to a certain Percentage/Step
  9. You can use more than one percentage (with unified properites) in one line, such as :

    [0%, 100% { CSS3 Attributes} ]

{Animation - [Name || Duration]}: These are the first and second child perperites of Animation syntax , with following details :

  1. { -webkit-animation-name || animation-name} =>

    Which is the Declared Animation Name

    Default value of Animation Name : none (due to it will be assigned by developer later)

    Animation Name is up to developer to assigned (but it is recommended to be suit to the declared animiation function)

  2. { -webkit-animation-duration || animation-duration} =>

    Which is the Animation period time to be proccesed in

    [Default value] : 0s

    Can be assigned by (s: Second ) || (ms: Milli Second)

[CSS3 / Animation - [Name || Duration] ] Importance & Benefits:

=> [CSS3 / Animation - [Name || Duration] ] : This Advanced sub properties enabling the following features & imporatance :

  • [animation-name] : Give a ability to assign a certain one/more Declared Animation Name to a specific Element
  • [animation-duration] : Give a ability to assign a certain one/more Declared Animation period of time for each animation name to a specific Element
