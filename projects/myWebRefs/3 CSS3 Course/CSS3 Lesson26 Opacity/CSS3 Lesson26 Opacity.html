[CSS3 Lesson26 Opacity] Concept:

  • {CSS2 Opacity} old Concept :

    • The old Concept was only certain for Background property

  • {CSS3 Opacity} new Concept :

    • This is a new property in CSS3 , which providing the ability to Opacity (hiding) to the Element and all it's children

    In CSS3 default Values types as following :


    • [#Value] => To Set the opacity by a singel Number0 (1 -> 0)
    • [%Percentage] => To Set the opacity by a singel Number (100% -> 0%)
  • {CSS3 Opacity} supportance :

    • Internet Explorer is only supporting this property by {filter: alpha(opacity= #value)}

  • {CSS3 Opacity} the differeces between (Opacity:0 || visibility: hidden || display: none ) :

    • [display: none]=> Is to hide the Element and it's space in page workfloor

    [visibility: hidden]=> Is to hide the Element but keeps it's space in page workfloor

    [opacity: 0]=> Is like the (visibility: hidden) , to hide the Element but keeps it's space in page workfloor
