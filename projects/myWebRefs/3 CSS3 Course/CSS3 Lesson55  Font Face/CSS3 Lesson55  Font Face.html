Font Face Main Concept :

In the Original Font property, We used to write the property of Font Famliy with multiple types to avoid un Supported ones by some web Browsers


They created [Web Safe Font] as a Rerferences Web site to offer the compatablity of each web Browser & Operating System of each type of Font & Font Family,
such as {www.cssfontstack.com}


As an example of most common unsupported font type in Operating System => {Tahoma is not supported in Linux OS}


This is a new feature to Create / Import a new {font type} that is not existed in your pc by your own


This is new feature has (2) methods to apply :


  1. External Method :=> {Importing the link from the web of certain font family by creaing link tag with this code }
  2. Intenal Method :=> {Creating the new font face in the CSS3 file [@font-face] & adding the font path in my pc with this code}

There are Multiple websites to pick up and test new font in web browser, and have the source code of it in order to
1- Embedd it in source code as [External code source]
2- Download it to declare the new font-face in CSS3 as [Internal Code source]

Font is more complicated and has more features to study and require more Advanced course to study

  1. TTF : True Type Font
  2. OTF : Open Type Font
  3. WOFF : Web Open Font
  4. WOFF2 : Web Open Font2
  5. SVG : Scalable Type Font
  6. EDF : Embedded Type Font
