> Main Subjects in this lesson are as following :

Selectors with different types following:

  1. [Checked] Selector => [:checked]
  2. [Enabled] Selector => [:enabled]
  3. [Disabled] Selector => [:disabled]
  4. [Empty] Selector => [:empty]

[50 Css3 - Selectors - Checked, Enabled, Disabled, Empty ] Concept:

{Selector- [:checked] {checked}} =>

  • This selector is only for form elements
  • This selector to select the Checked Checkable Element(s) with in a certain Form or any Form

{Selector- [:disabled] {disabled}} =>

  • This selector is only for form elements
  • This selector to select the enabled Element(s) with in a certain Form or any Form

{Selector- [:enabled] {enabled}} =>

  • This selector is only for form elements
  • This selector to select the disabled Element(s) [has a disabled Attribute] with in a certain Form or any Form

{Selector- [:empty] {empty}} =>

  • This selector is for public elements in the whole page document
  • This selector to select the empty Element(s) [has no children or text node ] with in whjole web page document
