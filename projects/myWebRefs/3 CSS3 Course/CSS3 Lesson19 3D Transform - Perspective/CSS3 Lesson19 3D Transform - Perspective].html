[3D Transform - Perspective/Perspective-Origin] Concept:

=> [3D Transform - Perspective] : It is the Distance between the user view and the Element {Obligatory}

=> [3D Transform - Perspective-Origin] : It is the angel of user view to the element inside the it's parent {Recommended}

=> [3D Transform - Perspective/Perspective-Origin] : This is one of CSS3 3D advancd Properties which is nessery to control all 3d-Transform Properties of the Element by assinging the amount of inside element Scale to display inside or through {3D Environment Frame}.

=> [3D Transform - Perspective/Perspective-Origin] : The Default Value of Perspective/Perspective-Origin is => {none}

=> [3D Transform - Perspective/Perspective-Origin] => It is containing following types of aspects/values/Parameters :

  1. {Perspective} :
    • [Value] => by Pixels (px)
  2. {Perspective-origin} :
    • [Value] => by (px)
    • [Percentage] => by (X %) (Y %)
    • [Direction Name] => by Direction name (left, right , top , botttom , center)

=> [3D Transform - Perspective/Perspective-Origin] => If the 3D-Transform has a value greater than persptive propety value => it will not be dispalyed

[3D Transform - Perspective/Perspective-Origin ] Importance & Benefits:

=> [3D Transform - Perspective/Perspective-Origin ] : This Advanced property enabling the following features & imporatance :

  • Essential for all 3D Property to work properly
  • Great control of 3d Environment {Amount of 3D Scale & Angle of User Viwe}
  • Ability of using multiple units to control Perspective-Origin
  • 3D Perspective/Perspective-Origin property will provide unique effects specilaly when merged with other 3D Transform properties animations effects {which will be studied will later lessons}
