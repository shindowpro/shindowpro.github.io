[3D Transform - Transform Origin, Style] Concept:

  • A) {Transform-Origin} main Concept :
    1. It a contorling tool of transforming aixe of the Element while performing the Transform effect
    2. This property Require {transforming} Effect first to be Activated on the Element
    3. It can accept the following types of aspects/values/Parameters (the Same of {perspective-origin} ):

      • [Percentage] => by (X %) (Y %)
      • [Direction Name] => by Direction name (left, right , top , botttom , center)
    4. The Differences between {Transform-origin} & {perspective-origin} as following:
      • {Transform-origin} => Controling (position) of the Element Trasnforming Axe {regaless of the Type of the Transforing}
      • {perspective-origin} => Controling (position) of the viewer of the Element Trasnforming Axe {regaless of the Type of the Transforing}
  • B) {Transform-Style} :
    1. It is a tool to control the {3D} Transforming element Display/Show. and it is being set into the Parent Container. And (2) Following options :
      • [Flat]: It is the Default opition of this property. and it is to keep the transforming Element with Flat mode
      • [Preserve-3d]: This option keep the transforming Element with 3D mode {parent Containe & it's Children Elements}
    2. [Flat, reserve-3d] => [Flat]: This mode will ont display all kinds of the Transforming Effect in {3D} Successfully [specially the Z-Effect types] such as {translateZ}
      but in [reserve-3d] :will display all featrues of all kinds of 3d Successfully

[3D Transform - Transform Origin, Style ] Importance & Benefits:

=> [3D Transform - Transform Origin, Style ] : This Advanced property enabling the following features & imporatance :

  • Adding Extraodinary features to the 3D-Transforming Elements, by controling the Axies of Transforming Effects
  • Great control of 3d Environment {Amount of 3D Scale & Angle of User Viwe}
  • Ability of using multiple Child Elements inside the parent Container with Different kinds of transforming types
  • Ability to have more control on activating 3D mode on spcsific containers & deactivating it on the others
  • 3D Transform Origin, Style property will provide unique effects specilaly when merged with other 3D Transform properties animations effects

[3D Transform - Transform Origin, Style] General Practical Applications :

=> Following is the Applied example of { [3D-Transform-origin] & [3d-Transform-style] with different kinds of Transforms } :

(2) Fieldsets contain (3) sections for each (of 3 types of transform types) && Each Section has all options available in the Property of this section as following:

  • A) First Fieldset: { Tranform-origin }: And it has (3) Sections as following :
    1. Rotate() Effect : and it has (6) sub Secitons as follwing :
      • (1) Sub Section : Rotate Transform With [transform-origin: right center]
      • (2) Sub Section : Rotate Transform With [transform-origin: left center]
      • (3) Sub Section : Rotate Transform With [transform-origin: top center]
      • (4) Sub Section : Rotate Transform With [transform-origin: bottom center]

      • (1) Sub Section : Scale Transform With [transform-origin: right center]
      • (2) Sub Section : Scale Transform With [transform-origin: left center]
