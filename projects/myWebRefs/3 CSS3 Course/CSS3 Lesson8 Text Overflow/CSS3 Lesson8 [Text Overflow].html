[Text Overflow] Concept:

=> [Text-Overflow] : This Advanced Tool to control Text over flow the parent Container.

=> Differnces between [Text-Overflow] & [overflow] & [white-space] :

  • [Text-Overflow] => To Control the text-overFlow over it's container by

    • {ellipsis} : Will show a sing of (...) => which means a sign of a hidden Text.
    • {clip} : It is the Default Value of Text-overflow => which will normally cut the text sentenece with out any sign of Contiuning Text
  • [Overflow] => To control the non-Textual over flow of it's content, with the following options :

    • {hidden} => To hide the overflow Content
    • {visible} => To show the overflow Content , but with out any foramting
    • {scroll} => To show the overflow Content with a scrolling bars (horz + Vert)
    • {inhert} => To inherting the Parent's property value of overflow
  • [white-space] => To control the Wrapping property of the Text with mainly following options :

    • {normal} => The Normal Value of this Property, which is meaning to Allow text Wrapping
    • {nowrap} => To prevent text wrapping

[ Text Overflow ] Syntax & Methods:

// *) [Text Overflow] Syntax as following :

//1) {Exmple 1} [(ellipsis) : show the sign of text overflow] Syntax:
//a) [HTML] file :
< div > This is A normal Div with long sentence with [overflow:hidden + white-space:nowrap]
</div>
// ------------------------------------------------------------------------------

//b) [CSS3] file :
div {
margin: 40px;
padding: 3px;
width: 400px;
border: 1px solid #CCC;
backgournd-color: #FFF;
overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;
}
// ------------------------------------------------------------------------------

//2) {Exmple 2} [(hovering) : to show up the hidden text] Syntax:
div: hover {
overflow: visible
white-space: normal
}
// ------------------------------------------------------------------------------

// ------------------------------------------------------------------------------

//3) {Exmple 3} [Resetting the [text-overflow] to normal value] Effect Syntax:
//*) [CSS3] file (Contiuning to the previous example) :
div {
text-overflow: clip;
}
// ------------------------------------------------------------------------------
